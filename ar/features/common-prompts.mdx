صُمم OpenYida للتفاعل باللغة الطبيعية. تم التحقق من صحة الموجهات التالية ويمكن استخدامها في جميع أدوات الذكاء الاصطناعي المدعومة.

---

## إنشاء التطبيقات

```
ساعدني في بناء تطبيق xxx
```
```
ساعدني في بناء CRM
```
```
ساعدني في بناء تطبيق حاسبة رواتب شخصية
```
```
ساعدني في إنشاء نظام IPD على Yida لإدارة العملية الكاملة لإنتاج الرقائق
```
```
إنشاء تطبيق بناءً على مستند المتطلبات: [لصق PRD]
```
```
إنشاء لعبة تهنئة عيد ميلاد على Yida
```
```
بناء لعبة تخمين ألغاز عيد الفوانيس
```

## عمليات النماذج والصفحات

### إنشاء صفحات النماذج

```
ساعدني في إنشاء صفحة نموذج xxx
```
```
إنشاء نموذج طلب إجازة يتضمن: اسم الموظف، القسم، تاريخ البدء، تاريخ الانتهاء، سبب الإجازة
```

### إضافة حقول النموذج

```
أضف حقل [اسم الحقل] إلى [اسم الصفحة]، نوع الحقل: [TextField/NumberField/DateField/SelectField/...]
```
```
أضف حقل "القسم" من نوع القائمة المنسدلة إلى نموذج معلومات الموظف، الخيارات: قسم الهندسة، قسم التسويق، قسم المبيعات، قسم الموارد البشرية
```

<Note>
يتم قبول [أنواع الحقول المدعومة](/ar/features/skills#أنواع-الحقول-المدعومة) فقط. الأنواع غير المدعومة (مثل `MobileField` و `EmailField`) سيتم رفضها.
</Note>

### تعديل خصائص الحقل

```
اجعل حقل [اسم الحقل] في [اسم الصفحة] إلزاميًا
```
```
اجعل حقل [اسم الحقل] في [اسم الصفحة] للقراءة فقط
```
```
غيّر تسمية حقل [اسم الحقل] في [اسم الصفحة] إلى [تسمية جديدة]
```

### صفحات العرض المخصصة

```
أنشئ صفحة عرض مخصصة لـ [اسم الصفحة]
```
```
إنشاء صفحة هبوط لمنتج التعاون المؤسسي
```
