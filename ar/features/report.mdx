## إنشاء التقارير

### الاستخدام الأساسي

```bash
openyida create-report <appType> "اسم التقرير" '<JSON تعريف الرسم البياني أو مسار ملف>'
```

يمكن أن يكون المعامل الثالث سلسلة JSON مضمّنة أو مسار ملف JSON.

### مثال: إنشاء تقرير مبيعات

```bash
openyida create-report APP_XXXXXX "Sales Dashboard" '{
  "charts": [
    {
      "type": "bar",
      "title": "Monthly Sales",
      "dataSource": "sales_form",
      "xField": "month",
      "yField": "amount"
    }
  ]
}'
```

### تحميل تعريف الرسم البياني من ملف

```bash
openyida create-report APP_XXXXXX "Annual Report" ./charts-config.json
```

## إضافة رسوم بيانية إلى التقرير

```bash
openyida append-chart <appType> <reportId> '<JSON تعريف الرسم البياني أو مسار ملف>'
```

### مثال

```bash
openyida append-chart APP_XXXXXX REPORT_XXXXX '{
  "type": "line",
  "title": "Growth Trend",
  "dataSource": "form_sales",
  "xField": "month",
  "yField": "growth_rate"
}'
```

## أنواع الرسوم البيانية

| النوع | الوصف | الاستخدام الأمثل |
|------|------|----------|
| **bar** | رسم بياني عمودي | مقارنة الفئات |
| **line** | رسم بياني خطي | الاتجاهات عبر الزمن |
| **pie** | رسم بياني دائري | علاقات الجزء بالكل |
| **table** | جدول بيانات | عرض البيانات المفصل |
| **area** | رسم بياني مساحي | الاتجاهات التراكمية |
| **scatter** | رسم بياني مبعثر | تحليل الارتباط |

## تنسيق تعريف الرسم البياني

### الرسم البياني العمودي

```json
{
  "type": "bar",
  "title": "Sales by Region",
  "dataSource": "form_uuid",
  "xField": "region",
  "yField": "amount",
  "aggregation": "sum"
}
```

### الرسم البياني الخطي

```json
{
  "type": "line",
  "title": "Traffic Trend",
  "dataSource": "form_uuid",
  "xField": "date",
  "yField": "visitors",
  "aggregation": "sum"
}
```

### الرسم البياني الدائري

```json
{
  "type": "pie",
  "title": "Market Share",
  "dataSource": "form_uuid",
  "categoryField": "product",
  "valueField": "share"
}
```

### جدول البيانات

```json
{
  "type": "table",
  "title": "Detail View",
  "dataSource": "form_uuid",
  "columns": [
    { "field": "name", "title": "Name" },
    { "field": "amount", "title": "Amount" },
    { "field": "date", "title": "Date" }
  ]
}
```

| الدالة | الوصف |

| **max** | القيمة القصوى |

```json
      "type": "table",

      "columns": [

```

إنشاء التقرير:

```bash
openyida create-report APP_XXXXXX "Sales Analysis Report" ./sales-report-charts.json
```

## أفضل الممارسات

1. **اختر نوع الرسم البياني المناسب** - الاتجاهات بالخطي، المقارنات بالعمودي، النسب بالدائري
2. **تحكم في عدد الرسوم البيانية** - يُنصح بعدم تجاوز 6 رسوم بيانية لكل تقرير
3. **استخدم جداول البيانات** - أكمل الرسوم البيانية بجداول لفحص البيانات المفصلة
4. **اضبط أحجام الصفحات المناسبة** - يُنصح بـ 10-20 سجلًا لكل صفحة للجداول
