## إنشاء تقرير

### الاستخدام الأساسي

```bash
openyida create-report <appType> "اسم التقرير" '<JSON تعريف الرسم البياني>'
```

### مثال: إنشاء تقرير المبيعات

```bash
openyida create-report APP_XXXXXX "تقرير المبيعات الشهري" '{
  "charts": [
    {
      "type": "bar",
      "title": "المبيعات الشهرية",
      "dataSource": "form_sales",
      "xField": "month",
      "yField": "amount"
    },
    {
      "type": "pie",
      "title": "نسبة مبيعات المنتجات",
      "dataSource": "form_sales",
      "categoryField": "product_type",
      "valueField": "amount"
    }
  ]
}'
```

### تحميل تعريف الرسم البياني من ملف

```bash
openyida create-report APP_XXXXXX "التقرير الإحصائي السنوي" ./charts-config.json
```

## تنسيق تعريف الرسم البياني

### الرسم البياني العمودي

```json
{
  "type": "bar",
  "title": "عنوان الرسم البياني العمودي",
  "dataSource": "form_uuid",
  "xField": "category_field",
  "yField": "value_field",
  "aggregation": "sum"
}
```

### الرسم البياني الخطي

```json
{
  "type": "line",
  "title": "عنوان رسم بياني الاتجاهات",
  "dataSource": "form_uuid",
  "xField": "date_field",
  "yField": "value_field",
  "aggregation": "sum"
}
```

### الرسم البياني الدائري

```json
{
  "type": "pie",
  "title": "عنوان رسم بياني النسبة",
  "dataSource": "form_uuid",
  "categoryField": "category_field",
  "valueField": "value_field"
}
```

### جدول البيانات

```json
{
  "type": "table",
  "title": "تفاصيل البيانات",
  "dataSource": "form_uuid",
  "columns": [
    { "field": "name", "title": "الاسم" },
    { "field": "amount", "title": "المبلغ" },
    { "field": "date", "title": "التاريخ" }
  ]
}
```

## إضافة رسوم بيانية إلى التقرير

```bash
openyida append-chart <appType> <reportId> '<JSON تعريف الرسم البياني>'
```

### مثال

```bash
openyida append-chart APP_XXXXXX REPORT_XXXXX '{
  "type": "line",
  "title": "اتجاه النمو",
  "dataSource": "form_sales",
  "xField": "month",
  "yField": "growth_rate"
}'
```

## مرجع أنواع الرسوم البيانية

| النوع | الوصف | الحقول المطلوبة |
|------|------|----------|
| `bar` | رسم بياني عمودي | xField, yField |
| `line` | رسم بياني خطي | xField, yField |
| `pie` | رسم بياني دائري | categoryField, valueField |
| `table` | جدول بيانات | columns |
| `area` | رسم بياني مساحي | xField, yField |
| `scatter` | رسم بياني مبعثر | xField, yField |

---

## دوال التجميع

| الدالة | الوصف |
|------|------|
| `sum` | المجموع |
| `avg` | المتوسط |
| `count` | العد |
| `max` | القيمة القصوى |
| `min` | القيمة الدنيا |

## أفضل الممارسات

1. **اختر نوع الرسم البياني المناسب** - الاتجاهات بالخطي، المقارنات بالعمودي، النسب بالدائري
2. **تحكم في عدد الرسوم البيانية** - يُنصح بعدم تجاوز 6 رسوم بيانية لكل تقرير
3. **استخدم جداول البيانات** - أكمل الرسوم البيانية ببيانات مفصلة لعرض القيم الأصلية
4. **اضبط التقسيم إلى صفحات المناسب** - يُنصح بـ 10-20 سجلًا لكل صفحة للجداول
