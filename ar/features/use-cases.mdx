---

## الأعمال المؤسسية

بناء منصات أعمال متعددة الوحدات، تدعم النماذج المعقدة وسير العمل وارتباطات البيانات.

<AccordionGroup>
  <Accordion title="IPD (تطوير المنتجات المتكامل)">
    **Prompt:**
    ```
    ساعدني في إنشاء نظام IPD في Yida لإدارة عملية إنتاج الرقائق الكاملة،
    بما في ذلك مراحل بدء المشروع ومراجعة التصميم وإنتاج الرقائق والإنتاج الضخم
    ```
    **النتيجة:** تطبيق متعدد النماذج مع تتبع المراحل وقوائم المراجعة وإدارة المعالم والرؤية بين الفرق.
  </Accordion>

  <Accordion title="CRM (إدارة علاقات العملاء)">
    **Prompt:**
    ```
    ساعدني في بناء CRM يحتوي على وحدات العملاء المحتملين والعملاء والفرص وسجلات المتابعة
    ```
    **النتيجة:** نماذج مرتبطة: عميل محتمل → عميل → فرصة → عقد، مع تتبع الحالة وحقول التعيين.
  </Accordion>

  <Accordion title="إدارة الموارد البشرية والإجازات">
    **Prompt:**
    ```
    أنشئ نظام إدارة إجازات الموارد البشرية مع طلبات الإجازة وسير الموافقة وإحصائيات رصيد الإجازات
    ```
    **النتيجة:** نموذج طلب إجازة مع سير الموافقة وآلة حاسبة للرصيد ولوحة تحكم المدير.
  </Accordion>

  <Accordion title="إدارة المشاريع">
    **Prompt:**
    ```
    بناء تطبيق إدارة مشاريع يحتوي على المهام والمعالم وأعضاء الفريق وتتبع التقدم
    ```
    **النتيجة:** قائمة مهام مع حقول الأولوية/الحالة/المسؤول ومتتبع المعالم وعرض مخطط جانت.
  </Accordion>
</AccordionGroup>

##💰 عملية وآلات حاسبة

أدوات حساب أو جمع بيانات ذات غرض واحد.

<AccordionGroup>
  <Accordion title="حاسبة الراتب الشخصي">
    **Prompt:**
    ```
    ساعدني في بناء حاسبة راتب شخصي، أدخل الراتب الإجمالي، احسب الضريبة والراتب الصافي تلقائيًا
    ```
    **النتيجة:** نموذج مع حقول صيغة للخصومات قبل الضريبة وحساب الضريبة وملخص الراتب الصافي.
  </Accordion>

  <Accordion title="متتبع الميزانية">
    **Prompt:**
    ```
    ساعدني في بناء متتبع ميزانية شخصي لتسجيل الدخل وفئات النفقات والملخصات الشهرية
    ```
    **النتيجة:** نموذج إدخال الدخل/النفقات مع قائمة منسدلة للفئات ورصيد شهري محسوب تلقائيًا.
  </Accordion>
</AccordionGroup>

صفحات التسويق وعروض المنتجات والبوابات الداخلية.

<AccordionGroup>
  <Accordion title="صفحة هبوط منتج مؤسسي">
    **Prompt:**
    ```

    ```
    **النتيجة:** صفحة عرض مخصصة كاملة مع بانر الشاشة الرئيسية وبطاقات الوظائف ونموذج الاتصال المضمن.
  </Accordion>

  <Accordion title="صفحة تسجيل الفعاليات">
    **Prompt:**
    ```

    ```
    **النتيجة:** صفحة تسجيل بعلامة تجارية مع التحقق من صحة النموذج ومنطق التأكيد.
  </Accordion>
</AccordionGroup>

## تفاعلية وألعاب مصغرة

أدوات تفاعلية للأنشطة الداخلية والمهرجانات والتسويق.

<AccordionGroup>
  <Accordion title="لعبة ألغاز مهرجان الفوانيس">
    **Prompt:**
    ```
    أنشئ لعبة ألغاز لمهرجان الفوانيس، يولد الذكاء الاصطناعي صور الألغاز، يرسل المستخدمون الإجابات، إذا أخطأوا يرد الذكاء الاصطناعي بفكاهة
    ```
    **النتيجة:** صفحة لعبة مع صور ألغاز مولدة بالذكاء الاصطناعي ونموذج إرسال الإجابات ومنطق الاستجابة المشروطة.
  </Accordion>

  <Accordion title="لعبة تهنئة عيد الميلاد المصغرة">
    **Prompt:**
    ```
    بناء لعبة تهنئة عيد ميلاد مصغرة حيث يدخل المستخدمون اسمهم ويحصلون على بطاقة تهنئة مخصصة من الذكاء الاصطناعي
    ```
    **النتيجة:** مولد بطاقات تفاعلي مع تخزين البيانات لتتبع الاستجابات.
  </Accordion>
</AccordionGroup>

##🔌امل API (الموصلات)

ربط تطبيقات Yida بالأنظمة الخارجية من خلال موصلات HTTP.

<AccordionGroup>
  <Accordion title="إشعارات Webhook">
    **Prompt:**
    ```
    أنشئ موصل HTTP لـ webhook الخاص بـ DingTalk، أضف إجراء POST لإرسال إشعارات إرسال النماذج
    ```
    **النتيجة:** موصل مُهيأ مع قدرة الاختبار، قابل للتشغيل عند أحداث النموذج.
  </Accordion>

  <Accordion title="إنشاء ذكي من curl">
    **Prompt:**
    ```
    أنشئ موصل بذكاء من أمر curl هذا: curl -X POST https://api.example.com/v1/users -H "Authorization: Bearer TOKEN" -d '{"name":"test"}'
    ```
    **النتيجة:** موصل مُهيأ تلقائيًا، يستخرج عنوان URL الأساسي والرؤوس ومعلمات الإجراء.
  </Accordion>
</AccordionGroup>
