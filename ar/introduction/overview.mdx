## نظرة عامة

**OpenYida** هي أداة سطر أوامر بدون تكوين تربط بين مساعد البرمجة بالذكاء الاصطناعي ومنصة [Yida](https://www.aliwork.com) منخفضة الكود. قم بالتثبيت مرة واحدة، ثم صف المتطلبات باللغة الطبيعية — سيتولى OpenYida تلقائياً: إنشاء التطبيق، إدارة النماذج، نشر الصفحات، تكوين الموصلات، إنشاء التقارير، والنشر في ثوانٍ على Yida.

🌐 **الموقع الرسمي**: [https://openyida.ai](https://openyida.ai)

## الوظائف الأساسية

| وحدة الوظيفة | الوصف |
|---------|------|
| **إدارة التطبيقات** | إنشاء التطبيق، الاستيراد/التصدير، الترحيل والنشر |
| **إدارة النماذج** | إنشاء/تحديث النماذج، الحصول على Schema، تكوين الأذونات |
| **نشر الصفحات** | تطوير الصفحات المخصصة، تجميع Babel، نشر بضغطة واحدة |
| **تكوين العمليات** | إنشاء نماذج العمليات، تكوين قواعد الموافقة |
| **إدارة البيانات** | استعلام بيانات النماذج، مثيلات العمليات، إدارة المهام |
| **الموصلات** | إنشاء موصل HTTP، تكامل API، استيراد ذكي لـ curl |
| **التقارير والرسوم البيانية** | إنشاء تقارير Yida، إضافة الرسوم البيانية، تصور البيانات |
| **إدارة CDN** | رفع الصور، تحديث ذاكرة التخزين المؤقت، تكوين OSS |
| **خادم MCP App** | إدارة تطبيقات Yida في محادثات الذكاء الاصطناعي عبر بروتوكول MCP، مع لوحات معلومات تفاعلية ورسوم بيانية |

## الفرق مع منصات البناء الأخرى بالذكاء الاصطناعي

| البعد | OpenYida | منصات البناء الأخرى بالذكاء الاصطناعي |
|------|----------|------------------|
| **المستخدمون المستهدفون** | المطورون (الذين يفهمون البرمجة) | موظفو الأعمال (غير المطورين) |
| **طريقة التفاعل** | اللغة الطبيعية + محادثة الذكاء الاصطناعي | السحب والإفلات المرئي + لوحة التكوين |
| **النتيجة** | تطبيق Yida (قابل للتعديل الثانوي، يدعم قدرات الكود المنخفضة الكاملة) | التكوين (تشغيل الصندوق الأسود) |
| **طريقة النشر** | منصة Yida | ربط منصة SaaS |
| **نموذج الذكاء الاصطناعي** | اختيار حسب الحاجة، اختيار النموذج الأنسب | تحديد المنصة، لا يمكن التغيير |
| **الأمان والامتثال** | Yida لديها قدرات أمان وامتثال شاملة | الاعتماد على قدرات المنصة |
