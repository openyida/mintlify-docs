## نظرة عامة

**OpenYida** هي أداة CLI بدون تكوين تربط مساعدي البرمجة بالذكاء الاصطناعي بمنصة [Yida](https://www.aliwork.com) منخفضة الكود. ثبّت مرة واحدة، ثم صف متطلباتك باللغة الطبيعية — يتولى OpenYida تلقائياً: إنشاء التطبيقات، إدارة النماذج، نشر الصفحات، تكوين الموصلات، إنشاء التقارير، والنشر على Yida في ثوانٍ.

🌐 **الموقع الرسمي**: [https://openyida.ai](https://openyida.ai)

## الميزات الرئيسية

| وحدة الميزة | الوصف |
|---------|------|
| **إدارة التطبيقات** | إنشاء التطبيقات، الاستيراد/التصدير، الترحيل والنشر |
| **إدارة النماذج** | إنشاء/تحديث النماذج، الحصول على Schema، إعداد الأذونات |
| **نشر الصفحات** | تطوير الصفحات المخصصة، تجميع Babel، النشر بنقرة واحدة |
| **تكوين سير العمل** | إنشاء نماذج العمليات، إعداد قواعد الموافقة |
| **إدارة البيانات** | استعلام بيانات النماذج، نسخ العمليات، إدارة المهام |
| **الموصلات** | إنشاء موصلات HTTP، تكامل API، استيراد curl ذكي |
| **التقارير والرسوم البيانية** | إنشاء تقارير Yida، إضافة الرسوم البيانية، تصور البيانات |
| **إدارة CDN** | رفع الصور، تحديث التخزين المؤقت، إعداد OSS |

## الاختلاف عن منصات البناء بالذكاء الاصطناعي الأخرى

| البُعد | OpenYida | منصات البناء بالذكاء الاصطناعي الأخرى |
|------|----------|------------------|
| **المستخدمون المستهدفون** | المطورون (الأشخاص الذين يفهمون الكود) | مستخدمو الأعمال (غير المطورين) |
| **وضع التفاعل** | اللغة الطبيعية + حوار الذكاء الاصطناعي | السحب والإفلات المرئي + لوحات التكوين |
| **المُخرجات** | تطبيق Yida (قابل للتعديل لاحقاً، دعم كامل للكود المنخفض) | تكوين (تنفيذ الصندوق الأسود) |
| **وضع النشر** | منصة Yida | ربط منصة SaaS |
| **نموذج الذكاء الاصطناعي** | اختيار حسب الحاجة، اختر الأنسب | محدد من المنصة، لا يمكن تغييره |
| **الأمان والامتثال** | Yida لديها قدرات أمان وامتثال كاملة | يعتمد على قدرات المنصة |
