## المتطلبات الأساسية

قبل البدء، تأكد من تثبيت التبعيات التالية:

| التبعية | الإصدار | أمر التحقق |
|------|------|----------|
| [Node.js](https://nodejs.org) | ≥ 18 | `node --version` |

<Tip>
  هذا كل ما تحتاجه. OpenYida هو تطبيق Node.js نقي — لا يحتاج إلى Python، ولا Playwright، ولا خطوات بناء.
</Tip>

---

## الخطوة 1 — التثبيت

```bash
npm install -g openyida
```

تحقق من التثبيت:

```bash
openyida --version
```

---

## الخطوة 2 — تسجيل الدخول إلى Yida

```bash
openyida login
```

سيفتح OpenYida نافذة متصفح (أو يعرض رمز QR) لمصادقتك باستخدام حساب Yida/Alibaba الخاص بك. يتم تخزين بيانات الاعتماد محليًا — تحتاج فقط إلى القيام بذلك مرة واحدة.

<Info>
  إذا كنت تستخدم **Wukong** كأداة ذكاء اصطناعي، فاستخدم علامة تسجيل الدخول الخاصة:

  ```bash
  openyida login --wukong
  ```
</Info>

---

## الخطوة 3 — التحقق من البيئة

```bash
openyida env
```

يكتشف هذا أداة ترميز الذكاء الاصطناعي النشطة حاليًا (Claude Code، Cursor، Wukong، إلخ) ويعرض حالة تسجيل الدخول الحالية الخاصة بك. قم بتشغيل هذا الأمر في أي وقت تقوم فيه بتبديل البيئات.

---

## الخطوة 4 — تهيئة مساحة عمل المشروع

```bash
openyida copy
```

ينسخ هذا قالب مساحة العمل `project/` إلى دليل عمل أداة الذكاء الاصطناعي الحالية الخاصة بك، مما يسمح للذكاء الاصطناعي بالوصول إلى تكوين التطبيق الخاص بك وقوالب الصفحات.

---

## الخطوة 5 — ابدأ البناء

الآن فقط تحدث إلى مساعد ترميز الذكاء الاصطناعي الخاص بك. إليك بعض مطالبات البدء:

```
ابنِ لي نظام IPD على Yida لإدارة سير عمل إنتاج الرقائق بالكامل
```

```
ساعدني في بناء نظام CRM مع leads و contacts و deals
```

```
أنشئ تطبيق حاسبة راتب شخصي
```

```
أنشئ لعبة مصغرة لتهنئة عيد الميلاد على Yida
```

سيستخدم الذكاء الاصطناعي حزمة مهارات OpenYida لاستدعاء أوامر CLI الصحيحة بالتسلسل: `create-app` → `create-form` / `create-page` → `publish` → `save-share-config`.

---

## الخطوة 6 — نشر التطبيق

بعد أن يولد الذكاء الاصطناعي التطبيق، استخدم الأمر التالي للنشر:

```bash
openyida publish
```

أو اسأل الذكاء الاصطناعي:
```
انشر صفحة [اسم الصفحة]
اجعل الصفحة متاحة للجمهور
```

---

## تشغيل التشخيص

إذا كان هناك شيء غير صحيح، قم بتشغيل أداة التشخيص المدمجة:

```bash
openyida doctor
```

يتحقق من إصدار Node.js الخاص بك، وحالة تسجيل الدخول، وتكوين البيئة، ويمكنه إصلاح المشكلات الشائعة تلقائيًا باستخدام `--fix`.

---

## تكامل OpenClaw

إذا كنت تستخدم OpenClaw، قم بتثبيت حزمة مهارات OpenYida مباشرة:

```bash
npx clawhub@latest install nicky1108/yida-app
```

ثم استخدمها عبر [yida-app على ClawhHub](https://clawhub.ai/nicky1108/yida-app).
