## المتطلبات الأساسية

قبل البدء، تأكد من تثبيت التبعيات التالية:

| التبعية | الإصدار | أمر التحقق |
|---------|---------|------------|
| [Node.js](https://nodejs.org) | ≥ 18 | `node --version` |

<Tip>
  يكفي Node.js فقط. لا حاجة لـ Python أو Playwright أو خطوات البناء — تنفيذ خالص بـ Node.js.
</Tip>

---

## الخطوة 1 — التثبيت

```bash
npm install -g openyida
```

تحقق من التثبيت:

```bash
openyida --version
```

---

## الخطوة 2 — تسجيل الدخول إلى Yida

```bash
openyida login
```

سيفتح OpenYida نافذة متصفح (أو يعرض رمز QR) للمصادقة بحساب Yida/Alibaba الخاص بك. يتم تخزين بيانات الاعتماد محليًا — مرة واحدة فقط.

<Info>
  إذا كنت تستخدم **Wukong** كأداة ذكاء اصطناعي، استخدم علامة تسجيل الدخول الخاصة:

  ```bash
  openyida login --wukong
  ```
</Info>

---

## الخطوة 3 — التحقق من البيئة

```bash
openyida env
```

يكشف أداة الترميز بالذكاء الاصطناعي النشطة حاليًا (Claude Code، Cursor، Wukong، إلخ) ويعرض حالة تسجيل الدخول. قم بتشغيل هذا الأمر في أي وقت عند تبديل البيئات.

---

## الخطوة 4 — تهيئة مساحة عمل المشروع

```bash
openyida copy
```

ينسخ قالب مساحة العمل `project/` إلى دليل العمل الحالي لأداتك الذكية، مما يسمح للذكاء الاصطناعي بالوصول إلى تكوين التطبيق وقوالب الصفحة.

---

## الخطوة 5 — البدء في البناء

تحدث مباشرة مع مساعد الترميز الذكي. إليك بعض أمثلة الموجهات للبدء:

```
ابنِ لي نظام IPD على Yida لإدارة سير عمل إنتاج الرقائق بالكامل
```

```
ساعدني في بناء CRM مع العملاء المحتملين والعملاء والصفقات
```

```
أنشئ تطبيق حاسبة راتب شخصية
```

```
أنشئ لعبة تهنئة عيد ميلاد صغيرة على Yida
```

سيستخدم الذكاء الاصطناعي حزمة مهارات OpenYida لاستدعاء أوامر CLI بالترتيب الصحيح: `create-app` → `create-form` / `create-page` → `publish` → `save-share-config`.

---

## الخطوة 6 — نشر التطبيق

بمجرد أن يقوم الذكاء الاصطناعي بإنشاء التطبيق، قم بتشغيل أمر النشر:

```bash
openyida publish
```

أو اسأل الذكاء الاصطناعي:
```
انشر صفحة [اسم الصفحة]
اجعل الصفحة متاحة للجمهور
```

---

## تشغيل التشخيص

إذا واجهت مشاكل، قم بتشغيل أداة التشخيص المدمجة:

```bash
openyida doctor
```

يفحص إصدار Node.js وحالة تسجيل الدخول وتكوين البيئة، ويمكنه استخدام `--fix` لإصلاح المشاكل الشائعة تلقائيًا.

---

## تكامل OpenClaw

إذا كنت تستخدم OpenClaw، ثبّت حزمة مهارات OpenYida مباشرة:

```bash
npx clawhub@latest install nicky1108/yida-app
```

ثم استخدمه عبر [yida-app على ClawHub](https://clawhub.ai/nicky1108/yida-app).
