يوضح هذا الدليل كيفية تثبيت OpenYida وإنشاء تطبيق أساسي.

## الخطوة 1 — التثبيت

```bash
npm install -g openyida
```

تحقق من نجاح التثبيت:

```bash
openyida --version
```

## الخطوة 2 — تسجيل الدخول

```bash
openyida login
```

سيفتح هذا نافذة متصفح للمصادقة. بعد تسجيل الدخول بنجاح، يمكنك إغلاق المتصفح والعودة إلى الطرفية.

## الخطوة 3 — إنشاء تطبيق

### الخيار أ: الإنشاء بالوصف باللغة الطبيعية

```bash
openyida create-app --prompt "أنشئ نظام CRM يتضمن إدارة العملاء وتتبع المبيعات ولوحة التحليلات"
```

### الخيار ب: الإنشاء من ملف المتطلبات

```bash
openyida create-app --file requirements.md
```

## الخطوة 4 — إدارة النماذج

### عرض قائمة النماذج

```bash
openyida list-forms --app-id YOUR_APP_ID
```

### إنشاء نموذج جديد

```bash
openyida create-form --app-id YOUR_APP_ID --prompt "نموذج معلومات العميل يتضمن الاسم والشركة والهاتف والبريد الإلكتروني والملاحظات"
```

### تحديث نموذج موجود

```bash
openyida update-form --form-id YOUR_FORM_ID --prompt "أضف حقل الحالة بالخيارات: عميل محتمل، فرصة، عميل"
```

## الخطوة 5 — نشر الصفحات

### تجميع ونشر صفحة مخصصة

```bash
openyida publish-page --app-id YOUR_APP_ID --file custom-page.jsx
```

### إعداد الوصول العام

```bash
openyida set-public --page-id YOUR_PAGE_ID
```

## الخطوة 6 — تكوين الموصلات

### إنشاء موصل HTTP

```bash
openyida create-connector --app-id YOUR_APP_ID --name "API الدفع" --config connector-config.json
```

### الاستيراد من curl

```bash
openyida import-curl --app-id YOUR_APP_ID --curl 'curl -X GET "https://api.example.com/data" -H "Authorization: Bearer token"'
```
