## Berichte erstellen

### Grundlegende Verwendung

```bash
openyida create-report <appType> "Berichtsname" '<Diagrammdefinition-JSON-oder-Datei>'
```

Das dritte Argument kann entweder ein Inline-JSON-String oder ein Pfad zu einer JSON-Datei sein.

### Beispiel: Verkaufsbericht erstellen

```bash
openyida create-report APP_XXXXXX "Sales Dashboard" '{
  "charts": [
    {
      "type": "bar",
      "title": "Monthly Sales",
      "dataSource": "sales_form",
      "xField": "month",
      "yField": "amount"
    }
  ]
}'
```

### Diagrammdefinition aus Datei laden

```bash
openyida create-report APP_XXXXXX "Annual Report" ./charts-config.json
```

## Diagramme zum Bericht hinzufügen

```bash
openyida append-chart <appType> <reportId> '<Diagrammdefinition-JSON-oder-Datei>'
```

### Beispiel

```bash
openyida append-chart APP_XXXXXX REPORT_XXXXX '{
  "type": "line",
  "title": "Growth Trend",
  "dataSource": "form_sales",
  "xField": "month",
  "yField": "growth_rate"
}'
```

## Diagrammtypen

| Typ | Beschreibung | Geeignet für |
|------|-------------|----------|
| **bar** | Balkendiagramm | Vergleich von Kategorien |
| **line** | Liniendiagramm | Trends über die Zeit |
| **pie** | Kreisdiagramm | Teil-zu-Ganzes-Verhältnisse |
| **table** | Datentabelle | Detaillierte Datenansicht |
| **area** | Flächendiagramm | Kumulative Trends |
| **scatter** | Streudiagramm | Korrelationsanalyse |

---

## Diagrammdefinitionsformat

### Balkendiagramm

```json
{
  "type": "bar",
  "title": "Sales by Region",
  "dataSource": "form_uuid",
  "xField": "region",
  "yField": "amount",
  "aggregation": "sum"
}
```

### Liniendiagramm

```json
{
  "type": "line",
  "title": "Traffic Trend",
  "dataSource": "form_uuid",
  "xField": "date",
  "yField": "visitors",
  "aggregation": "sum"
}
```

### Kreisdiagramm

```json
{
  "type": "pie",
  "title": "Market Share",
  "dataSource": "form_uuid",
  "categoryField": "product",
  "valueField": "share"
}
```

### Datentabelle

```json
{
  "type": "table",
  "title": "Detail View",
  "dataSource": "form_uuid",
  "columns": [
    { "field": "name", "title": "Name" },
    { "field": "amount", "title": "Amount" },
    { "field": "date", "title": "Date" }
  ]
}
```

## Aggregationsfunktionen

| Funktion | Beschreibung |
|----------|-------------|
| **sum** | Summe der Werte |
| **avg** | Durchschnittswert |
| **count** | Anzahl der Datensätze |
| **max** | Maximalwert |
| **min** | Minimalwert |

## Vollständiges Konfigurationsbeispiel

### sales-report-charts.json

```json
{
  "charts": [
    {
      "type": "bar",
      "title": "Monthly Sales",
      "dataSource": "form_monthly_sales",
      "xField": "month",
      "yField": "sales_amount",
      "aggregation": "sum"
    },
    {
      "type": "pie",
      "title": "Product Categories",
      "dataSource": "form_sales",
      "categoryField": "product_category",
      "valueField": "sales_amount"
    },
    {
      "type": "line",
      "title": "Sales Trend",
      "dataSource": "form_daily_sales",
      "xField": "date",
      "yField": "sales_amount",
      "aggregation": "sum"
    },
    {
      "type": "table",
      "title": "Sales Detail",
      "dataSource": "form_sales",
      "columns": [
        { "field": "order_no", "title": "Order No.", "width": 150 },
        { "field": "customer_name", "title": "Customer", "width": 200 },
        { "field": "product_name", "title": "Product", "width": 200 },
        { "field": "amount", "title": "Amount", "width": 120 },
        { "field": "sales_date", "title": "Date", "width": 150 }
      ],
      "pageSize": 10
    }
  ]
}
```

Bericht erstellen:

```bash
openyida create-report APP_XXXXXX "Sales Analysis Report" ./sales-report-charts.json
```

## Best Practices

1. **Wählen Sie den passenden Diagrammtyp** - Liniendiagramme für Trends, Balkendiagramme für Vergleiche, Kreisdiagramme für Anteile
2. **Begrenzen Sie die Diagrammanzahl** - Maximal 6 Diagramme pro Bericht
3. **Verwenden Sie Datentabellen** - Ergänzen Sie Diagramme mit Tabellen zur detaillierten Datenprüfung
4. **Legen Sie angemessene Seitengrößen fest** - Für Datentabellen 10-20 Zeilen pro Seite
