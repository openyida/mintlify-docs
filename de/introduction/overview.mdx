## Übersicht

**OpenYida** ist ein CLI-Tool ohne Konfiguration, das KI-Programmierassistenten mit der Low-Code-Plattform [Yida](https://www.aliwork.com) verbindet. Installieren Sie es einmal und beschreiben Sie dann Ihre Anforderungen in natürlicher Sprache — OpenYida übernimmt automatisch: Anwendungserstellung, Formularverwaltung, Seitenveröffentlichung, Konnektor-Konfiguration, Berichtserstellung, Sekundenschnelle Bereitstellung auf Yida.

🌐 **Offizielle Website**: [https://openyida.ai](https://openyida.ai)

## Kernfunktionen

| Funktionsmodul | Beschreibung |
|---------|------|
| **Anwendungsverwaltung** | Anwendungserstellung, Import/Export, Migration und Bereitstellung |
| **Formularverwaltung** | Erstellung/Aktualisierung von Formularen, Schema-Abfrage, Berechtigungskonfiguration |
| **Seitenveröffentlichung** | Entwicklung benutzerdefinierter Seiten, Babel-Kompilierung, Ein-Klick-Veröffentlichung |
| **Prozesskonfiguration** | Erstellung von Prozessformularen, Konfiguration von Genehmigungsregeln |
| **Datenverwaltung** | Formulardatenabfragen, Prozessinstanzen, Aufgabenverwaltung |
| **Konnektoren** | HTTP-Konnektor-Erstellung, API-Integration, intelligenter curl-Import |
| **Berichte und Diagramme** | Yida-Berichtserstellung, Diagramm-Anfügung, Datenvisualisierung |
| **CDN-Verwaltung** | Bild-Upload, Cache-Aktualisierung, OSS-Konfiguration |

## Unterschiede zu anderen KI-Bau-Plattformen

| Dimension | OpenYida | Andere KI-Bau-Plattformen |
|------|----------|------------------|
| **Zielbenutzer** | Entwickler (Personen mit Programmierkenntnissen) | Fachpersonal (Nicht-Entwickler) |
| **Interaktionsweise** | Natürliche Sprache + KI-Gespräch | Visuelles Drag-and-Drop + Konfigurationspanel |
| **Ergebnis** | Yida-Anwendung (nachbearbeitbar, unterstützt vollständige Low-Code-Fähigkeiten) | Konfiguration (Ausführung in Blackbox) |
| **Bereitstellungsart** | Yida-Plattform | Bindung an SaaS-Plattform |
| **KI-Modell** | Auswahl nach Bedarf, passendstes Modell | Von Plattform festgelegt, nicht austauschbar |
| **Sicherheit und Compliance** | Yida verfügt über vollständige Sicherheits- und Compliance-Fähigkeiten | Abhängig von Plattformfähigkeiten |
