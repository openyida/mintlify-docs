## Voraussetzungen

Bevor Sie beginnen, stellen Sie sicher, dass die folgenden Abhängigkeiten installiert sind:

| Abhängigkeit | Version | Überprüfungsbefehl |
|--------------|---------|-------------------|
| [Node.js](https://nodejs.org) | ≥ 18 | `node --version` |

<Tip>
  Nur Node.js wird benötigt. Kein Python, kein Playwright, keine Build-Schritte — reine Node.js-Implementierung.
</Tip>

---

## Schritt 1 — Installation

```bash
npm install -g openyida
```

Überprüfen Sie die Installation:

```bash
openyida --version
```

---

## Schritt 2 — Bei Yida anmelden

```bash
openyida login
```

OpenYida öffnet ein Browserfenster (oder zeigt einen QR-Code) zur Authentifizierung mit Ihrem Yida/Alibaba-Konto. Die Anmeldedaten werden lokal zwischengespeichert — nur einmal anmelden.

<Info>
  Wenn Sie **Wukong** als KI-Tool verwenden, verwenden Sie das spezielle Anmelde-Flag:

  ```bash
  openyida login --wukong
  ```
</Info>

---

## Schritt 3 — Umgebung prüfen

```bash
openyida env
```

Erkennt das aktuell aktive KI-Codierungstool (Claude Code, Cursor, Wukong usw.) und zeigt Ihren Anmeldestatus an. Führen Sie diesen Befehl jederzeit aus, wenn Sie die Umgebung wechseln.

---

## Schritt 4 — Projektarbeitsbereich initialisieren

```bash
openyida copy
```

Kopiert die `project/`-Arbeitsbereichsvorlage in das aktuelle Arbeitsverzeichnis Ihres KI-Tools, sodass die KI auf Ihre App-Konfiguration und Seitenvorlagen zugreifen kann.

---

## Schritt 5 — Mit dem Bauen beginnen

Sprechen Sie direkt mit Ihrem KI-Codierungsassistenten. Hier sind einige Beispiel-Prompts zum Starten:

```
Bauen Sie mir ein IPD-System auf Yida, um den gesamten Chip-Produktions-Workflow zu verwalten
```

```
Helfen Sie mir, ein CRM mit Leads, Kunden und Geschäften aufzubauen
```

```
Erstellen Sie eine persönliche Gehaltsrechner-App
```

```
Generieren Sie ein Mini-Geburtstagsgruß-Spiel auf Yida
```

Die KI verwendet das OpenYida-Skill-Paket, um CLI-Befehle in der richtigen Reihenfolge aufzurufen: `create-app` → `create-form` / `create-page` → `publish` → `save-share-config`.

---

## Schritt 6 — App veröffentlichen

Sobald die KI die App generiert hat, führen Sie den Veröffentlichungsbefehl aus:

```bash
openyida publish
```

Oder fragen Sie die KI:
```
Veröffentlichen Sie die Seite [Seitenname]
Machen Sie die Seite öffentlich zugänglich
```

---

## Diagnose ausführen

Wenn Probleme auftreten, führen Sie das integrierte Diagnosetool aus:

```bash
openyida doctor
```

Es überprüft Ihre Node.js-Version, den Anmeldestatus und die Umgebungskonfiguration und kann mit `--fix` häufige Probleme automatisch beheben.

---

## OpenClaw-Integration

Wenn Sie OpenClaw verwenden, installieren Sie direkt das OpenYida-Skill-Paket:

```bash
npx clawhub@latest install nicky1108/yida-app
```

Verwenden Sie es dann über [yida-app auf ClawHub](https://clawhub.ai/nicky1108/yida-app).
