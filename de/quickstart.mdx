## Systemanforderungen

Bevor Sie beginnen, stellen Sie sicher, dass die folgenden Abhängigkeiten installiert sind:

| Abhängigkeit | Version | Überprüfungsbefehl |
|------|------|----------|
| [Node.js](https://nodejs.org) | ≥ 18 | `node --version` |

<Tip>
  Das ist alles, was Sie brauchen. OpenYida ist eine reine Node.js-Implementierung — kein Python, kein Playwright, keine Build-Schritte erforderlich.
</Tip>

---

## Schritt 1 — Installation

```bash
npm install -g openyida
```

Überprüfen Sie die Installation:

```bash
openyida --version
```

---

## Schritt 2 — Bei Yida anmelden

```bash
openyida login
```

OpenYida öffnet ein Browserfenster (oder zeigt einen QR-Code) an, damit Sie sich mit Ihrem Yida/Alibaba-Konto authentifizieren können. Die Anmeldedaten werden lokal zwischengespeichert — Sie müssen dies nur einmal tun.

<Info>
  Wenn Sie **Wukong** als KI-Tool verwenden, verwenden Sie das spezielle Login-Flag:

  ```bash
  openyida login --wukong
  ```
</Info>

---

## Schritt 3 — Umgebung überprüfen

```bash
openyida env
```

Dies erkennt das aktuell aktive KI-Programmierwerkzeug (Claude Code, Cursor, Wukong usw.) und zeigt Ihren aktuellen Anmeldestatus an. Führen Sie diesen Befehl jedes Mal aus, wenn Sie die Umgebung wechseln.

---

## Schritt 4 — Projekt-Arbeitsbereich initialisieren

```bash
openyida copy
```

Dies kopiert die `project/`-Arbeitsbereichsvorlage in das Arbeitsverzeichnis Ihres aktuellen KI-Tools, sodass die KI auf Ihre Anwendungskonfiguration und Seitenvorlagen zugreifen kann.

---

## Schritt 5 — Mit dem Bauen beginnen

Sprechen Sie jetzt mit Ihrem KI-Programmierassistenten. Hier sind einige Prompts zum Starten:

```
Baue ein IPD-System auf Yida, das den gesamten Chip-Produktions-Workflow verwaltet
```

```
Baue ein CRM-System mit Leads, Kunden und Transaktionen
```

```
Erstelle eine Gehaltsrechner-Anwendung
```

```
Generiere ein Mini-Geburtstagsgruß-Spiel auf Yida
```

Die KI verwendet das OpenYida-Skill-Paket, um die CLI-Befehle in der richtigen Reihenfolge aufzurufen: `create-app` → `create-form` / `create-page` → `publish` → `save-share-config`.

---

## Schritt 6 — Anwendung veröffentlichen

Nachdem die KI die Anwendung erstellt hat, veröffentlichen Sie sie mit:

```bash
openyida publish
```

Oder fragen Sie die KI:
```
Veröffentliche die Seite [Seitenname]
Mache die Seite öffentlich zugänglich
```

---

## Diagnose ausführen

Wenn etwas nicht stimmt, führen Sie das integrierte Diagnose-Tool aus:

```bash
openyida doctor
```

Es überprüft Ihre Node.js-Version, den Anmeldestatus und die Umgebungskonfiguration und kann häufige Probleme mit `--fix` automatisch beheben.

---

## OpenClaw-Integration

Wenn Sie OpenClaw verwenden, installieren Sie direkt das OpenYida-Skill-Paket:

```bash
npx clawhub@latest install nicky1108/yida-app
```

Verwenden Sie es dann über [yida-app auf ClawHub](https://clawhub.ai/nicky1108/yida-app).
