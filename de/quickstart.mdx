## Voraussetzungen

Stellen Sie vor dem Beginn sicher, dass die folgenden Abhängigkeiten installiert sind:

| Abhängigkeit | Version | Überprüfungsbefehl |
|------|------|----------|
| [Node.js](https://nodejs.org) | ≥ 18 | `node --version` |

<Tip>
  Das ist alles, was Sie brauchen. OpenYida ist eine reine Node.js-Implementierung — kein Python, kein Playwright, keine Build-Schritte.
</Tip>

---

## Schritt 1 — Installation

```bash
npm install -g openyida
```

Überprüfen Sie die Installation:

```bash
openyida --version
```

---

## Schritt 2 — Bei Yida anmelden

```bash
openyida login
```

OpenYida öffnet ein Browserfenster (oder zeigt einen QR-Code an), damit Sie sich mit Ihrem Yida/Alibaba-Konto authentifizieren können. Die Anmeldeinformationen werden lokal zwischengespeichert — Sie müssen dies nur einmal tun.

<Info>
  Wenn Sie **Wukong** als KI-Tool verwenden, verwenden Sie das spezielle Login-Flag:

  ```bash
  openyida login --wukong
  ```
</Info>

---

## Schritt 3 — Umgebung überprüfen

```bash
openyida env
```

Dies erkennt das aktuell aktive KI-Coding-Tool (Claude Code, Cursor, Wukong usw.) und zeigt Ihren aktuellen Anmeldestatus an. Führen Sie diesen Befehl jederzeit aus, wenn Sie die Umgebung wechseln.

---

## Schritt 4 — Projektarbeitsbereich initialisieren

```bash
openyida copy
```

Dies kopiert die `project/`-Arbeitsbereichsvorlage in das Arbeitsverzeichnis Ihres aktuellen KI-Tools, sodass die KI auf Ihre App-Konfiguration und Seitenvorlagen zugreifen kann.

---

## Schritt 5 — Mit dem Erstellen beginnen

Sprechen Sie jetzt einfach mit Ihrem KI-Coding-Assistenten. Hier sind einige Starter-Prompts:

```
Erstellen Sie mir ein IPD-System auf Yida, um den vollständigen Chip-Produktions-Workflow zu verwalten
```

```
Helfen Sie mir, ein CRM-System mit Leads, Kontakten und Deals zu erstellen
```

```
Erstellen Sie eine persönliche Gehaltsrechner-App
```

```
Generieren Sie ein Geburtstagsgruß-Mini-Spiel auf Yida
```

Die KI verwendet das OpenYida-Skill-Pack, um die richtigen CLI-Befehle in der richtigen Reihenfolge aufzurufen: `create-app` → `create-form` / `create-page` → `publish` → `save-share-config`.

---

## Schritt 6 — App veröffentlichen

Nachdem die KI die App generiert hat, verwenden Sie den folgenden Befehl zum Veröffentlichen:

```bash
openyida publish
```

Oder fragen Sie die KI:
```
Veröffentlichen Sie die Seite [Seitenname]
Machen Sie die Seite öffentlich zugänglich
```

---

## Diagnose ausführen

Wenn etwas nicht stimmt, führen Sie das integrierte Diagnosetool aus:

```bash
openyida doctor
```

Es überprüft Ihre Node.js-Version, den Anmeldestatus und die Umgebungskonfiguration und kann häufige Probleme automatisch mit `--fix` beheben.

---

## OpenClaw-Integration

Wenn Sie OpenClaw verwenden, installieren Sie das OpenYida-Skill-Pack direkt:

```bash
npx clawhub@latest install nicky1108/yida-app
```

Verwenden Sie es dann über [yida-app auf ClawhHub](https://clawhub.ai/nicky1108/yida-app).
