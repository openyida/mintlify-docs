---

## Environment & Authentication

```bash
# Environment check
openyida env

# Login
openyida login [--qr]              # --qr for terminal QR code
openyida login --wukong            # Wukong environment

# Logout
openyida logout

# Auth management
openyida auth status               # Check login status
openyida auth login                # Perform login
openyida auth refresh              # Refresh session
openyida auth logout               # Logout

# Organization management
openyida org list                  # List accessible organizations
openyida org switch --corp-id <id> # Switch organization

# Initialize workspace
openyida copy [--force]            # Copy project directory

# Diagnostic tools
openyida doctor [--fix]            # Environment diagnosis
openyida doctor --monitor          # Health monitoring
openyida doctor --report <format>  # Generate report (json|markdown|html)
```

## Application Management

```bash
# Create application
openyida create-app "<name>" [desc] [icon] [color] [themeColor]

# List applications
openyida list-apps

# Import/Export
openyida import --file <path>      # Import from JSON
openyida export --app <id>         # Export to JSON

# Deploy
openyida deploy --env <env>        # Deploy to environment
```

## Form Management

```bash
# Create form
openyida create-form "<name>" [--app <appId>] [--folder <folder>]

# Create process form
openyida create-form "<name>" --process [--app <appId>]

# Get form schema
openyida get-form-schema --form <formUuid>

# Update form
openyida update-form --form <formUuid> --schema <file>
```

### Supported field types

The `create-form` command only accepts the following field types. Using an unsupported type returns an error.

#### Base fields

Built-in fields that require no additional plugins.

| Type | Description |
|------|-------------|
| `TextField` | Single-line text |
| `TextareaField` | Multi-line text |
| `NumberField` | Number |
| `RateField` | Rating |
| `DateField` | Date |
| `RadioField` | Radio buttons |
| `SelectField` | Dropdown (single select) |
| `CheckboxField` | Checkboxes |
| `MultiSelectField` | Dropdown (multi-select) |
| `CascadeSelectField` | Cascading select |
| `CascadeDateField` | Cascading date |
| `EmployeeField` | Member selector |
| `DepartmentSelectField` | Department selector |
| `AssociationFormField` | Associated form |
| `AssociationQuery` | Association query |
| `AttachmentField` | Attachment |
| `ImageField` | Image |
| `AddressField` | Address |
| `CountrySelectField` | Country/region selector |
| `TableField` | Sub-form / detail table |
| `SerialNumberField` | Serial number |

#### Advanced fields

Built-in fields with extended functionality. Some features may require activation.

| Type | Description |
|------|-------------|
| `EditorField` | Rich text editor |
| `DigitalSignatureField` | Digital signature |
| `LocationField` | GPS location |

#### Plugin fields

These fields require installing the corresponding plugin from the plugin center before use.

| Type | Required plugin |
|------|-----------------|
| `TimeZoneDateField` | Internationalization timezone component |
| `CC_AIButtonField` | AI component |
| `CC_PG_ESignField` | Smart finance component |
| `CC_TeambitionProjectLinksField` | Teambition plugin |
| `CC_TeambitionTaskLinksField` | Teambition plugin |
| `CC_MoneyField` | Smart finance component |
| `CC_PhoneNumberField` | Extension component |
| `CC_ChineseIdField` | Extension component |
| `YidaDingtalkAgentField` | DingTalk agent plugin |

<Warning>
Plugin fields will fail if the required plugin is not installed in your organization. Install plugins from the plugin center before using these field types.
</Warning>

<Warning>
Field types such as `MobileField` and `EmailField` are not supported and will be rejected. Only use the types listed above.
</Warning>

## Page Management

```bash
# Create custom page
openyida create-page <appType> "<name>" [--datasource <jsonOrFile>]

# Publish page
openyida publish --page <id> [--env <env>]

# Share page
openyida share --page <id> [--public] [--password <pwd>]
```

## Data Management

```bash
# Query form data
openyida data query --form <formUuid> [--where <conditions>] [--limit <n>]

# Create data entry
openyida data create --form <formUuid> --data '<json>'

# Update data
openyida data update --form <formUuid> --id <id> --data '<json>'

# Delete data
openyida data delete --form <formUuid> --id <id>

# Export data
openyida data export --form <formUuid> --format <csv|json|excel>
```

## Connector Management

```bash
# Create HTTP connector
openyida create-connector "<name>" --url <url> [--method <method>]

# Import from curl
openyida create-connector --curl '<curl-command>'

# Test connector
openyida test-connector --id <connectorId>

# List connectors
openyida list-connectors [--app <appId>]
```

## Report & Charts

```bash
# Create report
openyida create-report "<name>" [--app <appId>] [--type <type>]

openyida add-chart --report <id> --chart '<config>'
```

## Global Options

| Option | Description |
|--------|-------------|
| `--app <appId>` | Specify target application |
| `--env <env>` | Target environment (dev/prod) |
| `--format <format>` | Output format (json/table) |
| `--verbose` | Show detailed logs |
| `--dry-run` | Preview changes without executing |
