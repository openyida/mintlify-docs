## Prerequisites

- Node.js ≥ 18
- A Yida (宜搭) account with organization access
- An AI coding tool (Claude Code, Cursor, VS Code, etc.)

---

## Installation

Install OpenYida globally via npm:

```bash
npm install -g openyida
```

Verify installation:

```bash
openyida --version
```

---

## Login

Authenticate with your Yida account:

```bash
openyida login
```

This will open a browser for OAuth authentication. For DingTalk/Wukong environments:

```bash
openyida login --wukong
```

---

## Create Your First App

### Step 1: Create Application

```bash
openyida create-app --name "My First App"
```

Note the returned `appId` — you'll need it for subsequent commands.

### Step 2: Create a Form

```bash
openyida create-form \
  --page "Contact Form" \
  --schema '{
    "fields": [
      {"name": "name", "type": "TextField", "label": "Name", "required": true},
      {"name": "email", "type": "TextField", "label": "Email", "required": true},
      {"name": "message", "type": "TextareaField", "label": "Message"}
    ]
  }'
```

### Step 3: Publish

```bash
openyida publish
```

Your form is now live on Yida!

## Using with AI

The real power comes from AI integration. Here's how to use OpenYida with Claude Code:

1. **Install Claude Code** following [their documentation](https://docs.anthropic.com/en/docs/agents-and-tools/claude-code/overview)

2. **Start a session** in your project directory:
   ```bash
   claude
   ```

3. **Describe what you want**:
   ```
   Create a simple CRM with leads, contacts, and deals.
   Each lead should have: name, company, phone, email, status.
   Status should be a dropdown: New, Contacted, Qualified, Lost.
   ```

4. **Claude will use OpenYida** to build the application for you
