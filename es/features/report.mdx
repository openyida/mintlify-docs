## Crear informes

### Uso básico

```bash
openyida create-report <appType> "Nombre del informe" '<JSON-definición-o-archivo>'
```

El tercer argumento puede ser una cadena JSON en línea o una ruta a un archivo JSON.

### Ejemplo: crear un informe de ventas

```bash
openyida create-report APP_XXXXXX "Sales Dashboard" '{
  "charts": [
    {
      "type": "bar",
      "title": "Monthly Sales",
      "dataSource": "sales_form",
      "xField": "month",
      "yField": "amount"
    }
  ]
}'
```

### Cargar definición de gráfico desde archivo

```bash
openyida create-report APP_XXXXXX "Annual Report" ./charts-config.json
```

---

## Agregar gráficos a un informe

```bash
openyida append-chart <appType> <reportId> '<JSON-definición-o-archivo>'
```

### Ejemplo

```bash
openyida append-chart APP_XXXXXX REPORT_XXXXX '{
  "type": "line",
  "title": "Growth Trend",
  "dataSource": "form_sales",
  "xField": "month",
  "yField": "growth_rate"
}'
```

## Tipos de gráficos

| Tipo | Descripción | Ideal para |
|------|-------------|----------|
| **bar** | Gráfico de barras | Comparar categorías |
| **line** | Gráfico de líneas | Tendencias en el tiempo |
| **pie** | Gráfico circular | Relaciones parte-todo |
| **table** | Tabla de datos | Vista detallada de datos |
| **area** | Gráfico de área | Tendencias acumulativas |
| **scatter** | Gráfico de dispersión | Análisis de correlación |

---

## Formato de definición de gráfico

### Gráfico de barras

```json
{
  "type": "bar",
  "title": "Sales by Region",
  "dataSource": "form_uuid",
  "xField": "region",
  "yField": "amount",
  "aggregation": "sum"
}
```

### Gráfico de líneas

```json
{
  "type": "line",
  "title": "Traffic Trend",
  "dataSource": "form_uuid",
  "xField": "date",
  "yField": "visitors",
  "aggregation": "sum"
}
```

### Gráfico circular

```json
{
  "type": "pie",
  "title": "Market Share",
  "dataSource": "form_uuid",
  "categoryField": "product",
  "valueField": "share"
}
```

### Tabla de datos

```json
{
  "type": "table",
  "title": "Detail View",
  "dataSource": "form_uuid",
  "columns": [
    { "field": "name", "title": "Name" },
    { "field": "amount", "title": "Amount" },
    { "field": "date", "title": "Date" }
  ]
}
```

## Funciones de agregación

| Función | Descripción |
|----------|-------------|
| **sum** | Suma de valores |
| **avg** | Valor promedio |
| **count** | Conteo de registros |
| **max** | Valor máximo |
| **min** | Valor mínimo |

## Ejemplo de configuración completa

### sales-report-charts.json

```json
{
  "charts": [
    {
      "type": "bar",
      "title": "Monthly Sales",
      "dataSource": "form_monthly_sales",
      "xField": "month",
      "yField": "sales_amount",
      "aggregation": "sum"
    },
    {
      "type": "pie",
      "title": "Product Categories",
      "dataSource": "form_sales",
      "categoryField": "product_category",
      "valueField": "sales_amount"
    },
    {
      "type": "line",
      "title": "Sales Trend",
      "dataSource": "form_daily_sales",
      "xField": "date",
      "yField": "sales_amount",
      "aggregation": "sum"
    },
    {
      "type": "table",
      "title": "Sales Detail",
      "dataSource": "form_sales",
      "columns": [
        { "field": "order_no", "title": "Order No.", "width": 150 },
        { "field": "customer_name", "title": "Customer", "width": 200 },
        { "field": "product_name", "title": "Product", "width": 200 },
        { "field": "amount", "title": "Amount", "width": 120 },
        { "field": "sales_date", "title": "Date", "width": 150 }
      ],
      "pageSize": 10
    }
  ]
}
```

Crear el informe:

```bash
openyida create-report APP_XXXXXX "Sales Analysis Report" ./sales-report-charts.json
```

## Mejores prácticas

1. **Elige el tipo de gráfico apropiado** - Gráficos de líneas para tendencias, de barras para comparaciones, circulares para proporciones
2. **Limita la cantidad de gráficos** - Mantén los informes con 6 gráficos o menos
3. **Incluye tablas de datos** - Complementa los gráficos con tablas para inspección detallada
4. **Configura tamaños de página razonables** - Para tablas de datos, usa 10-20 filas por página
