## 🏢 Sistemas empresariales — IPD / CRM

Describe tus requisitos en una frase, la IA genera automáticamente un sistema de negocio completo con múltiples formularios.

**Ejemplo de prompt:**
```
Crea un sistema IPD en Yida para gestionar todo el proceso de producción de chips
```

## 🏮 Escenarios operativos — Adivinanzas

La IA genera imágenes de adivinanzas, el usuario adivina la respuesta, y la IA da pistas humorísticas si la respuesta es incorrecta.

**Ejemplo de prompt:**
```
Crea un juego de adivinanzas para el Festival de los Faroles, la IA genera imágenes de adivinanzas, el usuario envía una respuesta, y la IA responde de manera humorística si es incorrecta
```
