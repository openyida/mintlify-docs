## Descripción general

**OpenYida** es una herramienta CLI sin configuración que conecta asistentes de codificación IA con la plataforma low-code [Yida](https://www.aliwork.com). Instale una vez, luego describa sus requisitos en lenguaje natural — OpenYida maneja automáticamente: creación de aplicaciones, gestión de formularios, publicación de páginas, configuración de conectores, generación de informes, y despliegue en Yida en segundos.

🌐 **Sitio web oficial**: [https://openyida.ai](https://openyida.ai)

## Funciones principales

| Módulo de función | Descripción |
|---------|------|
| **Gestión de aplicaciones** | Creación de aplicaciones, importación/exportación, migración y despliegue |
| **Gestión de formularios** | Creación/actualización de formularios, obtención de esquema, configuración de permisos |
| **Publicación de páginas** | Desarrollo de páginas personalizadas, compilación Babel, publicación con un clic |
| **Configuración de flujos** | Creación de formularios de flujo, configuración de reglas de aprobación |
| **Gestión de datos** | Consulta de datos de formularios, instancias de flujo, gestión de tareas |
| **Conectores** | Creación de conectores HTTP, integración de API, importación inteligente curl |
| **Informes y gráficos** | Creación de informes Yida, adición de gráficos, visualización de datos |
| **Gestión CDN** | Carga de imágenes, actualización de caché, configuración OSS |

## Diferencias con otras plataformas de construcción IA

| Dimensión | OpenYida | Otras plataformas de construcción IA |
|------|----------|------------------|
| **Usuarios objetivo** | Desarrolladores (personas que entienden código) | Usuarios de negocio (no desarrolladores) |
| **Modo de interacción** | Lenguaje natural + diálogo IA | Arrastrar y soltar visual + paneles de configuración |
| **Entregables** | Aplicación Yida (editable posteriormente, soporte completo de low-code) | Configuración (ejecución de caja negra) |
| **Modo de despliegue** | Plataforma Yida | Vinculación de plataforma SaaS |
| **Modelo IA** | Selección según necesidad, elegir el mejor modelo | Especificado por plataforma, no modificable |
| **Seguridad y cumplimiento** | Yida tiene capacidades completas de seguridad y cumplimiento | Dependiente de capacidades de plataforma |
