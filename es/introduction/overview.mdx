## ¿Qué es OpenYida?

OpenYida es una interfaz de línea de comandos que conecta agentes de IA con la plataforma low-code [Yida](https://www.aliwork.com). Permite crear, gestionar y desplegar aplicaciones empresariales usando prompts en lenguaje natural.

## Inicio rápido

```bash
# Instalación
npm install -g openyida

# Inicio de sesión
openyida login

# Crear app
openyida create-app --name "Mi CRM"
```
