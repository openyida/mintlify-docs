## Requisitos previos

Antes de comenzar, asegúrese de tener instaladas las siguientes dependencias:

| Dependencia | Versión | Comando de verificación |
|-------------|---------|------------------------|
| [Node.js](https://nodejs.org) | ≥ 18 | `node --version` |

<Tip>
  Solo se necesita Node.js. No se requiere Python, Playwright ni pasos de compilación — implementación pura de Node.js.
</Tip>

---

## Paso 1 — Instalación

```bash
npm install -g openyida
```

Verifique la instalación:

```bash
openyida --version
```

---

## Paso 2 — Iniciar sesión en Yida

```bash
openyida login
```

OpenYida abrirá una ventana del navegador (o mostrará un código QR) para que se autentique con su cuenta de Yida/Alibaba. Las credenciales se almacenan en caché localmente — solo una vez.

<Info>
  Si utiliza **Wukong** como herramienta de IA, use la bandera de inicio de sesión especial:

  ```bash
  openyida login --wukong
  ```
</Info>

---

## Paso 3 — Verificar el entorno

```bash
openyida env
```

Detecta la herramienta de codificación con IA actualmente activa (Claude Code, Cursor, Wukong, etc.) y muestra su estado de inicio de sesión. Ejecute este comando en cualquier momento cuando cambie de entorno.

---

## Paso 4 — Inicializar el espacio de trabajo del proyecto

```bash
openyida copy
```

Copia la plantilla de espacio de trabajo `project/` al directorio de trabajo actual de su herramienta de IA, permitiendo que la IA acceda a su configuración de aplicación y plantillas de página.

---

## Paso 5 — Comenzar a construir

Hable directamente con su asistente de codificación con IA. Aquí hay algunos ejemplos de prompts para comenzar:

```
Constrúyame un sistema IPD en Yida para gestionar todo el flujo de trabajo de producción de chips
```

```
Ayúdame a construir un CRM con leads, clientes y transacciones
```

```
Crea una aplicación de calculadora de salario personal
```

```
Genera un mini juego de felicitación de cumpleaños en Yida
```

La IA utilizará el paquete de habilidades de OpenYida para llamar a los comandos CLI en el orden correcto: `create-app` → `create-form` / `create-page` → `publish` → `save-share-config`.

---

## Paso 6 — Publicar la aplicación

Una vez que la IA haya generado la aplicación, ejecute el comando de publicación:

```bash
openyida publish
```

O pregúntele a la IA:
```
Publica la página [nombre de la página]
Haz que la página sea de acceso público
```

---

## Ejecutar diagnóstico

Si tiene problemas, ejecute la herramienta de diagnóstico integrada:

```bash
openyida doctor
```

Verifica su versión de Node.js, estado de inicio de sesión y configuración del entorno, y puede usar `--fix` para reparar automáticamente problemas comunes.

---

## Integración con OpenClaw

Si utiliza OpenClaw, instale directamente el paquete de habilidades de OpenYida:

```bash
npx clawhub@latest install nicky1108/yida-app
```

Luego úselo desde [yida-app en ClawHub](https://clawhub.ai/nicky1108/yida-app).
