## Requisitos del entorno

Antes de comenzar, asegúrate de tener instaladas las siguientes dependencias:

| Dependencia | Versión | Comando de verificación |
|------|------|----------|
| [Node.js](https://nodejs.org) | ≥ 18 | `node --version` |

<Tip>
  Esto es todo lo que necesitas. OpenYida es una implementación pura de Node.js — no se necesita Python, Playwright ni pasos de compilación.
</Tip>

---

## Paso 1 — Instalación

```bash
npm install -g openyida
```

Verifica la instalación:

```bash
openyida --version
```

---

## Paso 2 — Iniciar sesión en Yida

```bash
openyida login
```

OpenYida abrirá una ventana del navegador (o mostrará un código QR) para permitirte autenticarte con tu cuenta de Yida/Alibaba. Las credenciales se almacenan en caché localmente — solo necesitas hacerlo una vez.

<Info>
  Si usas **Wukong** como herramienta de IA, usa la bandera de inicio de sesión especial:

  ```bash
  openyida login --wukong
  ```
</Info>

---

## Paso 3 — Verificar el entorno

```bash
openyida env
```

Esto detecta la herramienta de programación de IA actualmente activa (Claude Code, Cursor, Wukong, etc.) y muestra tu estado de inicio de sesión actual. Ejecuta este comando cada vez que cambies de entorno.

---

## Paso 4 — Inicializar el espacio de trabajo del proyecto

```bash
openyida copy
```

Esto copia la plantilla de espacio de trabajo `project/` al directorio de trabajo de tu herramienta de IA actual, permitiendo que la IA acceda a tu configuración de aplicación y plantillas de página.

---

## Paso 5 — Comenzar a construir

Ahora habla con tu asistente de programación de IA. Aquí hay algunos prompts para comenzar:

```
Construye un sistema IPD en Yida para gestionar el flujo de trabajo completo de producción de chips
```

```
Construye un sistema CRM con leads, clientes y transacciones
```

```
Crea una aplicación de calculadora de salario personal
```

```
Genera un mini juego de felicitación de cumpleaños en Yida
```

La IA utilizará el paquete de habilidades de OpenYida para llamar a los comandos CLI en el orden correcto: `create-app` → `create-form` / `create-page` → `publish` → `save-share-config`.

---

## Paso 6 — Publicar la aplicación

Después de que la IA haya creado la aplicación, publícala con:

```bash
openyida publish
```

O pregúntale a la IA:
```
Publica la página [nombre de la página]
Haz que la página sea accesible públicamente
```

---

## Ejecutar diagnósticos

Si algo parece incorrecto, ejecuta la herramienta de diagnóstico integrada:

```bash
openyida doctor
```

Verifica tu versión de Node.js, el estado de inicio de sesión y la configuración del entorno, y puede solucionar automáticamente problemas comunes con `--fix`.

## Integración con OpenClaw

Si usas OpenClaw, instala directamente el paquete de habilidades de OpenYida:

```bash
npx clawhub@latest install nicky1108/yida-app
```
