## Créer des rapports

### Utilisation de base

```bash
openyida create-report <appType> "Nom du rapport" '<définition-JSON-ou-fichier>'
```

Le troisième argument peut être une chaîne JSON en ligne ou un chemin vers un fichier JSON.

### Exemple : créer un rapport de ventes

```bash
openyida create-report APP_XXXXXX "Sales Dashboard" '{
  "charts": [
    {
      "type": "bar",
      "title": "Monthly Sales",
      "dataSource": "sales_form",
      "xField": "month",
      "yField": "amount"
    }
  ]
}'
```

### Charger la définition du graphique à partir d'un fichier

```bash
openyida create-report APP_XXXXXX "Annual Report" ./charts-config.json
```

## Ajouter des graphiques à un rapport

```bash
openyida append-chart <appType> <reportId> '<définition-JSON-ou-fichier>'
```

### Exemple

```bash
openyida append-chart APP_XXXXXX REPORT_XXXXX '{
  "type": "line",
  "title": "Growth Trend",
  "dataSource": "form_sales",
  "xField": "month",
  "yField": "growth_rate"
}'
```

## Types de graphiques

| Type | Description | Idéal pour |
|------|-------------|----------|
| **bar** | Graphique à barres | Comparer des catégories |
| **line** | Graphique linéaire | Tendances dans le temps |
| **pie** | Graphique circulaire | Relations partie-tout |
| **table** | Tableau de données | Vue détaillée des données |
| **area** | Graphique de zone | Tendances cumulatives |
| **scatter** | Graphique de dispersion | Analyse de corrélation |

## Format de définition du graphique

### Graphique à barres

```json
{
  "type": "bar",
  "title": "Sales by Region",
  "dataSource": "form_uuid",
  "xField": "region",
  "yField": "amount",
  "aggregation": "sum"
}
```

### Graphique linéaire

```json
{
  "type": "line",
  "title": "Traffic Trend",
  "dataSource": "form_uuid",
  "xField": "date",
  "yField": "visitors",
  "aggregation": "sum"
}
```

### Graphique circulaire

```json
{
  "type": "pie",
  "title": "Market Share",
  "dataSource": "form_uuid",
  "categoryField": "product",
  "valueField": "share"
}
```

### Tableau de données

```json
{
  "type": "table",
  "title": "Detail View",
  "dataSource": "form_uuid",
  "columns": [
    { "field": "name", "title": "Name" },
    { "field": "amount", "title": "Amount" },
    { "field": "date", "title": "Date" }
  ]
}
```

---

## Fonctions d'agrégation

| Fonction | Description |
|----------|-------------|
| **sum** | Somme des valeurs |
| **avg** | Valeur moyenne |
| **count** | Nombre d'enregistrements |
| **max** | Valeur maximale |
| **min** | Valeur minimale |

## Exemple de configuration complet

### sales-report-charts.json

```json
{
  "charts": [
    {
      "type": "bar",
      "title": "Monthly Sales",
      "dataSource": "form_monthly_sales",
      "xField": "month",
      "yField": "sales_amount",
      "aggregation": "sum"
    },
    {
      "type": "pie",
      "title": "Product Categories",
      "dataSource": "form_sales",
      "categoryField": "product_category",
      "valueField": "sales_amount"
    },
    {
      "type": "line",
      "title": "Sales Trend",
      "dataSource": "form_daily_sales",
      "xField": "date",
      "yField": "sales_amount",
      "aggregation": "sum"
    },
    {
      "type": "table",
      "title": "Sales Detail",
      "dataSource": "form_sales",
      "columns": [
        { "field": "order_no", "title": "Order No.", "width": 150 },
        { "field": "customer_name", "title": "Customer", "width": 200 },
        { "field": "product_name", "title": "Product", "width": 200 },
        { "field": "amount", "title": "Amount", "width": 120 },
        { "field": "sales_date", "title": "Date", "width": 150 }
      ],
      "pageSize": 10
    }
  ]
}
```

Créer le rapport :

```bash
openyida create-report APP_XXXXXX "Sales Analysis Report" ./sales-report-charts.json
```

## Meilleures pratiques

1. **Choisissez le type de graphique approprié** - Graphiques linéaires pour les tendances, à barres pour les comparaisons, circulaires pour les proportions
2. **Limitez le nombre de graphiques** - Gardez 6 graphiques ou moins par rapport
3. **Incluez des tableaux de données** - Associez des tableaux aux graphiques pour une inspection détaillée
4. **Définissez des tailles de page raisonnables** - Pour les tableaux de données, utilisez 10-20 lignes par page
