## Créer un rapport

### Utilisation de base

```bash
openyida create-report <appType> "Nom du rapport" '<définition du graphique JSON>'
```

### Exemple : Créer un rapport de ventes

```bash
openyida create-report APP_XXXXXX "Rapport des ventes mensuelles" '{
  "charts": [
    {
      "type": "bar",
      "title": "Ventes mensuelles",
      "dataSource": "form_sales",
      "xField": "month",
      "yField": "amount"
    },
    {
      "type": "pie",
      "title": "Répartition des ventes par produit",
      "dataSource": "form_sales",
      "categoryField": "product_type",
      "valueField": "amount"
    }
  ]
}'
```

### Charger la définition du graphique à partir d'un fichier

```bash
openyida create-report APP_XXXXXX "Rapport statistique annuel" ./charts-config.json
```

## Format de définition du graphique

### Graphique à barres

```json
{
  "type": "bar",
  "title": "Titre du graphique à barres",
  "dataSource": "form_uuid",
  "xField": "category_field",
  "yField": "value_field",
  "aggregation": "sum"
}
```

### Graphique linéaire

```json
{
  "type": "line",
  "title": "Titre du graphique de tendance",
  "dataSource": "form_uuid",
  "xField": "date_field",
  "yField": "value_field",
  "aggregation": "sum"
}
```

### Graphique circulaire

```json
{
  "type": "pie",
  "title": "Titre du graphique de proportion",
  "dataSource": "form_uuid",
  "categoryField": "category_field",
  "valueField": "value_field"
}
```

### Tableau de données

```json
{
  "type": "table",
  "title": "Détails des données",
  "dataSource": "form_uuid",
  "columns": [
    { "field": "name", "title": "Nom" },
    { "field": "amount", "title": "Montant" },
    { "field": "date", "title": "Date" }
  ]
}
```

## Ajouter un graphique au rapport

```bash
openyida append-chart <appType> <reportId> '<définition du graphique JSON>'
```

### Exemple

```bash
openyida append-chart APP_XXXXXX REPORT_XXXXX '{
  "type": "line",
  "title": "Tendance de croissance",
  "dataSource": "form_sales",
  "xField": "month",
  "yField": "growth_rate"
}'
```

## Exemple de configuration complet

### sales-report-charts.json

```json
{
  "charts": [
    {
      "type": "bar",
      "title": "Statistiques des ventes mensuelles",
      "dataSource": "form_monthly_sales",
      "xField": "month",
      "yField": "sales_amount",
      "aggregation": "sum",
      "color": "#0066FF"
    },
    {
      "type": "pie",
      "title": "Répartition des catégories de produits",
      "dataSource": "form_sales",
      "categoryField": "product_category",
      "valueField": "sales_amount",
      "showLegend": true
    },
    {
      "type": "line",
      "title": "Tendance des ventes",
      "dataSource": "form_daily_sales",
      "xField": "date",
      "yField": "sales_amount",
      "aggregation": "sum",
      "smooth": true
    },
    {
      "type": "table",
      "title": "Détails des ventes",
      "dataSource": "form_sales",
      "columns": [
        { "field": "order_no", "title": "Numéro de commande", "width": 150 },
        { "field": "customer_name", "title": "Nom du client", "width": 200 },
        { "field": "product_name", "title": "Nom du produit", "width": 200 },
        { "field": "amount", "title": "Montant", "width": 120, "align": "right" },
        { "field": "sales_date", "title": "Date de vente", "width": 150 }
      ],
      "pageSize": 10
    }
  ]
}
```

Créer le rapport :

```bash
openyida create-report APP_XXXXXX "Rapport d'analyse des ventes" ./sales-report-charts.json
```

## Référence des types de graphiques

| Type | Description | Champs requis |
|------|------|----------|
| `bar` | Graphique à barres | xField, yField |
| `line` | Graphique linéaire | xField, yField |
| `pie` | Graphique circulaire | categoryField, valueField |
| `table` | Tableau de données | columns |
| `area` | Graphique de zone | xField, yField |
| `scatter` | Graphique de dispersion | xField, yField |

---

## Fonctions d'agrégation

| Fonction | Description |
|------|------|
| `sum` | Somme |
| `avg` | Moyenne |
| `count` | Compte |
| `max` | Maximum |
| `min` | Minimum |

---

## Meilleures pratiques

1. **Choisissez raisonnablement le type de graphique** - Tendance avec graphique linéaire, comparaison avec graphique à barres, proportion avec graphique circulaire
2. **Contrôlez le nombre de graphiques** - Il est recommandé de ne pas dépasser 6 graphiques par rapport
3. **Utilisez des tableaux de données** - Complétez les graphiques avec des tableaux pour afficher des données détaillées, facilitant la consultation des valeurs originales
4. **Définissez une pagination appropriée** - Pour les tableaux de données, il est recommandé d'afficher 10-20 éléments par page
