## 🏢 Systèmes d'entreprise — IPD / CRM

Décrivez vos besoins en une phrase, l'IA génère automatiquement un système métier complet avec plusieurs formulaires.

**Exemple de prompt :**
```
Crée un système IPD sur Yida pour gérer tout le processus de production de puces
```

## 🏮 Scénarios opérationnels — Devinettes

L'IA génère des images de devinettes, l'utilisateur devine la réponse, et l'IA donne des indices humoristiques si la réponse est incorrecte.

**Exemple de prompt :**
```
Crée un jeu de devinettes pour le Festival des Lanternes, l'IA génère des images de devinettes, l'utilisateur soumet une réponse, et l'IA répond de manière humoristique si c'est incorrect
```
