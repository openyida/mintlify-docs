## Qu'est-ce qu'OpenYida ?

OpenYida est une interface en ligne de commande qui fait le pont entre les agents IA et la plateforme low-code [Yida](https://www.aliwork.com). Il permet de créer, gérer et déployer des applications d'entreprise à l'aide de prompts en langage naturel.

## Démarrage rapide

```bash
# Installation
npm install -g openyida

# Connexion
openyida login

# Création d'app
openyida create-app --name "Mon CRM"
```
