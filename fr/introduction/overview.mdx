## Vue d'ensemble

**OpenYida** est un outil CLI sans configuration qui connecte les assistants de programmation IA à la plateforme low-code [Yida](https://www.aliwork.com). Installez-le une fois, puis décrivez vos besoins en langage naturel — OpenYida gère automatiquement : création d'application, gestion de formulaires, publication de pages, configuration de connecteurs, génération de rapports, déploiement en secondes sur Yida.

🌐 **Site officiel**: [https://openyida.ai](https://openyida.ai)

## Fonctionnalités principales

| Module de fonctionnalité | Description |
|---------|------|
| **Gestion des applications** | Création d'application, import/export, migration et déploiement |
| **Gestion des formulaires** | Création/mise à jour de formulaires, récupération du Schema, configuration des permissions |
| **Publication de pages** | Développement de pages personnalisées, compilation Babel, publication en un clic |
| **Configuration des processus** | Création de formulaires de processus, configuration des règles d'approbation |
| **Intégration et automatisation** | Création et gestion de flux logiques, déclencheurs d'événements de formulaire, notifications, opérations de données inter-formulaires |
| **Gestion des données** | Requêtes de données de formulaire, instances de processus, gestion des tâches |
| **Connecteurs** | Création de connecteurs HTTP, intégration API, import intelligent curl |
| **Rapports et graphiques** | Création de rapports Yida, ajout de graphiques, visualisation des données |
| **Gestion CDN** | Téléchargement d'images, rafraîchissement du cache, configuration OSS |

## Différences avec d'autres plateformes de construction IA

| Dimension | OpenYida | Autres plateformes de construction IA |
|------|----------|------------------|
| **Utilisateurs cibles** | Développeurs (personnes connaissant le code) | Personnel métier (non-développeurs) |
| **Mode d'interaction** | Langage naturel + conversation IA | Glisser-déposer visuel + panneaux de configuration |
| **Résultat** | Application Yida (éditable, supporte des capacités low-code complètes) | Configuration (exécution en boîte noire) |
| **Mode de déploiement** | Plateforme Yida | Liaison à une plateforme SaaS |
| **Modèle IA** | Choix à la demande, modèle le plus approprié | Spécifié par la plateforme, non remplaçable |
| **Sécurité et conformité** | Yida dispose de capacités complètes de sécurité et de conformité | Dépend des capacités de la plateforme |
