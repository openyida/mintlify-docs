## Prérequis

Avant de commencer, assurez-vous d'avoir installé les dépendances suivantes :

| Dépendance | Version | Commande de vérification |
|------|------|----------|
| [Node.js](https://nodejs.org) | ≥ 18 | `node --version` |

<Tip>
  C'est tout ce dont vous avez besoin. OpenYida est une implémentation pure Node.js — pas besoin de Python, Playwright ou d'étapes de build.
</Tip>

---

## Étape 1 — Installation

```bash
npm install -g openyida
```

Vérifiez l'installation :

```bash
openyida --version
```

---

## Étape 2 — Connexion à Yida

```bash
openyida login
```

OpenYida ouvrira une fenêtre de navigateur (ou affichera un code QR) pour vous permettre de vous authentifier avec votre compte Yida/Alibaba. Les identifiants sont mis en cache localement — vous n'avez besoin de le faire qu'une seule fois.

<Info>
  Si vous utilisez **Wukong** comme outil IA, utilisez le flag de connexion spécial :

  ```bash
  openyida login --wukong
  ```
</Info>

---

## Étape 3 — Vérifier l'environnement

```bash
openyida env
```

Cela détecte l'outil de programmation IA actuellement actif (Claude Code, Cursor, Wukong, etc.) et affiche votre état de connexion actuel. Exécutez cette commande chaque fois que vous changez d'environnement.

---

## Étape 4 — Initialiser l'espace de travail du projet

```bash
openyida copy
```

Cela copie le modèle d'espace de travail `project/` dans le répertoire de travail de votre outil IA actuel, permettant à l'IA d'accéder à votre configuration d'application et aux modèles de page.

---

## Étape 5 — Commencer à construire

Parlez maintenant à votre assistant de programmation IA. Voici quelques prompts pour commencer :

```
Construis un système IPD sur Yida pour gérer le workflow complet de production de puces
```

```
Construis un système CRM avec des leads, des clients et des transactions
```

```
Crée une application de calculateur de salaire personnel
```

```
Génère un mini-jeu de vœux d'anniversaire sur Yida
```

L'IA utilisera le pack de compétences OpenYida pour appeler les commandes CLI dans le bon ordre : `create-app` → `create-form` / `create-page` → `publish` → `save-share-config`.

---

## Étape 6 — Publier l'application

Après que l'IA a créé l'application, publiez-la avec :

```bash
openyida publish
```

Ou demandez à l'IA :
```
Publie la page [nom de la page]
Rends la page accessible publiquement
```

---

## Exécuter les diagnostics

Si quelque chose semble incorrect, exécutez l'outil de diagnostic intégré :

```bash
openyida doctor
```

Il vérifie votre version Node.js, l'état de connexion et la configuration de l'environnement, et peut corriger automatiquement les problèmes courants avec `--fix`.

---

## Intégration OpenClaw

Si vous utilisez OpenClaw, installez directement le pack de compétences OpenYida :

```bash
npx clawhub@latest install nicky1108/yida-app
```

Utilisez-le ensuite via [yida-app sur ClawHub](https://clawhub.ai/nicky1108/yida-app).
