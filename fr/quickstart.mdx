## Prérequis

Avant de commencer, assurez-vous d'avoir installé les dépendances suivantes :

| Dépendance | Version | Commande de vérification |
|------------|---------|--------------------------|
| [Node.js](https://nodejs.org) | ≥ 18 | `node --version` |

<Tip>
  Node.js suffit. Pas besoin de Python, Playwright ou d'étapes de construction — implémentation pure Node.js.
</Tip>

---

## Étape 1 — Installation

```bash
npm install -g openyida
```

Vérifiez l'installation :

```bash
openyida --version
```

---

## Étape 2 — Connexion à Yida

```bash
openyida login
```

OpenYida ouvrira une fenêtre de navigateur (ou affichera un code QR) pour vous authentifier avec votre compte Yida/Alibaba. Les identifiants sont mis en cache localement — une seule connexion nécessaire.

<Info>
  Si vous utilisez **Wukong** comme outil d'IA, utilisez le drapeau de connexion spécial :

  ```bash
  openyida login --wukong
  ```
</Info>

---

## Étape 3 — Vérifier l'environnement

```bash
openyida env
```

Détecte l'outil de codage IA actuellement actif (Claude Code, Cursor, Wukong, etc.) et affiche votre état de connexion. Exécutez cette commande à tout moment lorsque vous changez d'environnement.

---

## Étape 4 — Initialiser l'espace de travail du projet

```bash
openyida copy
```

Copie le modèle d'espace de travail `project/` dans le répertoire de travail de votre outil d'IA actuel, permettant à l'IA d'accéder à votre configuration d'application et aux modèles de page.

---

## Étape 5 — Commencer à construire

Parlez directement à votre assistant de codage IA. Voici quelques exemples de prompts pour commencer :

```
Construisez-moi un système IPD sur Yida pour gérer l'ensemble du flux de production de puces
```

```
Aidez-moi à construire un CRM avec des leads, des clients et des transactions
```

```
Créez une application de calculatrice de salaire personnelle
```

```
Générez un mini-jeu de vœux d'anniversaire sur Yida
```

L'IA utilisera le package de compétences OpenYida pour appeler les commandes CLI dans le bon ordre : `create-app` → `create-form` / `create-page` → `publish` → `save-share-config`.

---

## Étape 6 — Publier l'application

Une fois l'IA a généré l'application, exécutez la commande de publication :

```bash
openyida publish
```

Ou demandez à l'IA :
```
Publiez la page [nom de la page]
Rendez la page accessible publiquement
```

---

## Exécuter les diagnostics

Si vous rencontrez des problèmes, exécutez l'outil de diagnostic intégré :

```bash
openyida doctor
```

Il vérifie votre version Node.js, l'état de connexion et la configuration de l'environnement, et peut utiliser `--fix` pour réparer automatiquement les problèmes courants.

## Intégration OpenClaw
