## Prérequis

Avant de commencer, assurez-vous d'avoir installé les dépendances suivantes :

| Dépendance | Version | Commande de vérification |
|------|------|----------|
| [Node.js](https://nodejs.org) | ≥ 18 | `node --version` |

<Tip>
  C'est tout ce dont vous avez besoin. OpenYida est une implémentation pure Node.js — pas de Python, pas de Playwright, pas d'étapes de build.
</Tip>

---

## Étape 1 — Installation

```bash
npm install -g openyida
```

Vérifiez l'installation :

```bash
openyida --version
```

---

## Étape 2 — Connexion à Yida

```bash
openyida login
```

OpenYida ouvrira une fenêtre de navigateur (ou affichera un code QR) pour vous authentifier avec votre compte Yida/Alibaba. Les identifiants sont mis en cache localement — vous n'avez besoin de le faire qu'une seule fois.

<Info>
  Si vous utilisez **Wukong** comme outil IA, utilisez le drapeau de connexion spécial :

  ```bash
  openyida login --wukong
  ```
</Info>

---

## Étape 3 — Vérifier l'environnement

```bash
openyida env
```

Cela détecte l'outil de codage IA actuellement actif (Claude Code, Cursor, Wukong, etc.) et affiche votre état de connexion actuel. Exécutez cette commande chaque fois que vous changez d'environnement.

---

## Étape 4 — Initialiser l'espace de travail du projet

```bash
openyida copy
```

Cela copie le modèle d'espace de travail `project/` dans le répertoire de travail de votre outil IA actuel, permettant à l'IA d'accéder à la configuration de votre application et aux modèles de page.

---

## Étape 5 — Commencer à construire

Maintenant, parlez simplement à votre assistant de codage IA. Voici quelques invites de démarrage :

```
Construisez-moi un système IPD sur Yida pour gérer le flux de travail complet de production de puces
```

```
Aidez-moi à construire un système CRM avec des prospects, des contacts et des affaires
```

```
Créez une application de calculatrice de salaire personnel
```

```
Générez un mini-jeu de vœux d'anniversaire sur Yida
```

L'IA utilisera le pack de compétences OpenYida pour appeler les commandes CLI correctes dans l'ordre : `create-app` → `create-form` / `create-page` → `publish` → `save-share-config`.

---

## Étape 6 — Publier l'application

Après que l'IA a généré l'application, utilisez la commande suivante pour publier :

```bash
openyida publish
```

Ou demandez à l'IA :
```
Publiez la page [nom de la page]
Rendez la page accessible publiquement
```

---

## Exécuter les diagnostics

Si quelque chose ne va pas, exécutez l'outil de diagnostic intégré :

```bash
openyida doctor
```

Il vérifie votre version de Node.js, l'état de connexion et la configuration de l'environnement, et peut corriger automatiquement les problèmes courants avec `--fix`.

## Intégration OpenClaw

Si vous utilisez OpenClaw, installez directement le pack de compétences OpenYida :

```bash
npx clawhub@latest install nicky1108/yida-app
```
