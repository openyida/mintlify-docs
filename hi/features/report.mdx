## रिपोर्ट बनाना

### बुनियादी उपयोग

```bash
openyida create-report <appType> "रिपोर्ट नाम" '<चार्ट-परिभाषा-JSON-या-फ़ाइल>'
```

तीसरा आर्गुमेंट एक इनलाइन JSON स्ट्रिंग या JSON फ़ाइल का पथ हो सकता है।

### उदाहरण: बिक्री रिपोर्ट बनाएं

```bash
openyida create-report APP_XXXXXX "Sales Dashboard" '{
  "charts": [
    {
      "type": "bar",
      "title": "Monthly Sales",
      "dataSource": "sales_form",
      "xField": "month",
      "yField": "amount"
    }
  ]
}'
```

### फ़ाइल से चार्ट परिभाषा लोड करें

```bash
openyida create-report APP_XXXXXX "Annual Report" ./charts-config.json
```

---

## रिपोर्ट में चार्ट जोड़ना

```bash
openyida append-chart <appType> <reportId> '<चार्ट-परिभाषा-JSON-या-फ़ाइल>'
```

### उदाहरण

```bash
openyida append-chart APP_XXXXXX REPORT_XXXXX '{
  "type": "line",
  "title": "Growth Trend",
  "dataSource": "form_sales",
  "xField": "month",
  "yField": "growth_rate"
}'
```

## चार्ट प्रकार

| प्रकार | विवरण | के लिए उपयुक्त |
|------|-------------|----------|
| **bar** | बार चार्ट | श्रेणियों की तुलना |
| **line** | लाइन चार्ट | समय के साथ प्रवृत्तियां |
| **pie** | पाई चार्ट | भाग-से-संपूर्ण संबंध |
| **table** | डेटा टेबल | विस्तृत डेटा दृश्य |
| **area** | एरिया चार्ट | संचयी प्रवृत्तियां |
| **scatter** | स्कैटर चार्ट | सहसंबंध विश्लेषण |

## चार्ट परिभाषा प्रारूप

### बार चार्ट

```json
{
  "type": "bar",
  "title": "Sales by Region",
  "dataSource": "form_uuid",
  "xField": "region",
  "yField": "amount",
  "aggregation": "sum"
}
```

### लाइन चार्ट

```json
{
  "type": "line",
  "title": "Traffic Trend",
  "dataSource": "form_uuid",
  "xField": "date",
  "yField": "visitors",
  "aggregation": "sum"
}
```

### पाई चार्ट

```json
{
  "type": "pie",
  "title": "Market Share",
  "dataSource": "form_uuid",
  "categoryField": "product",
  "valueField": "share"
}
```

### डेटा टेबल

```json
{
  "type": "table",
  "title": "Detail View",
  "dataSource": "form_uuid",
  "columns": [
    { "field": "name", "title": "Name" },
    { "field": "amount", "title": "Amount" },
    { "field": "date", "title": "Date" }
  ]
}
```

## एग्रीगेशन फ़ंक्शन

| फ़ंक्शन | विवरण |
|----------|-------------|
| **sum** | मानों का योग |
| **avg** | औसत मान |
| **count** | रिकॉर्ड की गणना |
| **max** | अधिकतम मान |
| **min** | न्यूनतम मान |

## पूर्ण कॉन्फ़िगरेशन उदाहरण

### sales-report-charts.json

```json
{
  "charts": [
    {
      "type": "bar",
      "title": "Monthly Sales",
      "dataSource": "form_monthly_sales",
      "xField": "month",
      "yField": "sales_amount",
      "aggregation": "sum"
    },
    {
      "type": "pie",
      "title": "Product Categories",
      "dataSource": "form_sales",
      "categoryField": "product_category",
      "valueField": "sales_amount"
    },
    {
      "type": "line",
      "title": "Sales Trend",
      "dataSource": "form_daily_sales",
      "xField": "date",
      "yField": "sales_amount",
      "aggregation": "sum"
    },
    {
      "type": "table",
      "title": "Sales Detail",
      "dataSource": "form_sales",
      "columns": [
        { "field": "order_no", "title": "Order No.", "width": 150 },
        { "field": "customer_name", "title": "Customer", "width": 200 },
        { "field": "product_name", "title": "Product", "width": 200 },
        { "field": "amount", "title": "Amount", "width": 120 },
        { "field": "sales_date", "title": "Date", "width": 150 }
      ],
      "pageSize": 10
    }
  ]
}
```

रिपोर्ट बनाएं:

```bash
openyida create-report APP_XXXXXX "Sales Analysis Report" ./sales-report-charts.json
```

## सर्वोत्तम अभ्यास

1. **उचित चार्ट प्रकार चुनें** - प्रवृत्तियों के लिए लाइन चार्ट, तुलना के लिए बार चार्ट, अनुपात के लिए पाई चार्ट
2. **चार्ट की संख्या सीमित रखें** - प्रति रिपोर्ट 6 या उससे कम चार्ट रखें
3. **डेटा टेबल शामिल करें** - विस्तृत डेटा निरीक्षण के लिए चार्ट के साथ टेबल जोड़ें
4. **उचित पेज साइज़ सेट करें** - डेटा टेबल के लिए प्रति पृष्ठ 10-20 पंक्तियां उपयोग करें
