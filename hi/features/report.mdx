## रिपोर्ट बनाएं

### बुनियादी उपयोग

```bash
openyida create-report <appType> "रिपोर्ट नाम" '<चार्ट परिभाषा JSON>'
```

### उदाहरण: बिक्री रिपोर्ट बनाएं

```bash
openyida create-report APP_XXXXXX "मासिक बिक्री रिपोर्ट" '{
  "charts": [
    {
      "type": "bar",
      "title": "मासिक बिक्री",
      "dataSource": "form_sales",
      "xField": "month",
      "yField": "amount"
    },
    {
      "type": "pie",
      "title": "उत्पाद बिक्री अनुपात",
      "dataSource": "form_sales",
      "categoryField": "product_type",
      "valueField": "amount"
    }
  ]
}'
```

### फ़ाइल से चार्ट परिभाषा लोड करें

```bash
openyida create-report APP_XXXXXX "वार्षिक सांख्यिकी रिपोर्ट" ./charts-config.json
```

## चार्ट परिभाषा प्रारूप

### बार चार्ट

```json
{
  "type": "bar",
  "title": "बार चार्ट शीर्षक",
  "dataSource": "form_uuid",
  "xField": "category_field",
  "yField": "value_field",
  "aggregation": "sum"
}
```

### लाइन चार्ट

```json
{
  "type": "line",
  "title": "प्रवृत्ति चार्ट शीर्षक",
  "dataSource": "form_uuid",
  "xField": "date_field",
  "yField": "value_field",
  "aggregation": "sum"
}
```

### पाई चार्ट

```json
{
  "type": "pie",
  "title": "अनुपात चार्ट शीर्षक",
  "dataSource": "form_uuid",
  "categoryField": "category_field",
  "valueField": "value_field"
}
```

### डेटा टेबल

```json
{
  "type": "table",
  "title": "डेटा विवरण",
  "dataSource": "form_uuid",
  "columns": [
    { "field": "name", "title": "नाम" },
    { "field": "amount", "title": "राशि" },
    { "field": "date", "title": "तारीख" }
  ]
}
```

## रिपोर्ट में चार्ट जोड़ें

```bash
openyida append-chart <appType> <reportId> '<चार्ट परिभाषा JSON>'
```

### उदाहरण

```bash
openyida append-chart APP_XXXXXX REPORT_XXXXX '{
  "type": "line",
  "title": "वृद्धि प्रवृत्ति",
  "dataSource": "form_sales",
  "xField": "month",
  "yField": "growth_rate"
}'
```


## चार्ट प्रकार संदर्भ

| प्रकार | विवरण | आवश्यक फ़ील्ड |
|------|------|----------|
| `bar` | बार चार्ट | xField, yField |
| `line` | लाइन चार्ट | xField, yField |
| `pie` | पाई चार्ट | categoryField, valueField |
| `table` | डेटा टेबल | columns |
| `area` | एरिया चार्ट | xField, yField |
| `scatter` | स्कैटर चार्ट | xField, yField |

---

## एग्रीगेशन फ़ंक्शन

| फ़ंक्शन | विवरण |
|------|------|
| `sum` | योग |
| `avg` | औसत |
| `count` | गणना |
| `max` | अधिकतम |
| `min` | न्यूनतम |

## सर्वोत्तम अभ्यास

1. **उचित चार्ट प्रकार चुनें** - प्रवृत्ति के लिए लाइन, तुलना के लिए बार, अनुपात के लिए पाई
2. **चार्ट की संख्या नियंत्रित करें** - प्रति रिपोर्ट 6 से अधिक चार्ट न रखें
3. **डेटा टेबल का उपयोग करें** - चार्ट के साथ विस्तृत डेटा प्रदर्शित करें, मूल मान देखने के लिए
4. **उचित पेजिनेशन सेट करें** - डेटा टेबल के लिए प्रति पृष्ठ 10-20 रिकॉर्ड की अनुशंसा
