OpenYida कई व्यवसाय परिदृश्यों के लिए उपयुक्त है, नीचे सामान्य अनुप्रयोग प्रकारों के उदाहरण दिए गए हैं।

---

##ंटरप्राइज़ व्यवसाय प्रणाली

मल्टी-मॉड्यूल व्यवसाय प्लेटफ़ॉर्म बनाएं, जटिल फॉर्म, वर्कफ़्लो और डेटा संबंधों का समर्थन करें।

<AccordionGroup>
  <Accordion title="IPD (एकीकृत उत्पाद विकास)">
    **Prompt:**
    ```
    मुझे Yida में एक IPD सिस्टम बनाने में मदद करें जो चिप उत्पादन की पूरी प्रक्रिया का प्रबंधन करे,
    जिसमें परियोजना शुरुआत, डिज़ाइन समीक्षा, वेफर उत्पादन और बड़े पैमाने पर उत्पादन चरण शामिल हों
    ```
    **परिणाम:** बहु-फॉर्म एप्लिकेशन जिसमें चरण ट्रैकिंग, समीक्षा सूची, मील के पत्थर प्रबंधन और टीमों के बीच दृश्यता शामिल है।
  </Accordion>

  <Accordion title="CRM (ग्राहक संबंध प्रबंधन)">
    **Prompt:**
    ```
    मुझे एक CRM बनाने में मदद करें जिसमें लीड, ग्राहक, अवसर और फॉलो-अप रिकॉर्ड मॉड्यूल शामिल हों
    ```
    **परिणाम:** संबद्ध फॉर्म: लीड → ग्राहक → अवसर → अनुबंध, स्थिति ट्रैकिंग और असाइनमेंट फ़ील्ड के साथ।
  </Accordion>

  <Accordion title="HR और छुट्टी प्रबंधन">
    **Prompt:**
    ```
    एक HR छुट्टी प्रबंधन सिस्टम बनाएं जिसमें छुट्टी आवेदन, अनुमोदन प्रवाह और छुट्टी शेष सांख्यिकी शामिल हों
    ```
    **परिणाम:** छुट्टी आवेदन फॉर्म जिसमें अनुमोदन प्रवाह, शेष कैलकुलेटर और प्रबंधक डैशबोर्ड शामिल है।
  </Accordion>

  <Accordion title="परियोजना प्रबंधन">
    **Prompt:**
    ```
    एक परियोजना प्रबंधन एप्लिकेशन बनाएं जिसमें कार्य, मील के पत्थर, टीम सदस्य और प्रगति ट्रैकिंग शामिल हों
    ```
    **परिणाम:** कार्य सूची जिसमें प्राथमिकता/स्थिति/जिम्मेदार फ़ील्ड, मील के पत्थर ट्रैकर और गैंट चार्ट दृश्य शामिल है।
  </Accordion>
</AccordionGroup>

##💰िता उपकरण और कैलकुलेटर

एकल-उद्देश्य गणना या डेटा संग्रह उपकरण।

<AccordionGroup>
  <Accordion title="व्यक्तिगत वेतन कैलकुलेटर">
    **Prompt:**
    ```
    मुझे एक व्यक्तिगत वेतन कैलकुलेटर बनाने में मदद करें, सकल वेतन दर्ज करें, स्वचालित रूप से कर और शुद्ध वेतन की गणना करें
    ```
    **परिणाम:** फॉर्म जिसमें कर से पहले कटौती, कर गणना और शुद्ध सारांश के लिए सूत्र फ़ील्ड शामिल हैं।
  </Accordion>

  <Accordion title="बजट ट्रैकर">
    **Prompt:**
    ```
    मुझे एक व्यक्तिगत बजट ट्रैकर बनाने में मदद करें जो आय, व्यय श्रेणियों और मासिक सारांश रिकॉर्ड करे
    ```
    **परिणाम:** आय/व्यय प्रविष्टि फॉर्म जिसमें श्रेणी ड्रॉपडाउन और स्वचालित रूप से गणना किया गया मासिक शेष शामिल है।
  </Accordion>
</AccordionGroup>

मार्केटिंग पेज, उत्पाद परिचय और आंतरिक पोर्टल।

<AccordionGroup>
  <Accordion title="एंटरप्राइज़ उत्पाद लैंडिंग पेज">
    **Prompt:**
    ```
    Zhilian Xietong एंटरप्राइज़ सहयोग उत्पाद के लिए एक लैंडिंग पेज बनाएं, जिसमें मुख्य स्क्रीन, फ़ंक्शन परिचय और संपर्क फॉर्म शामिल हों
    ```
    **परिणाम:** पूर्ण कस्टम डिस्प्ले पेज जिसमें मुख्य स्क्रीन बैनर, फ़ंक्शन कार्ड और एम्बेडेड संपर्क फॉर्म शामिल हैं।
  </Accordion>

  <Accordion title="इवेंट पंजीकरण पेज">
    **Prompt:**
    ```
    वार्षिक तकनीकी सम्मेलन के लिए एक इवेंट पंजीकरण पेज बनाएं जिसमें प्रतिभागी जानकारी फॉर्म और उप-फोरम चयन शामिल हो
    ```
    **परिणाम:** ब्रांडेड पंजीकरण पेज जिसमें फॉर्म सत्यापन और पुष्टि तर्क शामिल है।
  </Accordion>
</AccordionGroup>

##टरैक्टिव गतिविधियां और मिनी गेम्स

आंतरिक गतिविधियों, त्योहारों और मार्केटिंग के लिए इंटरैक्टिव उपकरण।

<AccordionGroup>
  <Accordion title="लालटेन त्योहार पहेली गेम">
    **Prompt:**
    ```
    लालटेन त्योहार के लिए एक पहेली गेम बनाएं, AI पहेली छवियां बनाता है, उपयोगकर्ता उत्तर भेजते हैं, यदि गलत तो AI हास्यपूर्ण प्रतिक्रिया देता है
    ```
    **परिणाम:** गेम पेज जिसमें AI द्वारा बनाई गई पहेली छवियां, उत्तर सबमिशन फॉर्म और सशर्त प्रतिक्रिया तर्क शामिल है।
  </Accordion>

  <Accordion title="जन्मदिन बधाई मिनी गेम">
    **Prompt:**
    ```
    एक जन्मदिन बधाई मिनी गेम बनाएं जहां उपयोगकर्ता अपना नाम दर्ज करते हैं और AI से व्यक्तिगत बधाई कार्ड प्राप्त करते हैं
    ```
    **परिणाम:** इंटरैक्टिव कार्ड जनरेटर जिसमें प्रतिक्रिया ट्रैकिंग के लिए डेटा स्टोरेज शामिल है।
  </Accordion>
</AccordionGroup>

##🔌 एकीकरण (कनेक्टर)

HTTP कनेक्टर के माध्यम से Yida एप्लिकेशन को बाहरी सिस्टम से कनेक्ट करें।

<AccordionGroup>
  <Accordion title="Webhook सूचनाएं">
    **Prompt:**
    ```
    हमारे DingTalk webhook के लिए एक HTTP कनेक्टर बनाएं, फॉर्म सबमिशन सूचनाएं भेजने के लिए POST एक्शन जोड़ें
    ```
    **परिणाम:** परीक्षण क्षमता के साथ कॉन्फ़िगर किया गया कनेक्टर, फॉर्म इवेंट पर ट्रिगर किया जा सकता है।
  </Accordion>

  <Accordion title="curl से स्मार्ट निर्माण">
    **Prompt:**
    ```
    इस curl कमांड से स्मार्ट तरीके से कनेक्टर बनाएं: curl -X POST https://api.example.com/v1/users -H "Authorization: Bearer TOKEN" -d '{"name":"test"}'
    ```
    **परिणाम:** स्वचालित रूप से कॉन्फ़िगर किया गया कनेक्टर, बेस URL, हेडर और एक्शन पैरामीटर निकालता है।
  </Accordion>
</AccordionGroup>

<Note>
ये शुरुआती बिंदु हैं। चूंकि परिणाम Yida प्लेटफॉर्म पर पूरी तरह से संपादन योग्य एप्लिकेशन है (ब्लैक बॉक्स कॉन्फ़िगरेशन नहीं), आप हमेशा Yida विज़ुअल एडिटर या AI को बाद के निर्देश देकर जनरेट किए गए एप्लिकेशन को अनुकूलित कर सकते हैं।
</Note>
