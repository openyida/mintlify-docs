## अवलोकन

**OpenYida** एक शून्य-कॉन्फ़िगरेशन CLI टूल है जो AI कोडिंग सहायकों को [Yida](https://www.aliwork.com) लो-कोड प्लेटफॉर्म से जोड़ता है। एक बार इंस्टॉल करें, फिर प्राकृतिक भाषा में अपनी आवश्यकताओं का वर्णन करें — OpenYida स्वचालित रूप से संभालता है: ऐप निर्माण, फॉर्म प्रबंधन, पेज प्रकाशन, कनेक्टर कॉन्फ़िगरेशन, रिपोर्ट जनरेशन, और सेकंडों में Yida पर डिप्लॉयमेंट।

🌐 **आधिकारिक वेबसाइट**: [https://openyida.ai](https://openyida.ai)

## मुख्य सुविधाएँ

| सुविधा मॉड्यूल | विवरण |
|---------|------|
| **ऐप प्रबंधन** | ऐप निर्माण, आयात/निर्यात, माइग्रेशन और डिप्लॉयमेंट |
| **फॉर्म प्रबंधन** | फॉर्म निर्माण/अपडेट, Schema प्राप्ति, अनुमति कॉन्फ़िगरेशन |
| **पेज प्रकाशन** | कस्टम पेज डेवलपमेंट, Babel संकलन, वन-क्लिक प्रकाशन |
| **वर्कफ्लो कॉन्फ़िगरेशन** | प्रोसेस फॉर्म निर्माण, अनुमोदन नियम कॉन्फ़िगरेशन |
| **डेटा प्रबंधन** | फॉर्म डेटा क्वेरी, प्रोसेस इंस्टेंस, टास्क प्रबंधन |
| **कनेक्टर** | HTTP कनेक्टर निर्माण, API एकीकरण, स्मार्ट curl आयात |
| **रिपोर्ट और चार्ट** | Yida रिपोर्ट निर्माण, चार्ट जोड़ना, डेटा विज़ुअलाइज़ेशन |
| **CDN प्रबंधन** | छवि अपलोड, कैश रिफ्रेश, OSS कॉन्फ़िगरेशन |

## अन्य AI बिल्डिंग प्लेटफॉर्म से अंतर

| आयाम | OpenYida | अन्य AI बिल्डिंग प्लेटफॉर्म |
|------|----------|------------------|
| **लक्षित उपयोगकर्ता** | डेवलपर्स (कोड समझने वाले लोग) | व्यावसायिक उपयोगकर्ता (गैर-डेवलपर्स) |
| **इंटरैक्शन मोड** | प्राकृतिक भाषा + AI संवाद | विज़ुअल ड्रैग-एंड-ड्रॉप + कॉन्फ़िगरेशन पैनल |
| **आउटपुट** | Yida ऐप (बाद में संपादन योग्य, पूर्ण लो-कोड समर्थन) | कॉन्फ़िगरेशन (ब्लैक-बॉक्स निष्पादन) |
| **डिप्लॉयमेंट मोड** | Yida प्लेटफॉर्म | SaaS प्लेटफॉर्म बाइंडिंग |
| **AI मॉडल** | आवश्यकतानुसार चयन, सबसे उपयुक्त मॉडल चुनें | प्लेटफॉर्म द्वारा निर्दिष्ट, बदला नहीं जा सकता |
| **सुरक्षा और अनुपालन** | Yida में पूर्ण सुरक्षा और अनुपालन क्षमताएं | प्लेटफॉर्म क्षमताओं पर निर्भर |
