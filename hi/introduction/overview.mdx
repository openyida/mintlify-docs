## अवलोकन

**OpenYida** एक शून्य-कॉन्फ़िगरेशन CLI टूल है जो AI प्रोग्रामिंग सहायक को [Yida](https://www.aliwork.com) लो-कोड प्लेटफ़ॉर्म से जोड़ता है। एक बार इंस्टॉल करें, फिर प्राकृतिक भाषा में आवश्यकताओं का वर्णन करें — OpenYida स्वचालित रूप से संभालेगा: एप्लिकेशन बनाना, फ़ॉर्म प्रबंधित करना, पेज प्रकाशित करना, कनेक्टर कॉन्फ़िगर करना, रिपोर्ट बनाना, सेकंड में Yida पर तैनाती।

🌐 **आधिकारिक वेबसाइट**: [https://openyida.ai](https://openyida.ai)

## मुख्य कार्यक्षमता

| कार्यक्षमता मॉड्यूल | विवरण |
|---------|------|
| **एप्लिकेशन प्रबंधन** | एप्लिकेशन बनाना, आयात/निर्यात, माइग्रेशन और तैनाती |
| **फ़ॉर्म प्रबंधन** | फ़ॉर्म बनाना/अपडेट करना, Schema प्राप्त करना, अनुमति कॉन्फ़िगर करना |
| **पेज प्रकाशित करना** | कस्टम पेज विकास, Babel संकलन, एक-क्लिक प्रकाशन |
| **प्रक्रिया कॉन्फ़िगरेशन** | प्रक्रिया फ़ॉर्म बनाना, अनुमोदन नियम कॉन्फ़िगर करना |
| **डेटा प्रबंधन** | फ़ॉर्म डेटा क्वेरी, प्रक्रिया उदाहरण, कार्य प्रबंधन |
| **कनेक्टर** | HTTP कनेक्टर बनाना, API एकीकरण, curl स्मार्ट आयात |
| **रिपोर्ट और चार्ट** | Yida रिपोर्ट बनाना, चार्ट जोड़ना, डेटा विज़ुअलाइज़ेशन |
| **CDN प्रबंधन** | छवि अपलोड, कैश रिफ्रेश, OSS कॉन्फ़िगरेशन |

## अन्य AI बिल्ड प्लेटफ़ॉर्म से अंतर

| आयाम | OpenYida | अन्य AI बिल्ड प्लेटफ़ॉर्म |
|------|----------|------------------|
| **लक्षित उपयोगकर्ता** | डेवलपर (कोड जानने वाले) | व्यावसायिक कर्मचारी (गैर-डेवलपर) |
| **इंटरैक्शन विधि** | प्राकृतिक भाषा + AI संवाद | विज़ुअल ड्रैग-एंड-ड्रॉप + कॉन्फ़िगरेशन पैनल |
| **आउटपुट** | Yida एप्लिकेशन (दोबारा संपादन योग्य, पूर्ण लो-कोड क्षमता समर्थित) | कॉन्फ़िगरेशन (ब्लैक बॉक्स चलाना) |
| **तैनाती विधि** | Yida प्लेटफ़ॉर्म | SaaS प्लेटफ़ॉर्म बाध्य |
| **AI मॉडल** | मांग पर चुनें, सबसे उपयुक्त मॉडल चुनें | प्लेटफ़ॉर्म द्वारा निर्दिष्ट, बदल नहीं सकते |
| **सुरक्षा और अनुपालन** | Yida में पूर्ण सुरक्षा और अनुपालन क्षमता | प्लेटफ़ॉर्म क्षमता पर निर्भर |
