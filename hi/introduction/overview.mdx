## OpenYida क्या है?

OpenYida एक कमांड-लाइन इंटरफ़ेस है जो AI एजेंटों को [Yida](https://www.aliwork.com) लो-कोड प्लेटफॉर्म से जोड़ता है। यह प्राकृतिक भाषा प्रॉम्प्ट का उपयोग करके एंटरप्राइज़ एप्लिकेशन बनाने, प्रबंधित करने और तैनात करने की अनुमति देता है।

## त्वरित प्रारंभ

```bash
# इंस्टॉलेशन
npm install -g openyida

# लॉगिन
openyida login

# ऐप बनाएं
openyida create-app --name "मेरा CRM"
```
