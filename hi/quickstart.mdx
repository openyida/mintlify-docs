## पूर्वापेक्षाएँ

शुरू करने से पहले, सुनिश्चित करें कि निम्नलिखित निर्भरताएँ इंस्टॉल हैं:

| निर्भरता | संस्करण | जाँच कमांड |
|------|------|----------|
| [Node.js](https://nodejs.org) | ≥ 18 | `node --version` |

<Tip>
  बस इतना ही चाहिए। OpenYida एक शुद्ध Node.js कार्यान्वयन है — कोई Python नहीं, कोई Playwright नहीं, कोई build चरण नहीं।
</Tip>

---

## चरण 1 — इंस्टॉलेशन

```bash
npm install -g openyida
```

इंस्टॉलेशन सत्यापित करें:

```bash
openyida --version
```

---

## चरण 2 — Yida में लॉगिन

```bash
openyida login
```

OpenYida एक ब्राउज़र विंडो खोलेगा (या QR कोड प्रदर्शित करेगा) ताकि आप अपने Yida/Alibaba खाते से प्रमाणित कर सकें। क्रेडेंशियल स्थानीय रूप से कैश किए जाते हैं — आपको इसे केवल एक बार करने की आवश्यकता है।

<Info>
  यदि आप AI टूल के रूप में **Wukong** का उपयोग कर रहे हैं, तो विशेष लॉगिन फ्लैग का उपयोग करें:

  ```bash
  openyida login --wukong
  ```
</Info>

---

## चरण 3 — वातावरण जाँचें

```bash
openyida env
```

यह वर्तमान में सक्रिय AI कोडिंग टूल (Claude Code, Cursor, Wukong, आदि) का पता लगाता है और आपकी वर्तमान लॉगिन स्थिति प्रदर्शित करता है। जब भी आप वातावरण बदलते हैं तो इस कमांड को चलाएं।

---

## चरण 4 — प्रोजेक्ट वर्कस्पेस प्रारंभ करें

```bash
openyida copy
```

यह `project/` वर्कस्पेस टेम्पलेट को आपके वर्तमान AI टूल की कार्य निर्देशिका में कॉपी करता है, जिससे AI आपकी ऐप कॉन्फ़िगरेशन और पेज टेम्पलेट्स तक पहुँच सकता है।

---

## चरण 5 — निर्माण शुरू करें

अब बस अपने AI कोडिंग सहायक से बात करें। यहाँ कुछ स्टार्टर प्रॉम्प्ट हैं:

```
Yida पर मुझे एक IPD सिस्टम बनाएं जो पूर्ण चिप उत्पादन वर्कफ़्लो का प्रबंधन करे
```

```
मुझे leads, contacts और deals के साथ एक CRM सिस्टम बनाने में मदद करें
```

```
एक व्यक्तिगत वेतन कैलकुलेटर ऐप बनाएं
```

```
Yida पर एक जन्मदिन की बधाई मिनी-गेम बनाएं
```

AI OpenYida के स्किल पैक का उपयोग करके सही CLI कमांड्स को क्रम में कॉल करेगा: `create-app` → `create-form` / `create-page` → `publish` → `save-share-config`।

---

## चरण 6 — ऐप प्रकाशित करें

AI द्वारा ऐप जनरेट होने के बाद, प्रकाशित करने के लिए निम्नलिखित कमांड का उपयोग करें:

```bash
openyida publish
```

या AI से पूछें:
```
[पेज नाम] पेज प्रकाशित करें
पेज को सार्वजनिक रूप से सुलभ बनाएं
```

---

## निदान चलाएं

यदि कुछ गलत लगता है, तो अंतर्निहित निदान टूल चलाएं:

```bash
openyida doctor
```

यह आपके Node.js संस्करण, लॉगिन स्थिति और वातावरण कॉन्फ़िगरेशन की जाँच करता है, और `--fix` के साथ सामान्य समस्याओं को स्वचालित रूप से ठीक कर सकता है।

## OpenClaw एकीकरण

यदि आप OpenClaw का उपयोग कर रहे हैं, तो OpenYida स्किल पैक सीधे इंस्टॉल करें:

```bash
npx clawhub@latest install nicky1108/yida-app
```

फिर इसे [ClawhHub पर yida-app](https://clawhub.ai/nicky1108/yida-app) के माध्यम से उपयोग करें।
