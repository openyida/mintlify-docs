## पूर्व-आवश्यकताएं

शुरू करने से पहले, सुनिश्चित करें कि निम्नलिखित निर्भरताएं इंस्टॉल हैं:

| निर्भरता | संस्करण | सत्यापन कमांड |
|----------|---------|---------------|
| [Node.js](https://nodejs.org) | ≥ 18 | `node --version` |

<Tip>
  केवल Node.js की आवश्यकता है। Python, Playwright या बिल्ड चरणों की आवश्यकता नहीं — शुद्ध Node.js कार्यान्वयन।
</Tip>

---

## चरण 1 — इंस्टॉलेशन

```bash
npm install -g openyida
```

इंस्टॉलेशन सत्यापित करें:

```bash
openyida --version
```

---

## चरण 2 — Yida में लॉगिन करें

```bash
openyida login
```

OpenYida आपके Yida/Alibaba खाते से प्रमाणित करने के लिए एक ब्राउज़र विंडो (या QR कोड प्रदर्शित करेगा) खोलेगा। क्रेडेंशियल्स स्थानीय रूप से कैश किए जाते हैं — केवल एक बार।

<Info>
  यदि आप **Wukong** को AI टूल के रूप में उपयोग करते हैं, तो विशेष लॉगिन फ्लैग का उपयोग करें:

  ```bash
  openyida login --wukong
  ```
</Info>

---

## चरण 3 — वातावरण जांचें

```bash
openyida env
```

वर्तमान में सक्रिय AI कोडिंग टूल (Claude Code, Cursor, Wukong, आदि) का पता लगाता है और आपकी लॉगिन स्थिति प्रदर्शित करता है। जब भी आप वातावरण बदलते हैं तो इस कमांड को चलाएं।

---

## चरण 4 — प्रोजेक्ट वर्कस्पेस प्रारंभ करें

```bash
openyida copy
```

`project/` वर्कस्पेस टेम्पलेट को आपके वर्तमान AI टूल के कार्य निर्देशिका में कॉपी करता है, जिससे AI आपकी ऐप कॉन्फ़िगरेशन और पेज टेम्पलेट तक पहुंच सकता है।

---

## चरण 5 — निर्माण शुरू करें

सीधे अपने AI कोडिंग सहायक से बात करें। यहां शुरू करने के लिए कुछ उदाहरण प्रॉम्प्ट हैं:

```
Yida पर पूरे चिप उत्पादन वर्कफ़्लो को प्रबंधित करने के लिए मेरे लिए एक IPD सिस्टम बनाएं
```

```
लीड्स, ग्राहकों और लेनदेन के साथ एक CRM बनाने में मेरी मदद करें
```

```
एक व्यक्तिगत वेतन कैलकुलेटर ऐप बनाएं
```

```
Yida पर एक जन्मदिन की बधाई मिनी गेम बनाएं
```

AI सही क्रम में CLI कमांड को कॉल करने के लिए OpenYida स्किल पैकेज का उपयोग करेगी: `create-app` → `create-form` / `create-page` → `publish` → `save-share-config`।

---

## चरण 6 — ऐप प्रकाशित करें

एक बार AI ने ऐप जेनरेट कर दिया, प्रकाशन कमांड चलाएं:

```bash
openyida publish
```

या AI से पूछें:
```
[पेज नाम] पेज प्रकाशित करें
पेज को सार्वजनिक रूप से सुलभ बनाएं
```

---

## निदान चलाएं

यदि आपको समस्याएं हैं, तो अंतर्निहित निदान टूल चलाएं:

```bash
openyida doctor
```

यह आपके Node.js संस्करण, लॉगिन स्थिति और वातावरण कॉन्फ़िगरेशन की जांच करता है, और `--fix` का उपयोग करके सामान्य समस्याओं को स्वचालित रूप से ठीक कर सकता है।

---

## OpenClaw एकीकरण

यदि आप OpenClaw का उपयोग करते हैं, तो सीधे OpenYida स्किल पैकेज इंस्टॉल करें:

```bash
npx clawhub@latest install nicky1108/yida-app
```

फिर इसे [ClawHub पर yida-app](https://clawhub.ai/nicky1108/yida-app) के माध्यम से उपयोग करें।
