यह गाइड दिखाता है कि OpenYida कैसे इंस्टॉल करें और एक बुनियादी ऐप कैसे बनाएं।

## चरण 1 — इंस्टॉलेशन

```bash
npm install -g openyida
```

सत्यापित करें कि इंस्टॉलेशन सफल रहा:

```bash
openyida --version
```

## चरण 2 — लॉगिन

```bash
openyida login
```

यह प्रमाणीकरण के लिए एक ब्राउज़र विंडो खोलेगा। सफल लॉगिन के बाद, आप ब्राउज़र बंद करके टर्मिनल पर वापस आ सकते हैं।

## चरण 3 — ऐप बनाएं

### विकल्प A: प्राकृतिक भाषा विवरण से बनाएं

```bash
openyida create-app --prompt "एक CRM सिस्टम बनाएं जिसमें ग्राहक प्रबंधन, बिक्री ट्रैकिंग और एनालिटिक्स डैशबोर्ड शामिल हो"
```

### विकल्प B: आवश्यकता फ़ाइल से बनाएं

```bash
openyida create-app --file requirements.md
```

## चरण 4 — फॉर्म प्रबंधन

### फॉर्म सूचीबद्ध करें

```bash
openyida list-forms --app-id YOUR_APP_ID
```

### नया फॉर्म बनाएं

```bash
openyida create-form --app-id YOUR_APP_ID --prompt "ग्राहक जानकारी फॉर्म जिसमें नाम, कंपनी, फोन, ईमेल और नोट्स शामिल हों"
```

### मौजूदा फॉर्म अपडेट करें

```bash
openyida update-form --form-id YOUR_FORM_ID --prompt "स्टेटस फ़ील्ड जोड़ें जिसमें विकल्प: लीड, अवसर, ग्राहक"
```

## चरण 5 — पेज प्रकाशित करें

### कस्टम पेज संकलित और प्रकाशित करें

```bash
openyida publish-page --app-id YOUR_APP_ID --file custom-page.jsx
```

### सार्वजनिक पहुंच कॉन्फ़िगर करें

```bash
openyida set-public --page-id YOUR_PAGE_ID
```

## चरण 6 — कनेक्टर कॉन्फ़िगर करें

### HTTP कनेक्टर बनाएं

```bash
openyida create-connector --app-id YOUR_APP_ID --name "भुगतान API" --config connector-config.json
```

### curl से आयात करें

```bash
openyida import-curl --app-id YOUR_APP_ID --curl 'curl -X GET "https://api.example.com/data" -H "Authorization: Bearer token"'
```
