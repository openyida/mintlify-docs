
### フォームフィールドの追加

```
[ページ名]に[フィールド名]フィールドを追加してください。フィールドタイプ：[TextField/NumberField/DateField/SelectField/...]
```
```
従業員情報フォームに「部門」ドロップダウンフィールドを追加してください。オプション：エンジニアリング部、マーケティング部、セールス部、人事部
```

<Note>
[サポートされているフィールドタイプ](/ja/features/skills#サポートされているフィールドタイプ)のみ受け付けられます。サポートされていないタイプ（例：`MobileField`、`EmailField`）は拒否されます。
</Note>

<Info>
`dws`コマンドにはDingTalk CLIの別途インストールが必要です。まだインストールしていない場合は、`openyida dws install`を実行してください。出力はデフォルトでテーブル形式です。JSON形式で出力するには`-f json`を追加してください。
</Info>
