## フォームとページ操作

### フォームページの作成

```
xxxフォームページを作成してください
```
```
休暇申請フォームを作成してください。含めるフィールド：従業員名、部門、開始日、終了日、休暇理由
```

### フォームフィールドの追加

```
[ページ名]に[フィールド名]フィールドを追加してください。フィールドタイプ：[TextField/NumberField/DateField/SelectField/...]
```
```
従業員情報フォームに「部門」ドロップダウンフィールドを追加してください。オプション：エンジニアリング部、マーケティング部、セールス部、人事部
```

<Note>
[サポートされているフィールドタイプ](/ja/features/skills#サポートされているフィールドタイプ)のみ受け付けられます。サポートされていないタイプ（例：`MobileField`、`EmailField`）は拒否されます。
</Note>

### フィールド属性の変更

```
[ページ名]の[フィールド名]フィールドを必須に設定してください
```
```
[ページ名]の[フィールド名]フィールドを読み取り専用に設定してください
```
```
[ページ名]の[フィールド名]フィールドのラベルを[新しいラベル]に変更してください
```

### カスタム表示ページ

```
[ページ名]のカスタム表示ページを作成してください
```
```
エンタープライズコラボレーション製品のランディングページを生成してください
```
