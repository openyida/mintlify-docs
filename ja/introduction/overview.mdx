## 概要

**OpenYida** はゼロ設定の CLI ツールで、AI コーディングアシスタントと[宜搭 (Yida)](https://www.aliwork.com) ローコードプラットフォームを接続します。一度インストールすれば、自然言語で要件を記述するだけ — OpenYida が自動的に処理します：アプリ作成、フォーム管理、ページ公開、コネクター設定、レポート生成、そして秒単位で宜搭にデプロイします。

🌐 **公式サイト**: [https://openyida.ai](https://openyida.ai)

## コア機能

| 機能モジュール | 説明 |
|---------|------|
| **アプリ管理** | アプリ作成、インポート/エクスポート、移行デプロイ |
| **フォーム管理** | フォームの作成/更新、Schema の取得、権限設定 |
| **ページ公開** | カスタムページ開発、Babel コンパイル、ワンクリック公開 |
| **フロー設定** | フローフォーム作成、承認ルール設定 |
| **データ管理** | フォームデータクエリ、フローインスタンス、タスク管理 |
| **コネクター** | HTTP コネクター作成、API 統合、curl スマートインポート |
| **レポートチャート** | 宜搭レポート作成、チャート追加、データ可視化 |
| **CDN 管理** | 画像アップロード、キャッシュ更新、OSS 設定 |

## 他の AI 構築プラットフォームとの違い

| 次元 | OpenYida | 他の AI 構築プラットフォーム |
|------|----------|------------------|
| **対象ユーザー** | 開発者（コードを理解する人） | ビジネスユーザー（非開発者） |
| **インタラクション方式** | 自然言語 + AI 対話 | ビジュアルドラッグ + 設定パネル |
| **産出物** | 宜搭アプリ（二次編集可能、完全なローコード機能をサポート） | 設定（ブラックボックス実行） |
| **デプロイ方式** | 宜搭プラットフォーム | SaaS プラットフォームバインディング |
| **AI モデル** | 必要に応じて選択、最適なモデルを選択 | プラットフォーム指定、変更不可 |
| **セキュリティとコンプライアンス** | 宜搭は完全なセキュリティとコンプライアンス機能を備える | プラットフォーム機能に依存 |

## サポートする AI コーディングツール

OpenYida は MCP 互換のスキルパックとして、すべての主要な AI コーディング環境とネイティブに統合されます：

| ツール | ステータス |
|------|------|
| [Claude Code](https://claude.ai/code) | ✅ 完全サポート |
| [Aone Copilot](https://copilot.code.alibaba-inc.com) | ✅ 完全サポート |
| [OpenCode](https://opencode.ai) | ✅ 完全サポート |
| [Cursor](https://cursor.com/) | ✅ 完全サポート |
| [Visual Studio Code](https://code.visualstudio.com/) | ✅ 完全サポート |
| [Qoder](https://qoder.com) | ✅ 完全サポート |
| [悟空](https://dingtalk.com/wukong) | ✅ 完全サポート |
