## 概要

**OpenYida**は設定不要のCLIツールで、AIプログラミングアシスタントと[Yida](https://www.aliwork.com)ローコードプラットフォームを接続します。一度インストールするだけで、自然言語で要件を記述するだけで、OpenYidaが自動的に処理します：アプリ作成、フォーム管理、ページ公開、コネクタ設定、レポート生成、Yidaへの秒単位でのデプロイ。

🌐 **公式ウェブサイト**: [https://openyida.ai](https://openyida.ai)

## コア機能

| 機能モジュール | 説明 |
|---------|------|
| **アプリ管理** | アプリ作成、インポート/エクスポート、移行デプロイ |
| **フォーム管理** | フォーム作成/更新、Schema取得、権限設定 |
| **ページ公開** | カスタムページ開発、Babelコンパイル、ワンクリック公開 |
| **プロセス設定** | プロセスフォーム作成、承認ルール設定 |
| **統合と自動化** | ロジックフローの作成と管理、フォームイベントトリガー、通知、クロスフォームデータ操作 |
| **データ管理** | フォームデータクエリ、プロセスインスタンス、タスク管理 |
| **コネクタ** | HTTPコネクタ作成、API統合、curlスマートインポート |
| **レポート・チャート** | Yidaレポート作成、チャート追加、データ可視化 |
| **CDN管理** | 画像アップロード、キャッシュ更新、OSS設定 |
| **MCP Appサーバー** | MCPプロトコルを通じてAI会話でYidaアプリを管理、インタラクティブなダッシュボードとチャートに対応 |

## 他のAI構築プラットフォームとの違い

| 次元 | OpenYida | 他のAI構築プラットフォーム |
|------|----------|------------------|
| **ターゲットユーザー** | 開発者（コードを理解する人） | 業務担当者（非開発者） |
| **対話方式** | 自然言語 + AI対話 | ビジュアルドラッグ＆ドロップ + 設定パネル |
| **出力物** | Yidaアプリ（再編集可能、完全なローコード機能をサポート） | 設定（ブラックボックス実行） |
| **デプロイ方式** | Yidaプラットフォーム | SaaSプラットフォームにバインド |
| **AIモデル** | 必要に応じて選択、最適なモデルを選択可能 | プラットフォーム指定、変更不可 |
| **セキュリティ・コンプライアンス** | Yidaは完全なセキュリティとコンプライアンス機能を備えている | プラットフォームの機能に依存 |
