## 環境要件

開始する前に、以下の依存関係がインストールされていることを確認してください：

| 依存関係 | バージョン | 確認コマンド |
|----------|------------|--------------|
| [Node.js](https://nodejs.org) | ≥ 18 | `node --version` |

<Tip>
  Node.jsだけで十分です。Python、Playwright、ビルドステップは不要 — 純粋なNode.js実装です。
</Tip>

---

## ステップ 1 — インストール

```bash
npm install -g openyida
```

インストールを確認：

```bash
openyida --version
```

---

## ステップ 2 — 宜搭にログイン

```bash
openyida login
```

OpenYidaはブラウザウィンドウ（またはQRコード）を開き、宜搭/Alibabaアカウントで認証します。認証情報はローカルにキャッシュされ、一度だけログインすればOKです。

<Info>
  **悟空**をAIツールとして使用する場合は、特殊なログインフラグを使用してください：

  ```bash
  openyida login --wukong
  ```
</Info>

---

## ステップ 3 — 環境を確認

```bash
openyida env
```

現在アクティブなAIコーディングツール（Claude Code、Cursor、悟空など）を検出し、ログイン状態を表示します。環境を切り替える際にいつでもこのコマンドを実行してください。

---

## ステップ 4 — プロジェクトワークスペースを初期化

```bash
openyida copy
```

`project/`ワークスペーステンプレートを現在のAIツールの作業ディレクトリにコピーし、AIがアプリ設定とページテンプレートにアクセスできるようにします。

---

## ステップ 5 — 構築を開始

AIコーディングアシスタントと直接会話してください。以下は開始用のプロンプト例です：

```
宜搭上でIPDシステムを構築し、チップ生産ワークフロー全体を管理してください
```

```
リード、顧客、商談を含むCRMを構築してください
```

```
個人給与計算アプリを作成してください
```

```
宜搭上でバースデー祝福ミニゲームを作成してください
```

AIはOpenYidaのスキルパッケージを使用して、正しい順序でCLIコマンドを呼び出します：`create-app` → `create-form` / `create-page` → `publish` → `save-share-config`。

---

## ステップ 6 — アプリを公開

AIがアプリを生成したら、公開コマンドを実行：

```bash
openyida publish
```

またはAIに尋ねてください：
```
[ページ名]ページを公開してください
ページを公開アクセス可能にしてください
```

---

## 診断を実行

問題がある場合は、組み込みの診断ツールを実行：

```bash
openyida doctor
```

Node.jsバージョン、ログイン状態、環境設定をチェックし、`--fix`で一般的な問題を自動修正できます。

---

## OpenClaw統合

OpenClawを使用する場合は、OpenYidaスキルパッケージを直接インストール：

```bash
npx clawhub@latest install nicky1108/yida-app
```

その後、[ClawHubのyida-app](https://clawhub.ai/nicky1108/yida-app)から使用してください。
