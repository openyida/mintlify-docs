`@openyida/mcp-app`는 Yida 로우코드 플랫폼의 인터랙티브 UI를 AI 대화에 직접 가져오는 [MCP Apps](https://modelcontextprotocol.io) 서버입니다. 일반 텍스트 대신 카드 스타일 대시보드, 인터랙티브 차트, 폼 미리보기를 확인할 수 있습니다.

## 사전 요구 사항

- **Node.js** 18 이상
- **openyida** CLI 전역 설치 (`npm install -g openyida`)
- 인증된 세션 — MCP 도구 사용 전 `openyida login` 실행

## 빠른 시작

### stdio 전송 (Claude Desktop, VS Code, Cursor)

MCP 클라이언트 구성에 다음을 추가하세요:

```json
{
  "mcpServers": {
    "openyida": {
      "command": "npx",
      "args": ["-y", "@openyida/mcp-app", "--stdio"]
    }
  }
}
```

### HTTP 전송 (StreamableHTTP)

```bash
npx @openyida/mcp-app
# 서버가 http://localhost:3001/mcp에서 수신 대기
```

`PORT` 환경 변수로 사용자 정의 포트를 설정할 수 있습니다:

```bash
PORT=8080 npx @openyida/mcp-app
```

## 사용 가능한 도구

### 인증 도구

| 도구 | 설명 |
|------|------|
| `yida_login_status` | 현재 로그인 상태 확인 |
| `yida_login` | Yida 로그인 플로우 시작 (`method`: `"auto"` 또는 `"qr"`) |
| `yida_env` | 현재 OpenYida 환경 및 구성 상태 확인 |

### 애플리케이션 도구

| 도구 | 설명 |
|------|------|
| `yida_list_apps` | 모든 Yida 애플리케이션 목록 조회 (인터랙티브 대시보드 포함) |
| `yida_create_app` | 새 Yida 애플리케이션 생성 |

### 폼 도구

| 도구 | 설명 |
|------|------|
| `yida_get_schema` | 폼 스키마 전체 조회 (인터랙티브 폼 미리보기 포함) |
| `yida_create_form` | 새 폼 생성 |

<Warning>
  필드 ID는 플랫폼에서 무작위로 생성됩니다. 데이터 작업 전에 반드시 `yida_get_schema`로 올바른 필드 ID를 확인하세요.
</Warning>

### 데이터 도구

| 도구 | 설명 |
|------|------|
| `yida_query_data` | 폼 데이터 인스턴스 페이지네이션 쿼리 |
| `yida_create_instance` | 새 폼 데이터 레코드 생성 |

### 리포트 도구

| 도구 | 설명 |
|------|------|
| `yida_query_report` | 리포트 시각화용 데이터 쿼리 (인터랙티브 ECharts 차트 포함) |
| `yida_create_report` | Yida 리포트 생성 |

## 인터랙티브 뷰

| 도구 | 뷰 | 설명 |
|------|-----|------|
| `yida_list_apps` | App Dashboard | 카드 스타일 애플리케이션 목록 |
| `yida_query_report` | Report Chart | 인터랙티브 ECharts 시각화 |
| `yida_get_schema` | Form Preview | 시각적 폼 구조 미리보기 |
