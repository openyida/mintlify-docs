## 🏢 비즈니스 시스템 — IPD / CRM

한 마디로 요구사항을 설명하면 AI가 완전한 다중 양식 비즈니스 시스템을 자동으로 생성합니다.

**프롬프트 예시:**
```
Yida로 IPD 시스템을 만들어줘, 칩 생산 전체 프로세스를 관리하는 시스템
```

## 🌐 랜딩 페이지 — 지능형 협업

엔터프라이즈급 제품 소개 페이지, 한 마디로 완전한 랜딩 페이지를 생성합니다.

**프롬프트 예시:**
```
지능형 협업 엔터프라이즈 협업 제품의 랜딩 페이지를 생성해줘
```

## 🏮 운영 시나리오 — 퍼즐 맞추기

AI가 퍼즐 이미지를 생성하고 사용자가 답을 맞추며, 틀리면 AI가 유머러스한 힌트를 제공합니다.

**프롬프트 예시:**
```
원소절 퍼즐 맞추기 게임을 만들어줘, AI가 퍼즐 이미지를 생성하고 사용자가 답을 제출하면, 틀렸을 때 AI가 유머러스하게 답변해줘
```

<Note>
위의 모든 데모 케이스는 단일 자연어 프롬프트로 생성되었으며 수동 구성이 필요 없습니다. 생성된 애플리케이션은 Yida 플랫폼에서 편집할 수 있는 실제 애플리케이션이며 정적 데모가 아닙니다.
