## 개요

**OpenYida**는 제로 설정 CLI 도구로, AI 코딩 어시스턴트와 [Yida](https://www.aliwork.com) 로우코드 플랫폼을 연결합니다. 한 번 설치하면 자연어로 요구사항을 설명하기만 하면 됩니다 — OpenYida가 자동으로 처리합니다: 앱 생성, 양식 관리, 페이지 게시, 커넥터 설정, 보고서 생성, 그리고 Yida에 초 단위로 배포합니다.

🌐 **공식 웹사이트**: [https://openyida.ai](https://openyida.ai)

## 핵심 기능

| 기능 모듈 | 설명 |
|---------|------|
| **앱 관리** | 앱 생성, 가져오기/날내기, 마이그레이션 배포 |
| **양식 관리** | 양식 생성/업데이트, Schema 가져오기, 권한 설정 |
| **페이지 게시** | 커스텀 페이지 개발, Babel 컴파일, 원클릭 게시 |
| **프로세스 설정** | 프로세스 양식 생성, 승인 규칙 설정 |
| **데이터 관리** | 양식 데이터 쿼리, 프로세스 인스턴스, 작업 관리 |
| **커넥터** | HTTP 커넥터 생성, API 통합, curl 스마트 가져오기 |
| **보고서 차트** | Yida 보고서 생성, 차트 추가, 데이터 시각화 |
| **CDN 관리** | 이미지 업로드, 캐시 새로고침, OSS 설정 |

## 다른 AI 구축 플랫폼과의 차이점

| 차원 | OpenYida | 다른 AI 구축 플랫폼 |
|------|----------|------------------|
| **대상 사용자** | 개발자(코드를 이해하는 사람) | 비즈니스 사용자(비개발자) |
| **상호작용 방식** | 자연어 + AI 대화 | 시각적 드래그 + 설정 패널 |
| **산출물** | Yida 앱(2차 편집 가능, 완전한 로우코드 기능 지원) | 설정(블랙박스 실행) |
| **배포 방식** | Yida 플랫폼 | SaaS 플랫폼 바인딩 |
| **AI 모델** | 필요에 따라 선택, 가장 적합한 모델 선택 | 플랫폼 지정, 변경 불가 |
| **보안 및 규정 준수** | Yida는 완전한 보안 및 규정 준수 기능을 갖춤 | 플랫폼 기능에 의존 |
