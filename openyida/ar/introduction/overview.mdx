## ما هو OpenYida؟

OpenYida هو واجهة سطر أوامر تربط وكلاء الذكاء الاصطناعي بمنصة [Yida](https://www.aliwork.com) منخفضة الكود. يسمح بإنشاء تطبيقات المؤسسات وإدارتها ونشرها باستخدام أوامر اللغة الطبيعية.

## البدء السريع

```bash
# التثبيت
npm install -g openyida

# تسجيل الدخول
openyida login

# إنشاء تطبيق
openyida create-app --name "CRM الخاص بي"
```
