## Was ist OpenYida?

OpenYida ist eine Befehlszeilenschnittstelle, die KI-Agenten mit der [Yida](https://www.aliwork.com) Low-Code-Plattform verbindet. Sie ermöglicht die Erstellung, Verwaltung und Bereitstellung von Unternehmensanwendungen mithilfe natürlicher Spracheingaben.

## Schnellstart

```bash
# Installation
npm install -g openyida

# Anmeldung
openyida login

# App erstellen
openyida create-app --name "Mein CRM"
```
