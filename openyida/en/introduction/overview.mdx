## Overview

**OpenYida** is a zero-configuration CLI tool that bridges AI coding assistants with the [Yida](https://www.aliwork.com) low-code platform. Install once, then describe your needs in natural language — OpenYida automatically handles: app creation, form management, page publishing, connector configuration, report generation, and deployment to Yida in seconds.

## Core Features

| Feature Module | Description |
|---------------|-------------|
| **App Management** | Create apps, import/export, migration deployment |
| **Form Management** | Create/update forms, get Schema, configure permissions |
| **Page Publishing** | Custom page development, Babel compilation, one-click publish |
| **Process Configuration** | Process form creation, approval rule configuration |
| **Data Management** | Form data query, process instances, task management |
| **Connectors** | HTTP connector creation, API integration, curl smart import |
| **Reports & Charts** | Yida report creation, chart append, data visualization |
| **CDN Management** | Image upload, cache refresh, OSS configuration |

## Differences from Other AI Builders

| Dimension | OpenYida | Other AI Builders |
|-----------|----------|-------------------|
| **Target Users** | Developers (code-savvy) | Business users (non-developers) |
| **Interaction** | Natural language + AI chat | Visual drag-and-drop + config panels |
| **Output** | Yida app (editable, full low-code capabilities) | Config (black-box execution) |
| **Deployment** | Yida platform | SaaS platform lock-in |
| **AI Model** | Choose the best model for the job | Platform-specified, not swappable |
| **Security & Compliance** | Yida's enterprise-grade security | Platform-dependent |
