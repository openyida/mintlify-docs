## OpenYidaとは？

OpenYidaは、AIエージェントと[宜搭](https://www.aliwork.com)ローコードプラットフォームを橋渡しするコマンドラインインターフェースです。自然言語のプロンプトを使用して、エンタープライズアプリケーションを作成、管理、デプロイできます。

## 主な機能

<CardGroup cols={2}>
  <Card title="自然言語アプリ作成" icon="message">
    日本語でアプリケーションを説明するだけ。AIが宜搭のフォーム、ページ、ワークフローに変換します。
  </Card>
  <Card title="多言語AIツール対応" icon="robot">
    Claude Code、Cursor、VS Code、Wukong、およびMCP互換のAIアシスタントと連携します。
  </Card>
  <Card title="完全な宜搭統合" icon="plug">
    実際の宜搭アプリケーションを作成 — モックアップではありません。
  </Card>
  <Card title="カスタムページコンパイル" icon="code">
    JSXベースのカスタムページをBabelで自動コンパイルして宜搭に公開します。
  </Card>
</CardGroup>

## クイックスタート

```bash
# インストール
npm install -g openyida

# ログイン
openyida login

# アプリ作成
openyida create-app --name "私のCRM"
```
