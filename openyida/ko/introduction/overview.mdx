## OpenYida란?

OpenYida는 AI 에이전트와 [Yida](https://www.aliwork.com) 로우코드 플랫폼을 연결하는 명령줄 인터페이스입니다. 자연어 프롬프트를 사용하여 엔터프라이즈 애플리케이션을 생성, 관리 및 배포할 수 있습니다.

## 빠른 시작

```bash
# 설치
npm install -g openyida

# 로그인
openyida login

# 앱 생성
openyida create-app --name "내 CRM"
```
