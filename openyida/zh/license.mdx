## 中文参考

MIT 许可证是一种宽松的开源许可证，允许您：

- **自由使用**: 个人或商业用途
- **自由修改**: 修改源代码
- **自由分发**: 分发原始或修改后的版本
- **自由再许可**: 使用其他许可证发布

唯一要求是：

- 保留版权声明
- 保留许可证文本

## 第三方依赖

OpenYida 使用了以下开源项目，各自遵循其许可证：

| 项目 | 许可证 |
|------|--------|
| Node.js | MIT |
| TypeScript | Apache-2.0 |
| Commander.js | MIT |
| Axios | MIT |
| OpenAI SDK | Apache-2.0 |

完整的依赖列表请查看 `package.json`。

"OpenYida" 名称和 Logo 是项目的商标。使用 MIT 许可证不授予商标使用权。

宜搭（Yida）是钉钉/阿里巴巴的商标，本项目与其无官方关联。

## 贡献者协议

向本项目贡献代码，即表示您：

1. 声明您有权提交该代码
2. 同意将代码授权给项目，遵循 MIT 许可证
3. 理解您的贡献将被公开记录

## 许可证变更

项目许可证的变更将通过以下流程：

1. 在 GitHub Discussions 发起讨论
2. 收集社区反馈
3. 核心维护者投票决定
4. 提前 30 天公告变更
