## Criar relatórios

### Uso básico

```bash
openyida create-report <appType> "Nome do relatório" '<JSON-definição-ou-arquivo>'
```

O terceiro argumento pode ser uma string JSON inline ou um caminho para um arquivo JSON.

### Exemplo: criar um relatório de vendas

```bash
openyida create-report APP_XXXXXX "Sales Dashboard" '{
  "charts": [
    {
      "type": "bar",
      "title": "Monthly Sales",
      "dataSource": "sales_form",
      "xField": "month",
      "yField": "amount"
    }
  ]
}'
```

### Carregar definição de gráfico de arquivo

```bash
openyida create-report APP_XXXXXX "Annual Report" ./charts-config.json
```

## Adicionar gráficos a um relatório

```bash
openyida append-chart <appType> <reportId> '<JSON-definição-ou-arquivo>'
```

### Exemplo

```bash
openyida append-chart APP_XXXXXX REPORT_XXXXX '{
  "type": "line",
  "title": "Growth Trend",
  "dataSource": "form_sales",
  "xField": "month",
  "yField": "growth_rate"
}'
```

## Tipos de gráficos

| Tipo | Descrição | Ideal para |
|------|-------------|----------|
| **bar** | Gráfico de barras | Comparar categorias |
| **line** | Gráfico de linhas | Tendências ao longo do tempo |
| **pie** | Gráfico circular | Relações parte-todo |
| **table** | Tabela de dados | Visualização detalhada de dados |
| **area** | Gráfico de área | Tendências cumulativas |
| **scatter** | Gráfico de dispersão | Análise de correlação |

## Formato de definição de gráfico

### Gráfico de barras

```json
{
  "type": "bar",
  "title": "Sales by Region",
  "dataSource": "form_uuid",
  "xField": "region",
  "yField": "amount",
  "aggregation": "sum"
}
```

### Gráfico de linhas

```json
{
  "type": "line",
  "title": "Traffic Trend",
  "dataSource": "form_uuid",
  "xField": "date",
  "yField": "visitors",
  "aggregation": "sum"
}
```

### Gráfico circular

```json
{
  "type": "pie",
  "title": "Market Share",
  "dataSource": "form_uuid",
  "categoryField": "product",
  "valueField": "share"
}
```

### Tabela de dados

```json
{
  "type": "table",
  "title": "Detail View",
  "dataSource": "form_uuid",
  "columns": [
    { "field": "name", "title": "Name" },
    { "field": "amount", "title": "Amount" },
    { "field": "date", "title": "Date" }
  ]
}
```

---

## Funções de agregação

| Função | Descrição |
|----------|-------------|
| **sum** | Soma dos valores |
| **avg** | Valor médio |
| **count** | Contagem de registros |
| **max** | Valor máximo |
| **min** | Valor mínimo |

## Exemplo de configuração completa

### sales-report-charts.json

```json
{
  "charts": [
    {
      "type": "bar",
      "title": "Monthly Sales",
      "dataSource": "form_monthly_sales",
      "xField": "month",
      "yField": "sales_amount",
      "aggregation": "sum"
    },
    {
      "type": "pie",
      "title": "Product Categories",
      "dataSource": "form_sales",
      "categoryField": "product_category",
      "valueField": "sales_amount"
    },
    {
      "type": "line",
      "title": "Sales Trend",
      "dataSource": "form_daily_sales",
      "xField": "date",
      "yField": "sales_amount",
      "aggregation": "sum"
    },
    {
      "type": "table",
      "title": "Sales Detail",
      "dataSource": "form_sales",
      "columns": [
        { "field": "order_no", "title": "Order No.", "width": 150 },
        { "field": "customer_name", "title": "Customer", "width": 200 },
        { "field": "product_name", "title": "Product", "width": 200 },
        { "field": "amount", "title": "Amount", "width": 120 },
        { "field": "sales_date", "title": "Date", "width": 150 }
      ],
      "pageSize": 10
    }
  ]
}
```

Criar o relatório:

```bash
openyida create-report APP_XXXXXX "Sales Analysis Report" ./sales-report-charts.json
```

## Melhores práticas

1. **Escolha o tipo de gráfico apropriado** - Gráficos de linhas para tendências, de barras para comparações, circulares para proporções
2. **Limite a quantidade de gráficos** - Mantenha 6 gráficos ou menos por relatório
3. **Inclua tabelas de dados** - Combine gráficos com tabelas para inspeção detalhada dos dados
4. **Configure tamanhos de página razoáveis** - Para tabelas de dados, use 10-20 linhas por página
