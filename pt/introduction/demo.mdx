## de Negócios — IPD / CRM

Descreva os requisitos em uma frase, a IA gera automaticamente sistemas de negócios completos com múltiplos formulários.

**Exemplo de Prompt:**
```
Ajude-me a criar um sistema IPD usando Yida para gerenciar todo o processo de produção de chips
```

##ário de Operações — Jogo de Adivinhação de Enigmas de Lanterna

A IA gera imagens de enigmas de lanterna, os usuários adivinham respostas, com dicas humorísticas da IA para respostas erradas.

**Exemplo de Prompt:**
```
Crie um jogo de adivinhação de enigmas de lanterna para o Festival das Lanternas, a IA gera imagens de enigmas, os usuários enviam respostas, com respostas humorísticas da IA para respostas erradas
```
