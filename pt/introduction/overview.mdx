## Visão geral

**OpenYida** é uma ferramenta CLI sem configuração que conecta assistentes de codificação IA à plataforma low-code [Yida](https://www.aliwork.com). Instale uma vez, depois descreva seus requisitos em linguagem natural — o OpenYida lida automaticamente com: criação de aplicativos, gerenciamento de formulários, publicação de páginas, configuração de conectores, geração de relatórios e implantação no Yida em segundos.

🌐 **Site oficial**: [https://openyida.ai](https://openyida.ai)

## Funcionalidades principais

| Módulo de funcionalidade | Descrição |
|---------|------|
| **Gerenciamento de aplicativos** | Criação de aplicativos, importação/exportação, migração e implantação |
| **Gerenciamento de formulários** | Criação/atualização de formulários, obtenção de esquema, configuração de permissões |
| **Publicação de páginas** | Desenvolvimento de páginas personalizadas, compilação Babel, publicação com um clique |
| **Configuração de fluxos** | Criação de formulários de fluxo, configuração de regras de aprovação |
| **Gerenciamento de dados** | Consulta de dados de formulários, instâncias de fluxo, gerenciamento de tarefas |
| **Conectores** | Criação de conectores HTTP, integração de API, importação inteligente curl |
| **Relatórios e gráficos** | Criação de relatórios Yida, adição de gráficos, visualização de dados |
| **Gerenciamento CDN** | Upload de imagens, atualização de cache, configuração OSS |

## Diferenças de outras plataformas de construção IA

| Dimensão | OpenYida | Outras plataformas de construção IA |
|------|----------|------------------|
| **Usuários-alvo** | Desenvolvedores (pessoas que entendem código) | Usuários de negócios (não desenvolvedores) |
| **Modo de interação** | Linguagem natural + diálogo IA | Arrastar e soltar visual + painéis de configuração |
| **Entregáveis** | Aplicativo Yida (editável posteriormente, suporte completo de low-code) | Configuração (execução de caixa preta) |
| **Modo de implantação** | Plataforma Yida | Vinculação de plataforma SaaS |
| **Modelo IA** | Seleção conforme necessidade, escolher o melhor modelo | Especificado pela plataforma, não modificável |
| **Segurança e conformidade** | Yida possui capacidades completas de segurança e conformidade | Dependente de capacidades da plataforma |
