## O que é OpenYida?

OpenYida é uma interface de linha de comando que conecta agentes de IA à plataforma low-code [Yida](https://www.aliwork.com). Permite criar, gerenciar e implantar aplicativos empresariais usando prompts em linguagem natural.

## Início rápido

```bash
# Instalação
npm install -g openyida

# Login
openyida login

# Criar app
openyida create-app --name "Meu CRM"
```
