## Pré-requisitos

Antes de começar, certifique-se de ter as seguintes dependências instaladas:

| Dependência | Versão | Comando de verificação |
|-------------|--------|----------------------|
| [Node.js](https://nodejs.org) | ≥ 18 | `node --version` |

<Tip>
  Apenas Node.js é necessário. Não é necessário Python, Playwright ou etapas de compilação — implementação pura Node.js.
</Tip>

---

## Etapa 1 — Instalação

```bash
npm install -g openyida
```

Verifique a instalação:

```bash
openyida --version
```

---

## Etapa 2 — Login no Yida

```bash
openyida login
```

O OpenYida abrirá uma janela do navegador (ou exibirá um código QR) para você se autenticar com sua conta Yida/Alibaba. As credenciais são armazenadas em cache localmente — apenas uma vez.

<Info>
  Se você usar o **Wukong** como ferramenta de IA, use a flag de login especial:

  ```bash
  openyida login --wukong
  ```
</Info>

---

## Etapa 3 — Verificar ambiente

```bash
openyida env
```

Detecta a ferramenta de codificação com IA atualmente ativa (Claude Code, Cursor, Wukong, etc.) e exibe seu status de login. Execute este comando a qualquer momento ao alternar ambientes.

---

## Etapa 4 — Inicializar espaço de trabalho do projeto

```bash
openyida copy
```

Copia o modelo de espaço de trabalho `project/` para o diretório de trabalho atual da sua ferramenta de IA, permitindo que a IA acesse sua configuração de aplicativo e modelos de página.

---

## Etapa 5 — Começar a construir

Fale diretamente com seu assistente de codificação com IA. Aqui estão alguns exemplos de prompts para começar:

```
Construa um sistema IPD no Yida para gerenciar todo o fluxo de trabalho de produção de chips
```

```
Ajude-me a construir um CRM com leads, clientes e transações
```

```
Crie um aplicativo de calculadora de salário pessoal
```

```
Gere um mini jogo de felicitação de aniversário no Yida
```

A IA usará o pacote de habilidades do OpenYida para chamar os comandos CLI na ordem correta: `create-app` → `create-form` / `create-page` → `publish` → `save-share-config`.

---

## Etapa 6 — Publicar aplicativo

Assim que a IA gerar o aplicativo, execute o comando de publicação:

```bash
openyida publish
```

Ou pergunte à IA:
```
Publique a página [nome da página]
Torne a página acessível publicamente
```

---

## Executar diagnóstico

Se tiver problemas, execute a ferramenta de diagnóstico integrada:

```bash
openyida doctor
```

Verifica sua versão do Node.js, status de login e configuração do ambiente, e pode usar `--fix` para reparar automaticamente problemas comuns.
