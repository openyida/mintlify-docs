Este guia mostra como instalar o OpenYida e criar um aplicativo básico.

## Passo 1 — Instalação

```bash
npm install -g openyida
```

Verifique se a instalação foi bem-sucedida:

```bash
openyida --version
```

## Passo 2 — Login

```bash
openyida login
```

Isso abrirá uma janela do navegador para autenticação. Após o login bem-sucedido, você pode fechar o navegador e retornar ao terminal.

## Passo 3 — Criar um aplicativo

### Opção A: Criar com descrição em linguagem natural

```bash
openyida create-app --prompt "Crie um sistema CRM com gestão de clientes, acompanhamento de vendas e painel de análises"
```

### Opção B: Criar com arquivo de requisitos

```bash
openyida create-app --file requirements.md
```

## Passo 4 — Gerenciar formulários

### Listar formulários

```bash
openyida list-forms --app-id YOUR_APP_ID
```

### Criar um novo formulário

```bash
openyida create-form --app-id YOUR_APP_ID --prompt "Formulário de informações do cliente com nome, empresa, telefone, email e observações"
```

### Atualizar um formulário existente

```bash
openyida update-form --form-id YOUR_FORM_ID --prompt "Adicionar campo de status com opções: Lead, Oportunidade, Cliente"
```

## Passo 5 — Publicar páginas

### Compilar e publicar uma página personalizada

```bash
openyida publish-page --app-id YOUR_APP_ID --file custom-page.jsx
```

### Configurar acesso público

```bash
openyida set-public --page-id YOUR_PAGE_ID
```

## Passo 6 — Configurar conectores

### Criar um conector HTTP

```bash
openyida create-connector --app-id YOUR_APP_ID --name "API de Pagamento" --config connector-config.json
```

### Importar de curl

```bash
openyida import-curl --app-id YOUR_APP_ID --curl 'curl -X GET "https://api.example.com/data" -H "Authorization: Bearer token"'
```
