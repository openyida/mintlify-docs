## Tạo Báo Cáo

### Sử Dụng Cơ Bản

```bash
openyida create-report <appType> "Tên Báo Cáo" '<JSON định nghĩa biểu đồ>'
```

### Ví Dụ: Tạo Báo Cáo Bán Hàng

```bash
openyida create-report APP_XXXXXX "Báo Cáo Bán Hàng Hàng Tháng" '{
  "charts": [
    {
      "type": "bar",
      "title": "Doanh Số Hàng Tháng",
      "dataSource": "form_sales",
      "xField": "month",
      "yField": "amount"
    },
    {
      "type": "pie",
      "title": "Tỷ Lệ Bán Hàng Theo Sản Phẩm",
      "dataSource": "form_sales",
      "categoryField": "product_type",
      "valueField": "amount"
    }
  ]
}'
```

### Tải Định Nghĩa Biểu Đồ Từ Tệp

```bash
openyida create-report APP_XXXXXX "Báo Cáo Thống Kê Hàng Năm" ./charts-config.json
```

## Định Dạng Định Nghĩa Biểu Đồ

### Biểu Đồ Cột

```json
{
  "type": "bar",
  "title": "Tiêu Đề Biểu Đồ Cột",
  "dataSource": "form_uuid",
  "xField": "category_field",
  "yField": "value_field",
  "aggregation": "sum"
}
```

### Biểu Đồ Đường

```json
{
  "type": "line",
  "title": "Tiêu Đề Biểu Đồ Xu Hướng",
  "dataSource": "form_uuid",
  "xField": "date_field",
  "yField": "value_field",
  "aggregation": "sum"
}
```

### Biểu Đồ Tròn

```json
{
  "type": "pie",
  "title": "Tiêu Đề Biểu Đồ Tỷ Lệ",
  "dataSource": "form_uuid",
  "categoryField": "category_field",
  "valueField": "value_field"
}
```

### Bảng Dữ Liệu

```json
{
  "type": "table",
  "title": "Chi Tiết Dữ Liệu",
  "dataSource": "form_uuid",
  "columns": [
    { "field": "name", "title": "Tên" },
    { "field": "amount", "title": "Số Tiền" },
    { "field": "date", "title": "Ngày" }
  ]
}
```

## Thêm Biểu Đồ Vào Báo Cáo

```bash
openyida append-chart <appType> <reportId> '<JSON định nghĩa biểu đồ>'
```

### Ví Dụ

```bash
openyida append-chart APP_XXXXXX REPORT_XXXXX '{
  "type": "line",
  "title": "Xu Hướng Tăng Trưởng",
  "dataSource": "form_sales",
  "xField": "month",
  "yField": "growth_rate"
}'
```

```bash

```

## Tham Khảo Loại Biểu Đồ

| Loại | Mô Tả | Trường Bắt Buộc |
|------|------|----------|
| `bar` | Biểu đồ cột | xField, yField |
| `line` | Biểu đồ đường | xField, yField |
| `pie` | Biểu đồ tròn | categoryField, valueField |
| `table` | Bảng dữ liệu | columns |
| `area` | Biểu đồ vùng | xField, yField |
| `scatter` | Biểu đồ phân tán | xField, yField |

---

## Hàm Tổng Hợp

| Hàm | Mô Tả |
|------|------|
| `sum` | Tổng |
| `avg` | Trung bình |
| `count` | Đếm |
| `max` | Giá trị lớn nhất |
| `min` | Giá trị nhỏ nhất |

---

## Thực Hành Tốt Nhất

1. **Chọn loại biểu đồ phù hợp** - Xu hướng dùng đường, so sánh dùng cột, tỷ lệ dùng tròn
2. **Kiểm soát số lượng biểu đồ** - Khuyến nghị không quá 6 biểu đồ mỗi báo cáo
3. **Sử dụng bảng dữ liệu** - Bổ sung biểu đồ bằng dữ liệu chi tiết để xem giá trị gốc
4. **Cấu hình phân trang phù hợp** - Khuyến nghị 10-20 bản ghi mỗi trang cho bảng
