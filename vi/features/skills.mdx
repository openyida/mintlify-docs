OpenYida cung cấp các lệnh CLI phong phú, hỗ trợ quản lý toàn bộ vòng đời ứng dụng.

---

## Môi Trường và Xác Thực

```bash
# Phát hiện môi trường
openyida env

# Đăng nhập
openyida login [--qr]              # --qr sử dụng mã QR trong terminal
openyida login --wukong            # Dành riêng cho môi trường Wukong

# Đăng xuất
openyida logout

# Quản lý phiên đăng nhập
openyida auth status               # Xem trạng thái đăng nhập
openyida auth login                # Thực hiện đăng nhập
openyida auth refresh              # Làm mới phiên
openyida auth logout               # Đăng xuất

# Quản lý tổ chức
openyida org list                  # Liệt kê các tổ chức có thể truy cập
openyida org switch --corp-id <id> # Chuyển tổ chức

# Khởi tạo thư mục làm việc
openyida copy [--force]            # Sao chép thư mục project

# Công cụ chẩn đoán
openyida doctor [--fix]            # Chẩn đoán môi trường
openyida doctor --monitor          # Giám sát sức khỏe
openyida doctor --report <format>  # Tạo báo cáo (json|markdown|html)
```

## Quản Lý Ứng Dụng

```bash
# Tạo ứng dụng
openyida create-app "<tên>" [desc] [icon] [color] [themeColor]

# Xuất ứng dụng
openyida export <appType> [output] # Tạo gói di chuyển

# Nhập ứng dụng
openyida import <file> [name]      # Tái tạo ứng dụng từ gói
```

## Quản Lý Biểu Mẫu

```bash
# Tạo biểu mẫu
openyida create-form create <appType> "<tên>" <trườngJSON> \
  [--layout <bố_cục>] \
  [--theme <chủ_đề>] \
  [--label-align <căn_chỉnh>]

# Cập nhật biểu mẫu
openyida create-form update <appType> <formUuid> <sửa_đổiJSON>

# Lấy Schema biểu mẫu
openyida get-schema <appType> <formUuid>

# Cập nhật cấu hình biểu mẫu
openyida update-form-config <appType> <formUuid> <isRenderNav> <tiêu_đề>
```

### Các loại trường được hỗ trợ

Lệnh `create-form` chỉ chấp nhận các loại trường sau. Sử dụng loại không được hỗ trợ sẽ trả về lỗi.

#### Trường cơ bản

Trường tích hợp sẵn, không cần cài đặt plugin.

| Loại | Mô tả |
|------|-------|
| `TextField` | Văn bản một dòng |
| `TextareaField` | Văn bản nhiều dòng |
| `NumberField` | Số |
| `RateField` | Đánh giá |
| `DateField` | Ngày |
| `RadioField` | Nút radio |
| `SelectField` | Danh sách thả xuống (chọn một) |
| `CheckboxField` | Hộp kiểm |
| `MultiSelectField` | Danh sách thả xuống (chọn nhiều) |
| `CascadeSelectField` | Chọn dạng chuỗi |
| `CascadeDateField` | Ngày dạng chuỗi |
| `EmployeeField` | Bộ chọn thành viên |
| `DepartmentSelectField` | Bộ chọn phòng ban |
| `AssociationFormField` | Biểu mẫu liên kết |
| `AssociationQuery` | Truy vấn liên kết |
| `AttachmentField` | Tệp đính kèm |
| `ImageField` | Hình ảnh |
| `AddressField` | Địa chỉ |
| `CountrySelectField` | Bộ chọn quốc gia/vùng |
| `TableField` | Biểu mẫu con / chi tiết |
| `SerialNumberField` | Số thứ tự |

#### Trường nâng cao

Trường tích hợp sẵn với chức năng mở rộng. Một số tính năng có thể cần kích hoạt.

| Loại | Mô tả |
|------|-------|
| `EditorField` | Trình soạn thảo văn bản phong phú |
| `DigitalSignatureField` | Chữ ký số |
| `LocationField` | Vị trí GPS |

#### Trường plugin

Các trường này yêu cầu cài đặt plugin tương ứng từ trung tâm plugin trước khi sử dụng.

| Loại | Plugin cần thiết |
|------|------------------|
| `TimeZoneDateField` | Thành phần múi giờ quốc tế |
| `CC_AIButtonField` | Thành phần AI |
| `CC_PG_ESignField` | Thành phần tài chính thông minh |
| `CC_TeambitionProjectLinksField` | Plugin Teambition |
| `CC_TeambitionTaskLinksField` | Plugin Teambition |
| `CC_MoneyField` | Thành phần tài chính thông minh |
| `CC_PhoneNumberField` | Thành phần mở rộng |
| `CC_ChineseIdField` | Thành phần mở rộng |
| `YidaDingtalkAgentField` | Plugin đại lý DingTalk |

<Warning>
Trường plugin sẽ thất bại nếu plugin cần thiết chưa được cài đặt trong tổ chức của bạn. Cài đặt plugin từ trung tâm plugin trước khi sử dụng các loại trường này.
</Warning>

<Warning>
Các loại trường như `MobileField` và `EmailField` không được hỗ trợ và sẽ bị từ chối. Chỉ sử dụng các loại được liệt kê ở trên.
</Warning>

## Quản Lý Trang

```bash
# Tạo trang tùy chỉnh
openyida create-page <appType> "<tên>" [--datasource <jsonOrFile>]

# Biên dịch và xuất bản trang
openyida publish <đường_dẫn_tệp_nguồn> <appType> <formUuid>

# Cấu hình chia sẻ trang
openyida save-share-config <appType> <formUuid> <url> <isOpen> [openAuth]
openyida get-page-config <appType> <formUuid>
openyida verify-short-url <appType> <formUuid> <url>
```

## Quản Lý Dữ Liệu

```bash
# Quản lý dữ liệu thống nhất
openyida data <hành_động> <tài_nguyên> [args]

# Dữ liệu biểu mẫu
openyida data query form <appType> <formUuid> [--page N] [--size N] [--search-json JSON]
openyida data get form <appType> --inst-id <id>
openyida data create form <appType> <formUuid> --data <JSON>
openyida data update form <appType> --inst-id <id> --data <JSON>

# Dữ liệu quy trình
openyida data query process <appType> <formUuid> [--instance-status TRẠNG_THÁI]
openyida data get process <appType> --process-inst-id <id>
openyida data create process <appType> <formUuid> --process-code <mã> --data <JSON>

# Quản lý tác vụ
openyida data query tasks <appType> --type <todo|done|submitted|cc>
openyida data execute task <appType> --task-id <id> --process-inst-id <id> --out-result <AGREE|DISAGREE>

# Biểu mẫu con
openyida data query subform <appType> <formUuid> --inst-id <id> --table-field-id <fieldId>
```

## Quản Lý Quy Trình

```bash
# Tạo biểu mẫu quy trình (biểu mẫu mới + cấu hình quy trình)
openyida create-process <appType> "<tên>" <tệpTrườngJSON> <tệpĐịnhNghĩaQuyTrình>

# Tạo biểu mẫu quy trình (tái sử dụng biểu mẫu hiện có + cấu hình quy trình)
openyida create-process <appType> --formUuid <formUuid> <tệpĐịnhNghĩaQuyTrình>

# Cấu hình quy tắc quy trình từ tệp định nghĩa
openyida configure-process <appType> <formUuid> <tệpĐịnhNghĩaQuyTrình> [processCode]
```

## Quản Lý Connector

```bash
# Liệt kê connector
openyida connector list [--page N] [--size N]

# Tạo connector
openyida connector create "<tên>" "<miền>" --operations <tệp> [tùy_chọn]

# Xem chi tiết connector
openyida connector detail <connector-id>

# Xóa connector
openyida connector delete <connector-id> [--force]

# Quản lý action
openyida connector add-action --connector-id <id> --operations <tệp> [--confirm]
openyida connector list-actions <connector-id>
openyida connector delete-action <connector-id> <operation-id> [--force]
openyida connector test --connector-id <id> --action <actionId> [tùy_chọn]

# Quản lý tài khoản xác thực
openyida connector list-connections <connector-id>
openyida connector create-connection <connector-id> "<tên>" [tùy_chọn]

# Tạo thông minh
openyida connector smart-create --curl "<lệnh_curl>" [--name <tên>] [--desc <mô_tả>]

# Công cụ hỗ trợ
openyida connector parse-api --curl "<lệnh_curl>" --output <tệp>
openyida connector gen-template [đường_dẫn_xuất]
```

## Tùy Chọn Toàn Cục

| Tùy chọn | Mô tả |
|------|------|
| `--help`, `-h` | Hiển thị thông tin trợ giúp |
| `--version`, `-v` | Hiển thị số phiên bản |
| `--qr` | Sử dụng mã QR trong terminal để đăng nhập |
| `--force` | Buộc thực hiện, bỏ qua xác nhận |
