## Tổng quan

**OpenYida** là công cụ CLI không cần cấu hình, kết nối trợ lý lập trình AI với nền tảng low-code [Yida](https://www.aliwork.com). Cài đặt một lần, sau đó mô tả yêu cầu bằng ngôn ngữ tự nhiên — OpenYida sẽ tự động xử lý: tạo ứng dụng, quản lý biểu mẫu, xuất bản trang, cấu hình kết nối, tạo báo cáo, triển khai lên Yida trong vài giây.

🌐 **Trang web chính thức**: [https://openyida.ai](https://openyida.ai)

## Tính năng cốt lõi

| Mô-đun tính năng | Mô tả |
|---------|------|
| **Quản lý ứng dụng** | Tạo ứng dụng, nhập/xuất, di chuyển và triển khai |
| **Quản lý biểu mẫu** | Tạo/cập nhật biểu mẫu, lấy Schema, cấu hình quyền |
| **Xuất bản trang** | Phát triển trang tùy chỉnh, biên dịch Babel, xuất bản bằng một cú nhấp chuột |
| **Cấu hình quy trình** | Tạo biểu mẫu quy trình, cấu hình quy tắc phê duyệt |
| **Quản lý dữ liệu** | Truy vấn dữ liệu biểu mẫu, quy trình, quản lý tác vụ |
| **Kết nối** | Tạo kết nối HTTP, tích hợp API, nhập thông minh curl |
| **Báo cáo và biểu đồ** | Tạo báo cáo Yida, thêm biểu đồ, trực quan hóa dữ liệu |
| **Quản lý CDN** | Tải lên hình ảnh, làm mới bộ nhớ đệm, cấu hình OSS |

## Khác biệt với các nền tảng xây dựng AI khác

| Chiều | OpenYida | Các nền tảng xây dựng AI khác |
|------|----------|------------------|
| **Người dùng mục tiêu** | Nhà phát triển (người biết code) | Nhân viên kinh doanh (không phải nhà phát triển) |
| **Cách tương tác** | Ngôn ngữ tự nhiên + hội thoại AI | Kéo thả trực quan + bảng cấu hình |
| **Sản phẩm đầu ra** | Ứng dụng Yida (có thể chỉnh sửa lại, hỗ trợ đầy đủ khả năng low-code) | Cấu hình (chạy trong hộp đen) |
| **Cách triển khai** | Nền tảng Yida | Liên kết nền tảng SaaS |
| **Mô hình AI** | Chọn theo nhu cầu, chọn mô hình phù hợp nhất | Nền tảng chỉ định, không thể thay đổi |
| **Bảo mật và tuân thủ** | Yida có khả năng bảo mật và tuân thủ hoàn chỉnh | Phụ thuộc vào khả năng nền tảng |
