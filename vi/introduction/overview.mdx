## OpenYida là gì?

OpenYida là giao diện dòng lệnh kết nối các tác nhân AI với nền tảng low-code [Yida](https://www.aliwork.com). Nó cho phép tạo, quản lý và triển khai các ứng dụng doanh nghiệp bằng cách sử dụng các lệnh bằng ngôn ngữ tự nhiên.

## Bắt đầu nhanh

```bash
# Cài đặt
npm install -g openyida

# Đăng nhập
openyida login

# Tạo ứng dụng
openyida create-app --name "CRM của tôi"
```
