Hướng dẫn này chỉ cách cài đặt OpenYida và tạo một ứng dụng cơ bản.

## Bước 1 — Cài đặt

```bash
npm install -g openyida
```

Kiểm tra cài đặt thành công:

```bash
openyida --version
```

## Bước 2 — Đăng nhập

```bash
openyida login
```

Thao tác này sẽ mở cửa sổ trình duyệt để xác thực. Sau khi đăng nhập thành công, bạn có thể đóng trình duyệt và quay lại terminal.

## Bước 3 — Tạo ứng dụng

### Phương án A: Tạo với mô tả ngôn ngữ tự nhiên

```bash
openyida create-app --prompt "Tạo hệ thống CRM với quản lý khách hàng, theo dõi bán hàng và bảng điều khiển phân tích"
```

### Phương án B: Tạo với tệp yêu cầu

```bash
openyida create-app --file requirements.md
```

## Bước 4 — Quản lý biểu mẫu

### Liệt kê biểu mẫu

```bash
openyida list-forms --app-id YOUR_APP_ID
```

### Tạo biểu mẫu mới

```bash
openyida create-form --app-id YOUR_APP_ID --prompt "Biểu mẫu thông tin khách hàng với tên, công ty, điện thoại, email và ghi chú"
```

### Cập nhật biểu mẫu hiện có

```bash
openyida update-form --form-id YOUR_FORM_ID --prompt "Thêm trường trạng thái với các tùy chọn: Khách hàng tiềm năng, Cơ hội, Khách hàng"
```

## Bước 5 — Xuất bản trang

### Biên dịch và xuất bản trang tùy chỉnh

```bash
openyida publish-page --app-id YOUR_APP_ID --file custom-page.jsx
```

### Cấu hình truy cập công khai

```bash
openyida set-public --page-id YOUR_PAGE_ID
```

## Bước 6 — Cấu hình trình kết nối

### Tạo trình kết nối HTTP

```bash
openyida create-connector --app-id YOUR_APP_ID --name "API Thanh toán" --config connector-config.json
```

### Nhập từ curl

```bash
openyida import-curl --app-id YOUR_APP_ID --curl 'curl -X GET "https://api.example.com/data" -H "Authorization: Bearer token"'
```
