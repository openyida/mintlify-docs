## Yêu cầu trước khi bắt đầu

Trước khi bắt đầu, hãy đảm bảo bạn đã cài đặt các phụ thuộc sau:

| Phụ thuộc | Phiên bản | Lệnh kiểm tra |
|-----------|-----------|---------------|
| [Node.js](https://nodejs.org) | ≥ 18 | `node --version` |

<Tip>
  Chỉ cần Node.js. Không cần Python, Playwright hoặc các bước xây dựng — triển khai thuần Node.js.
</Tip>

---

## Bước 1 — Cài đặt

```bash
npm install -g openyida
```

Xác minh cài đặt:

```bash
openyida --version
```

---

## Bước 2 — Đăng nhập Yida

```bash
openyida login
```

OpenYida sẽ mở cửa sổ trình duyệt (hoặc hiển thị mã QR) để bạn xác thực bằng tài khoản Yida/Alibaba của mình. Thông tin đăng nhập được lưu trong bộ nhớ đệm cục bộ — chỉ cần một lần.

<Info>
  Nếu bạn sử dụng **Wukong** làm công cụ AI, hãy sử dụng cờ đăng nhập đặc biệt:

  ```bash
  openyida login --wukong
  ```
</Info>

---

## Bước 3 — Kiểm tra môi trường

```bash
openyida env
```

Phát hiện công cụ mã hóa AI hiện đang hoạt động (Claude Code, Cursor, Wukong, v.v.) và hiển thị trạng thái đăng nhập của bạn. Chạy lệnh này bất cứ lúc nào khi chuyển đổi môi trường.

---

## Bước 4 — Khởi tạo không gian làm việc dự án

```bash
openyida copy
```

Sao chép mẫu không gian làm việc `project/` vào thư mục làm việc hiện tại của công cụ AI, cho phép AI truy cập cấu hình ứng dụng và mẫu trang của bạn.

---

## Bước 5 — Bắt đầu xây dựng

Nói trực tiếp với trợ lý mã hóa AI của bạn. Dưới đây là một số ví dụ prompt để bắt đầu:

```
Xây dựng cho tôi một hệ thống IPD trên Yida để quản lý toàn bộ quy trình sản xuất chip
```

```
Giúp tôi xây dựng một CRM với khách hàng tiềm năng, khách hàng và giao dịch
```

```
Tạo một ứng dụng máy tính lương cá nhân
```

```
Tạo một trò chơi mini chúc mừng sinh nhật trên Yida
```

AI sẽ sử dụng gói kỹ năng OpenYida để gọi các lệnh CLI theo đúng thứ tự: `create-app` → `create-form` / `create-page` → `publish` → `save-share-config`.

---

## Bước 6 — Xuất bản ứng dụng

Khi AI đã tạo ứng dụng, hãy chạy lệnh xuất bản:

```bash
openyida publish
```

Hoặc hỏi AI:
```
Xuất bản trang [tên trang]
Đặt trang thành truy cập công khai
```

---

## Chạy chẩn đoán

Nếu gặp sự cố, hãy chạy công cụ chẩn đoán tích hợp:

```bash
openyida doctor
```

Nó kiểm tra phiên bản Node.js, trạng thái đăng nhập và cấu hình môi trường của bạn, và có thể sử dụng `--fix` để tự động sửa các vấn đề phổ biến.

---

## Tích hợp OpenClaw

Nếu bạn sử dụng OpenClaw, hãy cài đặt trực tiếp gói kỹ năng OpenYida:

```bash
npx clawhub@latest install nicky1108/yida-app
```

Sau đó sử dụng nó qua [yida-app trên ClawHub](https://clawhub.ai/nicky1108/yida-app).
