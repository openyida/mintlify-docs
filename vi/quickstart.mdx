## 环境准备

开始前，请确保已安装以下依赖：

| 依赖 | 版本 | 检查命令 |
|------|------|----------|
| [Node.js](https://nodejs.org) | ≥ 18 | `node --version` |

<Tip>
  仅需 Node.js，无需 Python、Playwright、构建步骤 — 纯 Node.js 实现。
</Tip>

---

## 步骤 1 — 安装

```bash
npm install -g openyida
```

验证安装：

```bash
openyida --version
```

---

## 步骤 2 — 登录宜搭

```bash
openyida login
```

OpenYida 会打开浏览器窗口（或显示二维码）供你使用宜搭/阿里账号认证。凭证本地缓存，只需登录一次。

<Info>
  如使用 **悟空** 作为 AI 工具，请使用特殊登录参数：

  ```bash
  openyida login --wukong
  ```
</Info>

---

## 步骤 3 — 检查环境

```bash
openyida env
```

检测当前激活的 AI 工具环境（Claude Code、Cursor、悟空等）并显示登录状态。切换环境时建议运行此命令。

---

## 步骤 4 — 初始化工作目录

```bash
openyida copy
```

将 `project/` 工作目录模板复制到当前 AI 工具的工作目录，使 AI 能够访问应用配置和页面模板。

---

## 步骤 5 — 开始构建

直接与 AI 编程助手对话，以下是一些启动提示：

```
帮我用宜搭创建一个 IPD 系统，管理芯片生产全流程
```

```
帮我搭建一个 CRM，包含线索、客户和商机
```

```
帮我搭建个人薪资计算器应用
```

```
帮我创建一个生日祝福小游戏
```

AI 会使用 OpenYida 技能包按序调用 CLI 命令：`create-app` → `create-form` / `create-page` → `publish` → `save-share-config`。

---

## 步骤 6 — 发布应用

AI 生成应用后，执行发布：

```bash
openyida publish
```

或让 AI 执行：
```
发布 xxx 页面
将页面设置为公开访问
```

---

## 运行诊断

如遇问题，运行内置诊断工具：

```bash
openyida doctor
```

检查 Node.js 版本、登录状态、环境配置，支持 `--fix` 自动修复常见问题。

---

## OpenClaw 集成
