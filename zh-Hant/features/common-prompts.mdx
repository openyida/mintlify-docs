OpenYida 專爲自然語言交互設計。以下提示詞經過驗證，可在所有支持的 AI 工具中使用。

---

## 應用創建

```
幫我搭建一個 xxx 應用
```
```
幫我搭建一個 CRM
```
```
幫我搭建個人薪資計算器應用
```
```
幫我用宜搭創建一個 IPD 系統，管理芯片生產全流程
```
```
根據需求文檔生成應用：[粘貼 PRD]
```
```
在宜搭上創建一個生日祝福小遊戲
```
```
搭建一個元宵節猜燈謎遊戲
```

---

## 表單與頁面操作

### 創建表單頁面

```
幫我創建一個 xxx 表單頁面
```
```
創建一個請假申請表單，包含：員工姓名、部門、開始日期、結束日期、請假原因
```

### 添加表單字段

```
給 [頁面名] 添加一個 [字段名] 字段，字段類型：[文本/數字/日期/下拉框/...]
```
```
給員工信息表單添加一個"部門"下拉框字段，選項：工程部、市場部、銷售部、人事部
```

### 修改字段屬性

```
把 [頁面名] 的 [字段名] 字段設爲必填
```
```
把 [頁面名] 的 [字段名] 字段設爲只讀
```
```
把 [頁面名] 的 [字段名] 字段標籤改爲 [新標籤]
```

### 自定義展示頁面

```
給 [頁面名] 創建一個自定義展示頁面
```
```
生成一個企業協作產品的落地頁
```

## 發佈與分享

```
發佈 [頁面名] 頁面
```
```
把 [頁面名] 頁面設爲公開訪問
```
```
配置 [頁面名] 的公開訪問權限 — 任何人可通過鏈接訪問
```

## 數據管理

```
查詢 [表單名] 的所有記錄
```
```
把 [表單名] 的數據導出爲 Excel
```
```
刪除 [表單名] 的所有測試數據
```

## 連接器（HTTP 集成）

```
爲 [API 名稱] 創建一個 HTTP 連接器，基礎 URL：[url]
```
```
給 [連接器名] 添加一個 POST 動作用於創建記錄
```
```
從這個 curl 命令智能創建連接器：[粘貼 curl]
```

## 報表與圖表

```
爲 [應用名] 創建一個宜搭報表，包含月度提交量的柱狀圖
```
```
在 [報表名] 報表中添加一個顯示狀態分佈的餅圖
```

## 賬號與環境

```
重新登錄 / 退出登錄
```
```
切換到 [組織名] 組織
```
```
查看當前登錄狀態
```
```
運行環境診斷
```

## CDN 與資源

```
爲 [應用名] 配置 CDN 圖片上傳
```
```
上傳 [圖片文件] 到 CDN
```

---

<Tip>
**提示：** 提供的上下文越多，結果越好。與其說"創建一個表單"，不如說"創建一個請假審批表單，包含員工姓名、部門、請假類型（下拉框：年假/病假/事假）、起止日期、原因 —— 員工姓名和日期設爲必填"。
</Tip>
