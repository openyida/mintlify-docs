OpenYida 專爲自然語言交互設計。以下提示詞經過驗證，可在所有支持的 AI 工具中使用。

---

## 應用創建

```
幫我搭建一個 xxx 應用
```
```
幫我搭建一個 CRM
```
```
幫我搭建個人薪資計算器應用
```
```
幫我用宜搭創建一個 IPD 系統，管理芯片生產全流程
```
```
根據需求文檔生成應用：[粘貼 PRD]
```
```
在宜搭上創建一個生日祝福小遊戲
```
```
搭建一個元宵節猜燈謎遊戲
```

修改應用的描述和圖示

```
把 APP_XXX 應用改名為「客戶入口」
```

## 表單與頁面操作

### 創建表單頁面

```
幫我創建一個 xxx 表單頁面
```
```
創建一個請假申請表單，包含：員工姓名、部門、開始日期、結束日期、請假原因
```

### 添加表單字段

```
給 [頁面名] 添加一個 [字段名] 字段，字段類型：[TextField/NumberField/DateField/SelectField/...]
```
```
給員工信息表單添加一個"部門"下拉框字段，選項：工程部、市場部、銷售部、人事部
```

<Note>
僅接受[支援的欄位類型](/zh-Hant/features/skills#支援的欄位類型)。不支援的類型（如 `MobileField`、`EmailField`）會被拒絕。
</Note>

### 修改字段屬性

```
把 [頁面名] 的 [字段名] 字段設爲必填
```
```
把 [頁面名] 的 [字段名] 字段設爲只讀
```
```
把 [頁面名] 的 [字段名] 字段標籤改爲 [新標籤]
```

### 自定義展示頁面

```
給 [頁面名] 創建一個自定義展示頁面
```
```
生成一個企業協作產品的落地頁
```

## 閃記轉 PRD

```
將釘釘閃記轉換為 PRD 文檔
```

```
從會議記錄生成 PRD：[粘貼筆記]
```

```
從附件閃記創建項目需求文檔
```

```
將會議筆記轉換為結構化產品需求文檔
```

<Note>
需要登錄。該命令讀取會議記錄並使用 AI 生成結構化的 PRD 文檔。輸出保存到 `prd/` 目錄。
</Note>

## 集成與自動化

### 創建集成流程

```
創建一個集成流程，在新記錄提交時發送通知
```

```
在 [表單名] 上設置自動化，在新提交時通知 [團隊/用戶]
```

```
創建一個 [表單名] 工作流，觸發郵件通知給 [收件人]
```

### 添加自動化操作

```
向 [流程名] 集成添加數據查詢操作
```

```
添加記錄創建操作，自動在 [目標表單] 中創建條目
```

<Note>
集成流程在表單事件（插入、更新、刪除）時觸發，可以執行通知、數據查詢和記錄創建。使用 CLI 創建和配置流程。
</Note>

<Info>
`dws` 命令需要單獨安裝釘釘 CLI。如果尚未安裝，請運行 `openyida dws install`。輸出默認為表格格式；添加 `-f json` 可輸出 JSON 格式。

<Tip>
**提示：** 提供的上下文越多，結果越好。與其說"創建一個表單"，不如說"創建一個請假審批表單，包含員工姓名、部門、請假類型（下拉框：年假/病假/事假）、起止日期、原因 —— 員工姓名和日期設爲必填"。
</Tip>
