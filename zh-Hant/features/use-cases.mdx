---

## 🏢 企業業務系統

構建多模塊業務平臺，支持複雜表單、工作流和數據關聯。

<AccordionGroup>
  <Accordion title="IPD（集成產品開發）">
    **提示詞：**
    ```
    幫我用宜搭創建一個 IPD 系統，管理芯片生產全流程，
    包括項目立項、設計評審、流片、量產等階段
    ```
    **產出：** 多表單應用，包含階段跟蹤、評審清單、里程碑管理、跨團隊可見性。
  </Accordion>

  <Accordion title="CRM（客戶關係管理）">
    **提示詞：**
    ```
    幫我搭建一個 CRM，包含線索、客戶、商機、跟進記錄等模塊
    ```
    **產出：** 關聯表單：線索 → 客戶 → 商機 → 合同，含狀態跟蹤和分配字段。
  </Accordion>

  <Accordion title="人事與請假管理">
    **提示詞：**
    ```
    創建一個人事請假管理系統，包含請假申請、審批流程、假期餘額統計
    ```
    **產出：** 請假申請表單含審批流程、餘額計算器、經理看板。
  </Accordion>

  <Accordion title="項目管理">
    **提示詞：**
    ```
    搭建一個項目管理應用，包含任務、里程碑、團隊成員、進度跟蹤
    ```
    **產出：** 任務列表含優先級/狀態/負責人字段、里程碑跟蹤器、甘特圖視圖。
  </Accordion>
</AccordionGroup>

## 🔌 API 集成（連接器）

通過 HTTP 連接器將宜搭應用與外部系統連接。

<AccordionGroup>
  <Accordion title="Webhook 通知">
    **提示詞：**
    ```
    爲我們的釘釘 webhook 創建一個 HTTP 連接器，添加 POST 動作發送表單提交通知
    ```
    **產出：** 配置好的連接器含測試能力，可在表單事件時觸發。
  </Accordion>

  <Accordion title="從 curl 智能創建">
    **提示詞：**
    ```
    從這個 curl 命令智能創建連接器：curl -X POST https://api.example.com/v1/users -H "Authorization: Bearer TOKEN" -d '{"name":"test"}'
    ```
    **產出：** 自動配置的連接器，提取基礎 URL、請求頭、動作參數。
  </Accordion>
</AccordionGroup>

---

<Note>
這些是起點。由於產出是可在宜搭平臺完全編輯的應用（非黑盒配置），你隨時可以通過宜搭可視化編輯器或給 AI 後續指令來優化生成的應用。
</Note>
