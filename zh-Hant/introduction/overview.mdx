## 概述

**OpenYida** 是一個零配置 CLI 工具，連接 AI 編程助手與[宜搭 (Yida)](https://www.aliwork.com) 低代碼平臺。安裝一次，然後用自然語言描述需求 — OpenYida 會自動處理：創建應用、管理表單、發佈頁面、配置連接器、生成報表，秒級部署到宜搭。

🌐 **官方網站**: [https://openyida.ai](https://openyida.ai)

## 核心功能

| 功能模塊 | 說明 |
|---------|------|
| **應用管理** | 創建應用、導入導出、遷移部署 |
| **表單管理** | 創建/更新表單、獲取 Schema、配置權限 |
| **頁面發佈** | 自定義頁面開發、Babel 編譯、一鍵發佈 |
| **流程配置** | 流程表單創建、審批規則配置 |
| **整合與自動化** | 邏輯流建立與管理，表單事件觸發、訊息通知、跨表資料操作 |
| **數據管理** | 表單數據查詢、流程實例、任務管理 |
| **連接器** | HTTP 連接器創建、API 集成、curl 智能導入 |
| **報表圖表** | 宜搭報表創建、圖表追加、數據可視化 |
| **CDN 管理** | 圖片上傳、緩存刷新、OSS 配置 |

## 與其他 AI 搭建平臺的區別

| 維度 | OpenYida | 其他 AI 搭建平臺 |
|------|----------|------------------|
| **目標用戶** | 開發者（懂代碼的人） | 業務人員（非開發者） |
| **交互方式** | 自然語言 + AI 對話 | 可視化拖拽 + 配置面板 |
| **產出物** | 宜搭應用（可二次編輯，支持完備低代碼能力） | 配置（黑盒運行） |
| **部署方式** | 宜搭平臺 | SaaS 平臺綁定 |
| **AI 模型** | 按需選擇，選擇最適合的模型 | 平臺指定，無法更換 |
| **安全合規** | 宜搭具備完善的安全和合規能力 | 依賴平臺能力 |
