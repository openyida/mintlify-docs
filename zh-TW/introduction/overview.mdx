## 什麼是 OpenYida？

OpenYida 是一個命令列介面，可將 AI 代理與 [Yida](https://www.aliwork.com) 低程式碼平台連接。它允許您使用自然語言提示來建立、管理和部署企業應用程式。

## 主要功能

  </Card>
  <Card title="自訂頁面編譯" icon="code">
    基於 JSX 的自訂頁面會使用 Babel 編譯並自動發佈到 Yida。
  </Card>
</CardGroup>

## 快速開始

```bash
# 安裝
npm install -g openyida

# 登入
openyida login

# 建立應用程式
openyida create-app --name "我的 CRM"
```
