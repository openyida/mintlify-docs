## 環境準備

開始前，請確保已安裝以下相依項目：

| 相依項目 | 版本 | 檢查命令 |
|----------|------|----------|
| [Node.js](https://nodejs.org) | ≥ 18 | `node --version` |

<Tip>
  僅需 Node.js，無需 Python、Playwright、建置步驟 — 純 Node.js 實作。
</Tip>

---

## 步驟 1 — 安裝

```bash
npm install -g openyida
```

驗證安裝：

```bash
openyida --version
```

---

## 步驟 2 — 登入宜搭

```bash
openyida login
```

OpenYida 會開啟瀏覽器視窗（或顯示 QR 碼）供你使用宜搭/阿里帳號認證。憑證會在本機快取，只需登入一次。

<Info>
  如使用 **悟空** 作為 AI 工具，請使用特殊登入參數：

  ```bash
  openyida login --wukong
  ```
</Info>

---

## 步驟 3 — 檢查環境

```bash
openyida env
```

偵測目前啟用的 AI 工具環境（Claude Code、Cursor、悟空等）並顯示登入狀態。切換環境時建議執行此命令。

---

## 步驟 4 — 初始化工作目錄

```bash
openyida copy
```

將 `project/` 工作目錄範本複製到目前 AI 工具的工作目錄，使 AI 能夠存取應用程式設定和頁面範本。

---

## 步驟 5 — 開始建置

直接與 AI 程式設計助理對話，以下是一些啟動提示：

```
幫我用宜搭建立一個 IPD 系統，管理晶片生產完整流程
```

```
幫我搭建一個 CRM，包含線索、客戶和商機
```

```
幫我搭建個人薪資計算器應用程式
```

```
幫我建立一個生日祝福小遊戲
```

AI 會使用 OpenYida 技能包依序呼叫 CLI 命令：`create-app` → `create-form` / `create-page` → `publish` → `save-share-config`。

---

## 步驟 6 — 發布應用程式

AI 產生應用程式後，執行發布：

```bash
openyida publish
```

或讓 AI 執行：
```
發布 xxx 頁面
將頁面設為公開存取
```

---

## 執行診斷

如遇問題，執行內建診斷工具：

```bash
openyida doctor
```

檢查 Node.js 版本、登入狀態、環境設定，支援 `--fix` 自動修復常見問題。

---

## OpenClaw 整合

如使用 OpenClaw，直接安裝 OpenYida 技能包：

```bash
npx clawhub@latest install nicky1108/yida-app
```

然後透過 [ClawHub 上的 yida-app](https://clawhub.ai/nicky1108/yida-app) 使用。
