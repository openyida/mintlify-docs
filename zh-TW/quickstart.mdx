本指南將展示如何安裝 OpenYida 並創建一個基本應用。

## 步驟 1 — 安裝

```bash
npm install -g openyida
```

驗證安裝是否成功：

```bash
openyida --version
```

## 步驟 2 — 登入

```bash
openyida login
```

這將開啟瀏覽器視窗進行身份驗證。登入成功後，您可以關閉瀏覽器並返回終端機。

## 步驟 3 — 創建應用

### 方案 A：使用自然語言描述創建

```bash
openyida create-app --prompt "創建一個 CRM 系統，包含客戶管理、銷售追蹤和分析儀表板"
```

### 方案 B：使用需求文件創建

```bash
openyida create-app --file requirements.md
```

## 步驟 4 — 管理表單

### 列出表單

```bash
openyida list-forms --app-id YOUR_APP_ID
```

### 創建新表單

```bash
openyida create-form --app-id YOUR_APP_ID --prompt "客戶資訊表單，包含姓名、公司、電話、郵箱和備註"
```

### 更新現有表單

```bash
openyida update-form --form-id YOUR_FORM_ID --prompt "新增狀態欄位，選項：潛在客戶、機會、客戶"
```

## 步驟 5 — 發布頁面

### 編譯並發布自定義頁面

```bash
openyida publish-page --app-id YOUR_APP_ID --file custom-page.jsx
```

### 配置公開訪問

```bash
openyida set-public --page-id YOUR_PAGE_ID
```

## 步驟 6 — 配置連接器

### 創建 HTTP 連接器

```bash
openyida create-connector --app-id YOUR_APP_ID --name "支付 API" --config connector-config.json
```

### 從 curl 匯入

```bash
openyida import-curl --app-id YOUR_APP_ID --curl 'curl -X GET "https://api.example.com/data" -H "Authorization: Bearer token"'
```
