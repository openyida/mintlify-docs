OpenYida 专为自然语言交互设计。以下提示词经过验证，可在所有支持的 AI 工具中使用。

---

## 应用创建

```
帮我搭建一个 xxx 应用
```
```
帮我搭建一个 CRM
```
```
帮我搭建个人薪资计算器应用
```
```
帮我用宜搭创建一个 IPD 系统，管理芯片生产全流程
```
```
根据需求文档生成应用：[粘贴 PRD]
```
```
在宜搭上创建一个生日祝福小游戏
```
```
搭建一个元宵节猜灯谜游戏
```

---

## 表单与页面操作

### 创建表单页面

```
帮我创建一个 xxx 表单页面
```
```
创建一个请假申请表单，包含：员工姓名、部门、开始日期、结束日期、请假原因
```

### 添加表单字段

```
给 [页面名] 添加一个 [字段名] 字段，字段类型：[TextField/NumberField/DateField/SelectField/...]
```
```
给员工信息表单添加一个"部门"下拉框字段，选项：工程部、市场部、销售部、人事部
```

<Note>
仅接受[支持的字段类型](/zh/features/skills#支持的字段类型)。不支持的类型（如 `MobileField`、`EmailField`）会被拒绝。
</Note>

### 修改字段属性

```
把 [页面名] 的 [字段名] 字段设为必填
```
```
把 [页面名] 的 [字段名] 字段设为只读
```
```
把 [页面名] 的 [字段名] 字段标签改为 [新标签]
```

### 自定义展示页面

```
给 [页面名] 创建一个自定义展示页面
```
```
生成一个企业协作产品的落地页
```

## 发布与分享

```
发布 [页面名] 页面
```
```
把 [页面名] 页面设为公开访问
```
```
配置 [页面名] 的公开访问权限 — 任何人可通过链接访问
```

```
查询 [表单名] 的所有记录
```
```
把 [表单名] 的数据导出为 Excel

```

## 连接器（HTTP 集成）

```
为 [API 名称] 创建一个 HTTP 连接器，基础 URL：[url]
```
```
给 [连接器名] 添加一个 POST 动作用于创建记录
```
```
从这个 curl 命令智能创建连接器：[粘贴 curl]
```

```
为 [应用名] 创建一个宜搭报表，包含月度提交量的柱状图
```
```
在 [报表名] 报表中添加一个显示状态分布的饼图
```

```
重新登录 / 退出登录
```
```
切换到 [组织名] 组织

```

```
运行环境诊断
```

## CDN 与资源

```
为 [应用名] 配置 CDN 图片上传
```
```
上传 [图片文件] 到 CDN
```

---

<Tip>
**提示：** 提供的上下文越多，结果越好。与其说"创建一个表单"，不如说"创建一个请假审批表单，包含员工姓名、部门、请假类型（下拉框：年假/病假/事假）、起止日期、原因 —— 员工姓名和日期设为必填"。
</Tip>
