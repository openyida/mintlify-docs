OpenYida 专为自然语言交互设计。以下提示词经过验证，可在所有支持的 AI 工具中使用。

---

## 应用创建

```
帮我搭建一个 xxx 应用
```
```
帮我搭建一个 CRM
```
```
帮我搭建个人薪资计算器应用
```
```
帮我用宜搭创建一个 IPD 系统，管理芯片生产全流程
```
```
根据需求文档生成应用：[粘贴 PRD]
```
```
在宜搭上创建一个生日祝福小游戏
```
```
搭建一个元宵节猜灯谜游戏
```

## 表单与页面操作

### 创建表单页面

```
帮我创建一个 xxx 表单页面
```
```
创建一个请假申请表单，包含：员工姓名、部门、开始日期、结束日期、请假原因
```

### 添加表单字段

```
给 [页面名] 添加一个 [字段名] 字段，字段类型：[TextField/NumberField/DateField/SelectField/...]
```
```
给员工信息表单添加一个"部门"下拉框字段，选项：工程部、市场部、销售部、人事部
```

<Note>
仅接受[支持的字段类型](/zh/features/skills#支持的字段类型)。不支持的类型（如 `MobileField`、`EmailField`）会被拒绝。
</Note>

### 修改字段属性

```
把 [页面名] 的 [字段名] 字段设为必填
```
```
把 [页面名] 的 [字段名] 字段设为只读
```
```
把 [页面名] 的 [字段名] 字段标签改为 [新标签]
```

### 自定义展示页面

```
给 [页面名] 创建一个自定义展示页面
```
```
生成一个企业协作产品的落地页
```

## 闪记转 PRD

```
将钉钉闪记转换为 PRD 文档
```

```
从会议记录生成 PRD：[粘贴笔记]
```

```
从附件闪记创建项目需求文档
```

```
将会议笔记转换为结构化产品需求文档
```

<Note>
需要登录。该命令读取会议记录并使用 AI 生成结构化的 PRD 文档。输出保存到 `prd/` 目录。
</Note>

## 集成与自动化

### 创建集成流程

```
创建一个集成流程，在新记录提交时发送通知
```

```
在 [表单名] 上设置自动化，在新提交时通知 [团队/用户]
```

```
创建一个 [表单名] 工作流，触发邮件通知给 [收件人]
```

### 添加自动化操作

```
向 [流程名] 集成添加数据查询操作
```

```
添加记录创建操作，自动在 [目标表单] 中创建条目
```

<Note>
集成流程在表单事件（插入、更新、删除）时触发，可以执行通知、数据查询和记录创建。使用 CLI 创建和配置流程。
</Note>

<Info>
`dws` 命令需要单独安装钉钉 CLI。如果尚未安装，请运行 `openyida dws install`。输出默认为表格格式；添加 `-f json` 可输出 JSON 格式。
</Info>

</Tip>
