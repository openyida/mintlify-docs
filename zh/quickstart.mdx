## 环境要求

在开始之前，请确保已安装以下依赖：

| 依赖 | 版本 | 检查命令 |
|------|------|----------|
| [Node.js](https://nodejs.org) | ≥ 18 | `node --version` |

<Tip>
  这就是全部所需。OpenYida 是纯 Node.js 实现 — 无需 Python、无需 Playwright、无需构建步骤。
</Tip>

---

## 第 1 步 — 安装

```bash
npm install -g openyida
```

验证安装：

```bash
openyida --version
```

---

## 第 2 步 — 登录宜搭

```bash
openyida login
```

OpenYida 将打开浏览器窗口（或显示二维码）供您使用宜搭/阿里巴巴账号进行身份验证。凭证会缓存在本地 — 只需执行一次。

<Info>
  如果您使用 **悟空** 作为 AI 工具，请使用特殊登录标志：

  ```bash
  openyida login --wukong
  ```
</Info>

---

## 第 3 步 — 检查环境

```bash
openyida env
```

这将检测当前激活的 AI 编程工具（Claude Code、Cursor、悟空等）并显示您的当前登录状态。切换环境时随时运行此命令。

---

## 第 4 步 — 初始化项目工作区

```bash
openyida copy
```

这将把 `project/` 工作区模板复制到您当前 AI 工具的工作目录中，让 AI 能够访问您的应用配置和页面模板。

---

## 第 5 步 — 开始构建

现在只需与您的 AI 编程助手对话。以下是一些入门提示词：

```
在宜搭上为我构建一个 IPD 系统，管理完整的芯片生产工作流程
```

```
帮我搭建一个包含线索、客户和交易的 CRM 系统
```

```
创建一个个人工资计算器应用
```

```
在宜搭上生成一个生日祝福小游戏
```

AI 将使用 OpenYida 的技能包按顺序调用正确的 CLI 命令：`create-app` → `create-form` / `create-page` → `publish` → `save-share-config`。

---

## 第 6 步 — 发布应用

AI 生成应用后，使用以下命令发布：

```bash
openyida publish
```

或者询问 AI：
```
发布 [页面名称] 页面
让页面可以公开访问
```

---

## 运行诊断

如果感觉有任何问题，运行内置的诊断工具：

```bash
openyida doctor
```

它会检查您的 Node.js 版本、登录状态和环境配置，并可以使用 `--fix` 自动修复常见问题。
